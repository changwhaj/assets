주제 #: 1

SIMULATION
-






작업
-

기존 네임스페이스 echo에allow-port-from-namespace라는 새 NetworkPolicy를 만듭니다.

새 NetworkPolicy가 내부 네임스페이스의 포드가 네임스페이스 에코에 있는 포드의 포트 9200/tcp에 연결되도록 허용하는지 확인하세요.

또한 새 NetworkPolicy가 다음 사항을 확인하는지 확인하세요.

• 포트 9200/tcp에서 수신 대기하지 않는 Pod에 대한 액세스를 허용하지 않습니다.
• 내부 네임스페이스에 있지 않은 Pod로부터의 액세스를 허용하지 않습니다.

Actual exam question from CNCF's CKA
Question #: 19
Topic #: 1

SIMULATION
-






Task
-

Create a new NetworkPolicy named allow-port-from-namespace in the existing namespace echo.

Ensure that the new NetworkPolicy allows Pods in namespace internal to connect to port 9200/tcp of Pods in namespace echo.

Further ensure that the new NetworkPolicy:

• does not allow access to Pods, which don't listen on port 9200/tcp
• does not allow access from Pods, which are not in namespace internal
