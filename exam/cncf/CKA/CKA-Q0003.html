시뮬레이션 -


작업 -
버전 1.22.1을 실행하는 기존 Kubernetes 클러스터가 있는 경우 마스터 노드의 모든 Kubernetes 제어 플레인과 노드 구성 요소를 버전 1.22.2로만 업그레이드합니다.
업그레이드하기 전에 반드시 마스터 노드를 드레이닝하고 업그레이드 후에는 차단을 해제하세요.

또한 마스터 노드에서 kubelet 및 kubectl을 업그레이드해야 합니다.

Actual exam question from CNCF's CKA
Question #: 3
Topic #: 1

SIMULATION -


Task -
Given an existing Kubernetes cluster running version 1.22.1, upgrade all of the Kubernetes control plane and node components on the master node only to version 1.22.2.
Be sure to drain the master node before upgrading it and uncordon it after the upgrade.

You are also expected to upgrade kubelet and kubectl on the master node.
