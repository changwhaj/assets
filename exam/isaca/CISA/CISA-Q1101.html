A finance department has a multi-year project to upgrade the enterprise resource planning (ERP) system hosting the general ledger, and in year one, the system version upgrade will be applied. Which of the following should be the PRIMARY focus of the IS auditor reviewing the first year of the project?

  • A. Network performance testing
  • B. User acceptance testing (UAT)
  • C. Unit testing
  • D. Regression testing

Selected Answer: B
While network performance testing, unit testing, and regression testing are also essential components of the SDLC, they are not the primary focus of the IS auditor reviewing the first year of the project. Network performance testing is typically performed during the system testing phase, which occurs after UAT. Unit testing is performed by developers during the coding phase, and regression testing is performed during the testing phase to ensure that changes to the system do not adversely affect existing functionality.

Selected Answer: C
The primary focus of the IS auditor reviewing the first year of the project, where the system version upgrade will be applied, should be on unit testing. Unit testing is a phase of the testing process where individual components or modules of the software are tested in isolation to ensure their correctness and functionality. This is particularly relevant when upgrading the system version to verify that the individual units or components of the ERP system are functioning as expected with the new version.
