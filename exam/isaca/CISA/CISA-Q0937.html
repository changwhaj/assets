Selected Answer: D
D. Function point analysis Function point analysis is a widely recognized technique for measuring the size and complexity of software applications. It helps in providing a more accurate and standardized way of estimating project size and complexity. By using function points, organizations can establish a common and objective metric for evaluating development efforts, which can lead to more reliable project planning and estimation. While Agile development approaches, Critical Path Methodology (CPM), and Rapid Application Development (RAD) are valuable methodologies for managing and executing software development projects, they don't directly address the issue of inconsistent and inaccurate estimation, which is the primary concern mentioned in the question. Function point analysis, on the other hand, is specifically designed to improve the accuracy of size and complexity estimation in software development projects.
