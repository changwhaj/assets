Selected Answer: C
When evaluating the accuracy of a spreadsheet that contains several macros, both the version history and reconciliation of key calculations are important. However, the **reconciliation of key calculations** is generally considered **more crucial**. This is because it directly verifies the accuracy of the calculations performed by the macros. While version history can provide information about the changes made to the spreadsheet over time, it does not necessarily ensure the accuracy of the current calculations. Therefore, an IS auditor should prioritize reviewing the reconciliation of key calculations.
