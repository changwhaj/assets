B Is correct answer here. When gathering information from storage devices, analysts should never access hard drives or other media from a live system. Instead, they should power off the system (after collecting other evidence), remove the storage device, and then attach the storage device to a dedicated forensic workstation, using a write blocker. Write blockers are hardware adapters that physically sever the portion of the cable used to connect the storage device that would write data to the device, reducing the likelihood of accidental tampering with the device. After connecting the device to a live workstation, the analyst should immediately calcu- late a cryptographic hash of the device contents and then use forensic tools to create a forensic image of the device: a bitwise copy of the data stored on the device. The analyst should then compute the cryptographic hash of that image to ensure that it is identical to the original media contents.

When a desktop computer is being used to perpetrate a fraud, and the data on the machine needs to be secured for evidence, the FIRST step to be taken is: D. Capture a forensic image of the computer. Capturing a forensic image of the computer is the initial and critical step in preserving the data and ensuring its integrity for potential legal proceedings. A forensic image is a bit-by-bit copy or snapshot of the entire hard drive, including all data, files, and system information. It captures the computer's entire state at a specific point in time, ensuring that the evidence remains intact and unaltered.

The BEST course of action in this scenario is to capture a forensic image of the computer FIRST. This involves creating an exact copy of the entire hard drive, including all data and system files, in a forensically sound manner, which can be used as evidence if required. The forensic image should be stored on a separate, secure storage device and not on the original computer's hard drive. This ensures that the original data is preserved in its original state and can be analyzed later, without making any changes to the original data. Once the forensic image is obtained, further actions can be taken, such as analyzing the image for evidence, encrypting it for security, or copying it to an external hard drive.
