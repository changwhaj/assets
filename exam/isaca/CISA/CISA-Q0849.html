the IS auditor should recommend a combination of options B and D: B. Use generalized audit software: Generalized audit software is specifically designed for auditing purposes and can help in extracting and analyzing data from databases efficiently. Leverage a random number generator: A random number generator can be used to select a random sample from the remaining client balances after extracting the 80 largest ones. This ensures that the sample is truly random and not biased. This approach combines the use of specialized software for data extraction and analysis (option B) with a random number generator (option D) to ensure that the sample is both representative and random, as required for testing purposes.
