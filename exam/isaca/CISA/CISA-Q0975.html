Selected Answer: A
Performing periodic reconciliations is a good practice to validate the accuracy of data interfaces between systems. It involves comparing the data in the transaction processing system with the data in the general ledger to identify and resolve any discrepancies. This approach helps ensure that transactions are correctly recorded in both systems and provides ongoing assurance that the interface is working as intended.
