Selected Answer: A
The option that would MOST effectively ensure the integrity of data transmitted over a network is A. Message encryption. Message encryption involves encoding data using cryptographic algorithms, making it unreadable to unauthorized individuals. By encrypting the data, it ensures that the integrity of the transmitted information is maintained. Any unauthorized modification or tampering with the data during transmission would render it unreadable and unusable to unauthorized parties. A message digest (option D) is a hash value generated from the data, which can be used to verify the integrity of the message. However, on its own, a message digest does not protect the data during transmission. It can be used alongside other mechanisms, such as message authentication codes (MAC), to provide both integrity and authentication.
