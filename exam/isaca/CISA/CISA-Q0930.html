Selected Answer: C
The BEST way for an IS auditor to determine the completeness of data migration is: C. Reconcile migrated records with records in the source system. Reconciling migrated records with records in the source system involves comparing the data in the target system (after migration) with the data in the source system (before migration). This method allows the auditor to verify that all the data has been successfully migrated and that there are no missing or incomplete records. It provides a direct and concrete way to ensure completeness. Option A (Review migration logs) is important but may not guarantee completeness as it primarily focuses on identifying failures and errors in the migration process. Option B (Review the implemented data cleanup process) is relevant but does not directly address the completeness of the migration. Option D (Examine formal departmental review of the data migration) is useful for understanding the process but may not provide a direct measure of data completeness.
