  • B. Power usage will increase.
  • C. The new server will not be fully utilized after migration.

Consolidating applications from multiple servers onto a single new server can lead to underutilization of resources if the new server is not properly configured or if the applications do not have consistent resource demands. This can result in wasted resources and increased costs without realizing the expected benefits of consolidation.
