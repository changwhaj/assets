Selected Answer: D
D. Inability to deploy updated security patches. From an IS auditor's perspective, an incomplete inventory of deployed software in an organization poses several risks. However, the greatest risk is the inability to deploy updated security patches. When an organization does not have a complete inventory of its deployed software, it becomes difficult to determine which systems are running specific software applications and therefore, which systems need to be updated with the latest security patches. This can leave the organization vulnerable to security threats and attacks, as attackers can exploit known vulnerabilities in the software that have already been patched by the software vendors. To mitigate this risk, IS auditors would recommend that the organization maintain an up-to-date inventory of its deployed software and regularly deploy security patches to all applicable systems.
