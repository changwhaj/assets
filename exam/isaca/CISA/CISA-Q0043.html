A. Quicker end user acceptance. An advantage of using the agile software development methodology over the waterfall methodology is that it typically leads to quicker end user acceptance. In agile development, the software is developed iteratively in smaller increments, allowing for continuous user feedback and involvement throughout the development process. This iterative approach often results in end users getting to interact with and provide feedback on working software sooner, leading to quicker acceptance and alignment with user needs.
