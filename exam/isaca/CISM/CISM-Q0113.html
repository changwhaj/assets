A critical server for a hospital has been encrypted by ransomware. The hospital is unable to function effectively without this server. Which of the following would
MOST effectively allow the hospital to avoid paying the ransom?

  • A. A continual server replication process
  • B. Employee training on ransomware
  • C. A properly tested offline backup system
  • D. A properly configured firewall

Selected Answer: C
A properly tested offline backup system would be the most effective way to avoid paying the ransom in this scenario. With a proper offline backup system, the hospital would be able to recover the encrypted server without paying the ransom. Continual server replication, employee training on ransomware, and a properly configured firewall may help prevent ransomware attacks, but they would not help the hospital avoid paying the ransom in this scenario.
