Amazon의 AWS Certified DevOps Engineer - Professional DOP-C02 의 시험 문제
질문 #: 330
주제 #: 1

한 회사가 여러 AWS 계정에서 비정상적인 로그인 시도를 감지했습니다. DevOps 엔지니어는 여러 번 로그인 시도가 실패하면 회사 보안 팀에 알림을 보내는 솔루션을 구현해야 합니다. DevOps 엔지니어는 이미 Amazon Simple Notification Service(Amazon SNS) 토픽을 생성했고 보안 팀을 SNS 토픽에 구독했습니다.

어떤 솔루션이 가장 적은 운영 노력으로 알림을 제공할까요?

  • A. AWS CloudTrail을 구성하여 관리 이벤트를 Amazon CloudWatch Logs 로그 그룹으로 보냅니다. 실패한 ConsoleLogin 이벤트와 일치하도록 CloudWatch Logs 메트릭 필터를 만듭니다. 메트릭 필터를 기반으로 하는 CloudWatch 알람을 만듭니다. SNS 토픽으로 메시지를 보내도록 알람 작업을 구성합니다. 가장 많이 투표된
  • B. AWS CloudTrail을 구성하여 Amazon S3 버킷으로 관리 이벤트를 보냅니다. S3 버킷의 로그에서 실패한 로그인을 발견하면 실패를 반환하는 Amazon Athena 쿼리를 만듭니다. Amazon EventBridge 규칙을 만들어 주기적으로 쿼리를 실행합니다. 두 번째 EventBridge 규칙을 만들어 쿼리가 실패할 때를 감지하고 SNS 토픽에 메시지를 보냅니다.
  • C. AWS CloudTrail을 구성하여 데이터 이벤트를 Amazon CloudWatch Logs 로그 그룹으로 전송합니다. 실패한 ConsoleLogin 이벤트와 일치하도록 CloudWatch Logs 메트릭 필터를 만듭니다. 메트릭 필터를 기반으로 하는 CloudWatch 알람을 만듭니다. SNS 토픽으로 메시지를 전송하도록 알람 작업을 구성합니다.
  • D. AWS CloudTrail을 구성하여 데이터 이벤트를 Amazon S3 버킷으로 전송합니다. s3:ObjectCreated 이벤트 유형에 대한 Amazon S3 이벤트 알림을 구성합니다. ConsoleLogin 실패 이벤트로 이벤트 유형을 필터링합니다. 이벤트 알림을 구성하여 SNS 토픽으로 전달합니다.
제안된 답변 보기 답변 숨기기
제안된 답변: A (OK)
A는 AWS CloudTrail을 사용하여 관리 이벤트를 CloudWatch Logs로 전송하고, CloudWatch Logs 메트릭 필터를 사용하여 ConsoleLogin 실패 이벤트를 감지합니다. 이를 통해 CloudWatch 알람을 생성하고, 알람이 발생하면 SNS 토픽으로 메시지를 보내는 방식입니다. 이 방법은 최소한의 운영 노력으로 빠르고 효율적으로 로그인 실패 알림을 제공하는 최적의 솔루션입니다.

다른 옵션들은 더 복잡한 구성을 요구하거나, S3를 사용한 접근법이므로 운영 오버헤드가 더 크고 관리가 복잡할 수 있습니다. A는 간단하고 효율적입니다.

Exam question from Amazon's AWS Certified DevOps Engineer - Professional DOP-C02
Question #: 330
Topic #: 1

A company detects unusual login attempts in many of its AWS accounts. A DevOps engineer must implement a solution that sends a notification to the company's security team when multiple failed login attempts occur. The DevOps engineer has already created an Amazon Simple Notification Service (Amazon SNS) topic and has subscribed the security team to the SNS topic.

Which solution will provide the notification with the LEAST operational effort?

  • A. Configure AWS CloudTrail to send management events to an Amazon CloudWatch Logs log group. Create a CloudWatch Logs metric filter to match failed ConsoleLogin events. Create a CloudWatch alarm that is based on the metric filter. Configure an alarm action to send messages to the SNS topic. Most Voted
  • B. Configure AWS CloudTrail to send management events to an Amazon S3 bucket. Create an Amazon Athena query that returns a failure if the query finds failed logins in the logs in the S3 bucket. Create an Amazon EventBridge rule to periodically run the query. Create a second EventBridge rule to detect when the query fails and to send a message to the SNS topic.
  • C. Configure AWS CloudTrail to send data events to an Amazon CloudWatch Logs log group. Create a CloudWatch logs metric filter to match failed ConsoleLogin events. Create a CloudWatch alarm that is based on the metric filter. Configure an alarm action to send messages to the SNS topic.
  • D. Configure AWS CloudTrail to send data events to an Amazon S3 bucket. Configure an Amazon S3 event notification for the s3:ObjectCreated event type. Filter the event type by ConsoleLogin failed events. Configure the event notification to forward to the SNS topic.
