Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 225
주제 #: 1

회사는 다중 AZ 배포에서 Amazon RDS의 Microsoft SQL Server를 애플리케이션의 데이터베이스 엔진으로 사용합니다. 이 회사는 최근 다른 회사에 인수되었습니다. 데이터베이스 전문가는 새로운 명명 표준을 따르도록 데이터베이스 이름을 바꿔야 합니다.
데이터베이스 전문가가 데이터베이스 이름을 바꾸려면 어떤 단계 조합을 수행해야 합니까? (2개를 선택하세요.)

  • A. DB 인스턴스에 대한 자동 스냅샷을 끄십시오. rdsadmin.dbo.rds_modify_db_name 저장 프로시저를 사용하여 데이터베이스 이름을 바꿉니다. 자동 스냅샷을 켭니다.
  • B. DB 인스턴스에 대해 다중 AZ를 끄십시오. rdsadmin.dbo.rds_modify_db_name 저장 프로시저를 사용하여 데이터베이스 이름을 바꿉니다. 다중 AZ 미러링을 활성화합니다. 최다 투표
  • C. DB 인스턴스의 기존 스냅샷을 모두 삭제합니다. rdsadmin.dbo.rds_modify_db_name 저장 프로시저를 사용합니다.
  • D. 새 데이터베이스 연결 문자열로 애플리케이션을 업데이트합니다. 최다 투표
  • E. DB 인스턴스의 DNS 레코드를 업데이트합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 225
Topic #: 1

A company uses Microsoft SQL Server on Amazon RDS in a Multi-AZ deployment as the database engine for its application. The company was recently acquired by another company. A database specialist must rename the database to follow a new naming standard.
Which combination of steps should the database specialist take to rename the database? (Choose two.)

  • A. Turn off automatic snapshots for the DB instance. Rename the database with the rdsadmin.dbo.rds_modify_db_name stored procedure. Turn on the automatic snapshots.
  • B. Turn off Multi-AZ for the DB instance. Rename the database with the rdsadmin.dbo.rds_modify_db_name stored procedure. Turn on Multi-AZ Mirroring. Most Voted
  • C. Delete all existing snapshots for the DB instance. Use the rdsadmin.dbo.rds_modify_db_name stored procedure.
  • D. Update the application with the new database connection string. Most Voted
  • E. Update the DNS record for the DB instance.
