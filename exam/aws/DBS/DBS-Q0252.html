Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 252
주제 #: 1

한 회사의 데이터베이스 전문가는 수백 개의 레코드를 CSV 형식으로 저장하기 위해 Microsoft SQL Server DB 인스턴스용 Amazon RDS를 구축하고 있습니다. 고객 서비스 도구는 Amazon S3 버킷에 레코드를 업로드합니다.
이전에 회사에서 근무했던 직원은 필요한 CSV 필드를 데이터베이스 테이블에 매핑하기 위해 이미 사용자 정의 저장 프로시저를 만들었습니다. 데이터베이스 전문가는 이전 작업을 재사용하고 운영 오버헤드를 최소화하는 솔루션을 구현해야 합니다.
어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. Amazon S3 이벤트를 생성하여 AWS Lambda 함수를 호출합니다. .csv 파일을 구문 분석하고 SQL 클라이언트 라이브러리를 사용하여 INSERT 문을 실행하여 데이터를 테이블에 로드하도록 Lambda 함수를 구성합니다.
  • B. Amazon EC2에서 호스팅되는 사용자 지정 .NET 앱을 작성합니다. .csv 파일을 로드하도록 .NET 앱을 구성하고 사용자 지정 저장 프로시저를 호출하여 테이블에 데이터를 삽입합니다.
  • C. AWS msdb 저장 프로시저를 사용하여 Amazon S3에서 RDS D 드라이브로 .csv 파일을 다운로드합니다. 사용자 지정 저장 프로시저를 호출하여 RDS D 드라이브의 데이터를 테이블에 삽입합니다. 최다 투표
  • D. Amazon S3 이벤트를 생성하여 AWS Step Functions를 호출하여 .csv 파일을 구문 분석하고 사용자 지정 저장 프로시저를 호출하여 데이터를 테이블에 삽입합니다. 최다 투표

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 252
Topic #: 1

A company's database specialist is building an Amazon RDS for Microsoft SQL Server DB instance to store hundreds of records in CSV format. A customer service tool uploads the records to an Amazon S3 bucket.
An employee who previously worked at the company already created a custom stored procedure to map the necessary CSV fields to the database tables. The database specialist needs to implement a solution that reuses this previous work and minimizes operational overhead.
Which solution will meet these requirements?

  • A. Create an Amazon S3 event to invoke an AWS Lambda function. Configure the Lambda function to parse the .csv file and use a SQL client library to run INSERT statements to load the data into the tables.
  • B. Write a custom .NET app that is hosted on Amazon EC2. Configure the .NET app to load the .csv file and call the custom stored procedure to insert the data into the tables.
  • C. Download the .csv file from Amazon S3 to the RDS D drive by using an AWS msdb stored procedure. Call the custom stored procedure to insert the data from the RDS D drive into the tables. Most Voted
  • D. Create an Amazon S3 event to invoke AWS Step Functions to parse the .csv file and call the custom stored procedure to insert the data into the tables. Most Voted

Step 1: Download S3 Files Amazon RDS for SQL Server comes with several custom stored procedures and functions. These are located in the msdb database. The stored procedure to download files from S3 is "rds_download_from_s3".
