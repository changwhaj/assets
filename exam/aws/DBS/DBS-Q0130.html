주제 #: 1

한 회사에서 온프레미스 MySQL 데이터베이스를 MySQL용 Amazon RDS로 마이그레이션하려고 합니다. 회사의 보안 정책을 준수하려면 모든 데이터베이스를 암호화해야 합니다. 또한 RDS DB 인스턴스 스냅샷은 테스트 및 스테이징 환경을 프로비저닝하기 위해 다양한 계정에서 공유되어야 합니다.
이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. AWS Key Management Service(AWS KMS) 고객 관리형 CMK를 사용하여 MySQL용 RDS DB 인스턴스를 생성합니다. 다른 AWS 계정의 Amazon 리소스 이름(ARN)을 보안 주체로 포함하도록 키 정책을 업데이트한 다음 kms:CreateGrant 작업을 허용합니다. 최다 투표
  • B. AWS 관리형 CMK를 사용하여 MySQL용 RDS DB 인스턴스를 생성합니다. 다른 AWS 계정의 Amazon 리소스 이름(ARN)을 보안 주체로 포함하는 새 키 정책을 생성한 다음 kms:CreateGrant 작업을 허용합니다.
  • C. AWS 소유 CMK를 사용하여 MySQL용 RDS DB 인스턴스를 생성합니다. 다른 AWS 계정의 관리자 사용자 이름을 보안 주체로 포함하도록 새 키 정책을 생성한 다음 kms:CreateGrant 작업을 허용합니다.
  • D. AWS CloudHSM 키를 사용하여 MySQL용 RDS DB 인스턴스를 생성합니다. 다른 AWS 계정의 Amazon 리소스 이름(ARN)을 보안 주체로 포함하도록 키 정책을 업데이트한 다음 kms:CreateGrant 작업을 허용합니다.

Question #: 130
Topic #: 1

A company wants to migrate its on-premises MySQL databases to Amazon RDS for MySQL. To comply with the company's security policy, all databases must be encrypted at rest. RDS DB instance snapshots must also be shared across various accounts to provision testing and staging environments.
Which solution meets these requirements?

  • A. Create an RDS for MySQL DB instance with an AWS Key Management Service (AWS KMS) customer managed CMK. Update the key policy to include the Amazon Resource Name (ARN) of the other AWS accounts as a principal, and then allow the kms:CreateGrant action. Most Voted
  • B. Create an RDS for MySQL DB instance with an AWS managed CMK. Create a new key policy to include the Amazon Resource Name (ARN) of the other AWS accounts as a principal, and then allow the kms:CreateGrant action.
  • C. Create an RDS for MySQL DB instance with an AWS owned CMK. Create a new key policy to include the administrator user name of the other AWS accounts as a principal, and then allow the kms:CreateGrant action.
  • D. Create an RDS for MySQL DB instance with an AWS CloudHSM key. Update the key policy to include the Amazon Resource Name (ARN) of the other AWS accounts as a principal, and then allow the kms:CreateGrant action.
