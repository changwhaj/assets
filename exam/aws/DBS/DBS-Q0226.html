Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 226
주제 #: 1

회사는 TDE(투명한 데이터 암호화)가 활성화된 온프레미스 Microsoft SQL Server Enterprise 에디션 데이터베이스를 호스팅합니다. 데이터베이스 크기는 20TB이며 스파스 테이블을 포함합니다. 회사는 다가오는 주말로 예정된 유지 관리 기간 동안 데이터베이스를 SQL Server용 Amazon RDS로 마이그레이션해야 합니다. 대상 DB 인스턴스에 대해 저장 데이터 암호화를 활성화해야 합니다.
운영상 가장 효율적인 방식으로 데이터베이스를 AWS로 마이그레이션하기 위해 회사는 어떤 단계 조합을 취해야 합니까? (2개를 선택하세요.)

  • A. AWS Database Migration Service(AWS DMS)를 사용하여 온프레미스 소스 데이터베이스에서 SQL Server용 RDS 대상 데이터베이스로 마이그레이션합니다.
  • B. TDE를 비활성화합니다. 암호화 없이 데이터베이스 백업을 생성합니다. 백업을 Amazon S3에 복사합니다. 최다 투표
  • C. 백업을 SQL Server DB 인스턴스용 RDS로 복원합니다. SQL Server DB 인스턴스용 RDS에 대해 TDE를 활성화합니다. 최다 투표
  • D. AWS Snowball Edge 디바이스를 설정합니다. 데이터베이스 백업을 장치에 복사합니다. 디바이스를 AWS로 보냅니다. Amazon S3에서 데이터베이스를 복원합니다.
  • E. 데이터를 Amazon RDS로 전송하기 전에 클라이언트 측 암호화로 데이터를 암호화합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 226
Topic #: 1

A company hosts an on-premises Microsoft SQL Server Enterprise edition database with Transparent Data Encryption (TDE) enabled. The database is 20 TB in size and includes sparse tables. The company needs to migrate the database to Amazon RDS for SQL Server during a maintenance window that is scheduled for an upcoming weekend. Data-at-rest encryption must be enabled for the target DB instance.
Which combination of steps should the company take to migrate the database to AWS in the MOST operationally efficient manner? (Choose two.)

  • A. Use AWS Database Migration Service (AWS DMS) to migrate from the on-premises source database to the RDS for SQL Server target database.
  • B. Disable TDE. Create a database backup without encryption. Copy the backup to Amazon S3. Most Voted
  • C. Restore the backup to the RDS for SQL Server DB instance. Enable TDE for the RDS for SQL Server DB instance. Most Voted
  • D. Set up an AWS Snowball Edge device. Copy the database backup to the device. Send the device to AWS. Restore the database from Amazon S3.
  • E. Encrypt the data with client-side encryption before transferring the data to Amazon RDS.
