Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 142
주제 #: 1

한 회사에 새로운 MySQL DB 인스턴스용 Amazon RDS를 시작하는 데 사용되는 JSON으로 작성된 AWS CloudFormation 템플릿이 있습니다. 보안팀은 템플릿을 사용하여 시작되는 모든 새 DB 인스턴스에 대해 마스터 비밀번호가 30일마다 자동으로 교체되도록 데이터베이스 전문가에게 요청했습니다.
이러한 요구 사항을 충족하는 가장 운영 효율적인 솔루션은 무엇입니까?

  • A. Amazon S3 객체에 비밀번호를 저장하십시오. AWS KMS 키를 사용하여 S3 객체를 암호화합니다. EnableKeyRotation 속성을 true로 설정하여 KMS 키가 30일마다 교체되도록 설정합니다. CloudFormation 사용자 지정 리소스를 사용하여 S3 객체를 읽고 비밀번호를 추출합니다.
  • B. 비밀을 교체하는 AWS Lambda 함수를 생성합니다. CloudFormation 템플릿을 수정하여 AWS::SecretsManager::RotationSchedule 리소스를 추가합니다. RotationLambdaARN 값을 구성하고 RotationRules 속성의 경우 AutomaticallyAfterDays 파라미터를 30으로 설정합니다. 최다 투표
  • C. AWS KMS 키를 데이터베이스 비밀번호로 사용하도록 CloudFormation 템플릿을 수정합니다. ScheduleExpression 매개변수를 ***/30***로 설정하여 30일마다 키를 교체하도록 KMS API를 호출하도록 Amazon EventBridge 규칙을 구성합니다.
  • D. MySQL용 Amazon RDS DB 인스턴스를 AWS IAM과 통합하고 마스터 데이터베이스 사용자 비밀번호를 중앙에서 관리합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 142
Topic #: 1

A company has an AWS CloudFormation template written in JSON that is used to launch new Amazon RDS for MySQL DB instances. The security team has asked a database specialist to ensure that the master password is automatically rotated every 30 days for all new DB instances that are launched using the template.
What is the MOST operationally efficient solution to meet these requirements?

  • A. Save the password in an Amazon S3 object. Encrypt the S3 object with an AWS KMS key. Set the KMS key to be rotated every 30 days by setting the EnableKeyRotation property to true. Use a CloudFormation custom resource to read the S3 object to extract the password.
  • B. Create an AWS Lambda function to rotate the secret. Modify the CloudFormation template to add an AWS::SecretsManager::RotationSchedule resource. Configure the RotationLambdaARN value and, for the RotationRules property, set the AutomaticallyAfterDays parameter to 30. Most Voted
  • C. Modify the CloudFormation template to use the AWS KMS key as the database password. Configure an Amazon EventBridge rule to invoke the KMS API to rotate the key every 30 days by setting the ScheduleExpression parameter to ***/30***.
  • D. Integrate the Amazon RDS for MySQL DB instances with AWS IAM and centrally manage the master database user password.
