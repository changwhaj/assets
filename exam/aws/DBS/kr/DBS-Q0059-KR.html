Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 59
주제 #: 1

IT 컨설팅 회사는 개발 환경 데이터베이스를 운영할 때 비용을 절감하려고 합니다. 이 회사의 워크플로는
각 개발 그룹에 대해 여러 Amazon Aurora MySQL DB 클러스터를 생성합니다. Aurora DB 클러스터는 하루에 8시간 동안만 사용됩니다. 그런 다음 DB 클러스터는 2주 동안 지속되는 개발 주기가 끝날 때 삭제할 수 있습니다.
다음 중 가장 비용 효율적인 솔루션을 제공하는 것은 무엇입니까?

  • A. AWS CloudFormation 템플릿을 사용하십시오. 각 개발 그룹에 대해 DB 클러스터와 함께 스택을 배포합니다. 개발 주기가 끝나면 스택을 삭제하십시오.
  • B. Aurora DB 복제 기능을 사용합니다. 단일 개발 및 테스트 Aurora DB 인스턴스를 배포하고 개발 그룹에 대한 복제 인스턴스를 생성합니다. 개발 주기가 끝나면 클론을 삭제하십시오. 최다 투표
  • C. Aurora 복제본을 사용합니다. 마스터 자동 일시 중지 컴퓨팅 용량 옵션에서 각 개발 그룹에 대한 복제본을 생성하고 각 복제본을 마스터로 승격합니다. 개발 주기가 끝나면 복제본을 삭제하십시오.
  • D. Aurora 서버리스를 사용합니다. 현재 Aurora 스냅샷을 복원하고 각 개발 그룹의 서버리스 클러스터에 배포합니다. 클러스터에서 컴퓨팅 용량을 일시 중지하고 적절한 제한 시간을 설정하는 옵션을 활성화합니다. 최다 투표
