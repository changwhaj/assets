Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 166
주제 #: 1

회사에서 IBM Informix 데이터베이스를 상시 가동 가용성 그룹(AG)을 사용하여 SQL Server용 Amazon RDS의 다중 AZ 배포로 마이그레이션하고 있습니다. Always On AG 수신기의 SQL
Server 에이전트 작업은 5분 간격으로 실행되어 Informix 데이터베이스와 SQL Server 데이터베이스 간에 데이터를 동기화합니다. 사용자는 최소한의 대기 시간으로 보조 노드로 성공적으로 장애 조치한 후 몇 시간 동안 오래된 데이터를 경험합니다.
장애 조치 후 사용자가 최신 데이터를 볼 수 있도록 데이터베이스 전문가는 무엇을 해야 합니까?

  • A. Always On AG 수신기에서 캐시된 DNS 값에 대해 TTL을 30초 미만으로 설정합니다. 최다 투표
  • B. 대규모 트랜잭션을 5분 이내에 완료되는 여러 개의 작은 트랜잭션으로 나눕니다.
  • C. 보조 노드의 데이터베이스를 읽기 전용 모드로 설정합니다.
  • D. 장애 후 보조 노드가 인계될 때 스크립트에서 보조 노드에 SQL Server 에이전트 작업을 생성합니다. 최다 투표
