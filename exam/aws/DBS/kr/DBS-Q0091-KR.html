Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 91
주제 #: 1

대규모 다국적 금융 회사의 데이터베이스 전문가는 개발 중인 고가용성 애플리케이션의 재해 복구 전략 설계를 담당하고 있습니다. 애플리케이션은 Amazon DynamoDB 테이블을 데이터 저장소로 사용합니다. 애플리케이션에는 1분의 RTO(복구 시간 목표)와 2분의 RPO(복구 지점 목표)가 필요합니다.
데이터베이스 전문가가 DynamoDB 테이블에 대해 권장해야 하는 운영상 효율적인 재해 복구 전략은 무엇입니까?

  • A. 다른 리전의 DynamoDB 테이블에 데이터를 복사하는 AWS Lambda 함수에 의해 처리되는 DynamoDB 스트림을 생성합니다.
  • B. 다른 리전에서 DynamoDB 전역 테이블 복제본을 사용합니다. 두 테이블 모두에 대해 지정 시간 복구를 활성화합니다. 최다 투표
  • C. 다른 리전에서 DynamoDB Accelerator 테이블을 사용합니다. 테이블에 대해 지정 시간 복구를 활성화합니다.
  • D. AWS Backup 계획을 생성하고 DynamoDB 테이블을 리소스로 할당합니다.
