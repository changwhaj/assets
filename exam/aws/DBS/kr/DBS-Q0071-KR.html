Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 71
주제 #: 1

한 회사가 하나의 AWS 계정에서 2계층 전자상거래 애플리케이션을 실행하고 있습니다. 웹 서버는 Amazon RDS for MySQL 다중 AZ DB 인스턴스를 사용하여 배포됩니다. 개발자가 실수로 프로덕션 환경에서 데이터베이스를 삭제했습니다. 데이터베이스가 복원되었지만 이로 인해 몇 시간의 가동 중지 시간과 수익 손실이 발생했습니다.
향후 이와 같은 오류가 발생하지 않도록 하려면 데이터베이스 전문가가 기존 IAM 정책의 어떤 조합을 변경해야 합니까? (3개를 선택하세요.)

  • A. 그룹, 사용자 및 역할에 최소 권한 부여 최다 투표
  • B. 모든 사용자가 백업에서 데이터베이스를 복원하도록 허용하여 데이터베이스 복원을 위한 전체 중단 시간을 줄입니다.
  • C. 민감한 리소스 및 API 작업에 액세스하기 위해 민감한 작업에 대한 다단계 인증 활성화 최다 투표
  • D. 정책 조건을 사용하여 선택적 IP 주소에 대한 액세스를 제한합니다. 최다 투표
  • E. AccessList Controls 정책 유형을 사용하여 데이터베이스 인스턴스 삭제에 대한 사용자 제한
  • F. AWS CloudTrail 로깅 및 향상된 모니터링 활성화
