Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 8
주제 #: 1

금융 서비스 회사는 회사 전체에서 다양한 애플리케이션을 지원하는 공유 데이터 서비스를 개발하고 있습니다. 데이터베이스 전문가는 클러스터 모드를 사용하여 Redis용 Amazon ElastiCache를 활용하여 성능과 확장성을 향상시키는 솔루션을 설계했습니다. 클러스터는 포트 6379에서 수신 대기하도록 구성되어 있습니다.
데이터베이스 전문가는 캐시 데이터를 보호하고 무단 액세스로부터 보호하기 위해 어떤 단계 조합을 수행해야 합니까? (3개를 선택하세요.)

  • A. ElastiCache 클러스터에서 전송 중 및 미사용 암호화를 활성화합니다. 최다 투표
  • B. Amazon CloudWatch 지표가 ElastiCache 클러스터에 구성되어 있는지 확인합니다.
  • C. ElastiCache 클러스터의 보안 그룹이 자신의 모든 인바운드 트래픽과 신뢰할 수 있는 클라이언트의 TCP 포트 6379에서의 인바운드 트래픽만 허용하는지 확인합니다. 최다 투표
  • D. 애플리케이션 서비스 역할이 모든 ElastiCache API 작업에 액세스할 수 있도록 IAM 정책을 생성합니다.
  • E. ElastiCache 클라이언트용 보안 그룹이 신뢰할 수 있는 ElastiCache 클러스터의 보안 그룹에서 들어오는 인바운드 TCP 포트 6379 및 포트 22 트래픽을 승인하는지 확인합니다.
  • F. 클러스터가 auth-token 매개변수로 생성되고 이 매개변수가 모든 후속 명령에서 사용되는지 확인합니다. 최다 투표
