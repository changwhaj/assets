시험 AWS 공인 데이터베이스 - 전문 분야 주제 1 질문 7 토론

Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 7
주제 #: 1

의류 회사는 맞춤형 전자상거래 애플리케이션과 PostgreSQL 데이터베이스를 사용하여 여러 국가의 수천 명의 사용자에게 의류를 판매합니다. 이 회사는 애플리케이션과 데이터베이스를 온프레미스 데이터 센터에서 AWS 클라우드로 마이그레이션하고 있습니다. 이 회사는 애플리케이션으로 Amazon EC2를, 데이터베이스로 PostgreSQL용 Amazon RDS를 선택했습니다. 회사는 데이터베이스 암호를 60일마다 변경하도록 요구합니다. 데이터베이스 전문가는 데이터베이스에 연결하기 위해 웹 애플리케이션에서 사용하는 자격 증명이 안전하게 관리되는지 확인해야 합니다.
데이터베이스 자격 증명을 안전하게 관리하기 위해 데이터베이스 전문가는 어떤 접근 방식을 취해야 합니까?

  • A. Amazon S3 버킷의 텍스트 파일에 자격 증명을 저장합니다. 버킷에 대한 권한을 인스턴스 프로파일과 연결된 IAM 역할로만 제한하십시오. 텍스트 파일을 다운로드하고 시작 시 자격 증명을 검색하도록 애플리케이션을 수정합니다. 60일마다 텍스트 파일을 업데이트합니다.
  • B. 애플리케이션이 데이터베이스에 연결하도록 IAM 데이터베이스 인증을 구성합니다. IAM 사용자를 생성하고 전자상거래 사용자별로 별도의 데이터베이스 사용자에 매핑합니다. 사용자가 60일마다 암호를 업데이트하도록 요구합니다.
  • C. 자격 증명을 AWS Secrets Manager에 저장합니다. 비밀에 대한 권한을 인스턴스 프로파일과 연결된 IAM 역할로만 제한하십시오. 시작 시 Secrets Manager에서 자격 증명을 검색하도록 애플리케이션을 수정합니다. 순환 간격을 60일로 구성합니다. 최다 투표
  • D. 애플리케이션 AMI의 암호화된 텍스트 파일에 자격 증명을 저장합니다. AWS KMS를 사용하여 텍스트 파일을 해독하기 위한 키를 저장합니다. 텍스트 파일의 암호를 해독하고 시작 시 자격 증명을 검색하도록 애플리케이션을 수정합니다. 텍스트 파일을 업데이트하고 60일마다 새 AMI를 게시합니다.
