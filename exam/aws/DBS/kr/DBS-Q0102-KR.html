Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 102
주제 #: 1

회사에는 사내 웹 애플리케이션을 프런트 엔드로 사용하여 SQL Server용 Amazon RDS에서 실행되는 프로덕션 환경이 있습니다. 마지막 애플리케이션 유지 관리 기간 동안 관리를 위한 보고 기능을 향상시키기 위해 웹 애플리케이션에 새로운 기능이 추가되었습니다. 업데이트 이후로 애플리케이션이 일부 보고 쿼리에 대한 응답 속도가 느립니다.
회사는 문제의 원인을 어떻게 식별해야 합니까?

  • A. Microsoft .NET 및 Microsoft SQL Server용 Amazon CloudWatch Application Insights를 설치하고 구성합니다. CloudWatch 대시보드를 사용하여 근본 원인을 식별하십시오.
  • B. RDS 성능 개선 도우미를 활성화하고 어떤 쿼리가 문제를 일으키는지 확인합니다. 문제를 해결하려면 쿼리 변경을 요청하십시오. 최다 투표
  • C. Amazon RDS와 함께 배포된 AWS X-Ray를 사용하여 쿼리 시스템 추적을 추적합니다.
  • D. 지원 요청을 생성하고 AWS Support와 협력하여 문제의 원인을 식별합니다.
