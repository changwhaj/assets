데이터베이스 전문가가 AWS DMS를 사용하여 Oracle DB 인스턴스용 2TB Amazon RDS를 PostgreSQL DB 인스턴스용 RDS로 마이그레이션하고 있습니다. 소스 RDS Oracle
DB 인스턴스는 us-east-1 리전의 VPC에 있습니다. PostgreSQL DB 인스턴스용 대상 RDS는 use-west-2 리전의 VPC에 있습니다.
가장 최적의 성능을 위해 AWS DMS 복제 인스턴스를 어디에 배치해야 합니까?

  • A. 원본 DB 인스턴스의 동일한 리전 및 VPC 내
  • B. 대상 DB 인스턴스와 동일한 리전 및 VPC 내
  • C. 대상 DB 인스턴스와 동일한 VPC 및 가용 영역에서 최다 투표
  • D. 원본 DB 인스턴스와 동일한 VPC 및 가용 영역에서
