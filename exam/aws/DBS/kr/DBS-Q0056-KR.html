Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 56
주제 #: 1

마케팅 회사는 Amazon DocumentDB를 사용하고 있으며 데이터베이스 감사 로그를 활성화해야 합니다. 데이터베이스 전문가는 수행된 모든 데이터 정의 언어(DDL) 문이 관리자에게 표시되도록 모니터링을 구성해야 합니다. 데이터베이스 전문가가 클러스터 파라미터 그룹에서 audit_logs 파라미터를 활성화로 설정했습니다.
관리자를 위해 데이터베이스 로그를 자동으로 수집하려면 데이터베이스 전문가는 어떻게 해야 합니까?

  • A. DocumentDB를 활성화하여 로그를 Amazon CloudWatch Logs로 내보냅니다. 최다 투표
  • B. DocumentDB를 활성화하여 로그를 AWS CloudTrail로 내보냅니다.
  • C. DocumentDB Events를 활성화하여 Amazon CloudWatch Logs로 로그 내보내기 최다 투표
  • D. download-db-log-file-portion 작업을 사용하여 로그를 다운로드하고 Amazon S3에 로그를 저장하도록 AWS Lambda 함수 구성
