Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 99
주제 #: 1

회사는 Amazon EC2 인스턴스에서 실행되는 Oracle Database Standard Edition을 다중 AZ가 있는 Oracle DB 인스턴스용 Amazon RDS로 마이그레이션해야 합니다.
데이터베이스는 지속적으로 실행되는 전자 상거래 웹사이트를 지원합니다. 회사는 최대 5분의 유지 관리 기간만 제공할 수 있습니다.
이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. EC2 인스턴스 및 RDS DB 인스턴스에서 Oracle RAC(Real Application Clusters)를 구성합니다. RAC 클러스터를 가리키도록 연결 문자열을 업데이트합니다. EC2 인스턴스와 RDS DB 인스턴스가 동기화되면 Amazon EC2에서 Amazon RDS로 장애 조치합니다.
  • B. Oracle Data Pump를 사용하여 EC2 인스턴스에서 Oracle 데이터베이스를 내보내고 Amazon RDS로 가져오기를 수행합니다. 전체 프로세스에 대해 애플리케이션을 중지합니다. 가져오기가 완료되면 데이터베이스 연결 문자열을 변경한 다음 애플리케이션을 다시 시작하십시오.
  • C. EC2 인스턴스를 소스로, RDS DB 인스턴스를 대상으로 AWS DMS를 구성합니다. 복제가 동기화되면 애플리케이션을 중지하고 데이터베이스 연결 문자열을 변경한 다음 애플리케이션을 다시 시작하십시오. 최다 투표
  • D. EC2 인스턴스를 소스로, RDS DB 인스턴스를 대상으로 AWS DataSync를 구성합니다. 복제가 동기화되면 애플리케이션을 중지하고 데이터베이스 연결 문자열을 변경한 다음 애플리케이션을 다시 시작하십시오.
