한 금융회사가 최근 포트폴리오 관리 솔루션을 출시했다. 애플리케이션의 백엔드는 MySQL과 호환되는 Amazon Aurora로 구동됩니다.
회사는 5분의 RTO와 5분의 RPO를 요구합니다. 데이터베이스 전문가는 최소한의 복제 지연으로 효율적인 재해 복구 솔루션을 구성해야 합니다.
데이터베이스 전문가는 이러한 요구 사항을 충족하기 위해 어떤 접근 방식을 취해야 합니까?

  • A. AWS Database Migration Service(AWS DMS)를 구성하고 다른 AWS 리전에 복제본을 생성합니다.
  • B. Amazon Aurora 글로벌 데이터베이스를 구성하고 다른 AWS 리전을 추가합니다. 최다 투표
  • C. binlog를 구성하고 다른 AWS 리전에서 복제본을 생성합니다.
  • D. 리전 간 읽기 전용 복제본을 구성합니다.
