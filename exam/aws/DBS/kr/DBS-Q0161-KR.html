Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 161
주제 #: 1

주식 시장을 분석하는 회사는 us-east-1 지역과 eu-west-2 지역에 두 개의 사무실을 두고 있습니다. 회사는 빠르고 정확한 업데이트를 제공할 수 있는 AWS 데이터베이스 솔루션을 구현하려고 합니다.
eu-west-2의 사무실에는 데이터를 표시하기 위한 복잡한 분석 쿼리가 포함된 대시보드가 ​​있습니다. 회사는 이러한 대시보드를 사용하여 구매 결정을 내리므로 대시보드는 1초 이내에 애플리케이션 데이터에 액세스할 수 있어야 합니다.
이러한 요구 사항을 충족하고 가장 최신 대시보드를 제공하는 솔루션은 무엇입니까?

  • A. eu-west-2의 읽기 전용 복제본 인스턴스와 함께 us-east-1에 Amazon RDS DB 인스턴스를 배포합니다. eu-west-2에서 Amazon ElastiCache 클러스터를 생성하여 읽기 전용 복제본의 데이터를 캐싱하여 대시보드를 생성합니다.
  • B. us-east-1에서 Amazon DynamoDB 전역 테이블을 사용하고 eu-west-2로 복제합니다. 다중 활성 복제를 사용하여 업데이트가 eu-west-2에 빠르게 전파되도록 합니다.
  • C. Amazon Aurora 글로벌 데이터베이스를 사용합니다. us-east-1에 기본 DB 클러스터를 배포합니다. eu-west-2에 보조 DB 클러스터를 배포합니다. 보조 클러스터에서 읽도록 대시보드 애플리케이션을 구성합니다. 최다 투표
  • D. eu-west-2에 읽기 전용 복제본 인스턴스가 있는 us-east-1에 Amazon RDS for MySQL DB 인스턴스를 배포합니다. 읽기 전용 복제본에서 읽도록 대시보드 애플리케이션을 구성합니다.
