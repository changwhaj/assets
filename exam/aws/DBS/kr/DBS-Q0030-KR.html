Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 30
주제 #: 1

회사가 며칠 동안 문을 닫을 계획입니다. 데이터베이스 전문가는 이 시간 동안 직원이 시스템에 액세스할 수 없도록 DB 인스턴스와 함께 모든 애플리케이션을 중지해야 합니다. 모든 데이터베이스는 MySQL용 Amazon RDS에서 실행됩니다.
데이터베이스 전문가는 모든 DB 인스턴스를 중지하는 스크립트를 작성하고 실행했습니다. 로그를 검토하던 중 데이터베이스 전문가는 읽기 전용 복제본이 있는 Amazon RDS DB 인스턴스가 중지되지 않았음을 발견했습니다.
데이터베이스 전문가는 이 문제를 해결하기 위해 스크립트를 어떻게 편집해야 합니까?

  • A. 읽기 전용 복제본을 중지하기 전에 원본 인스턴스를 중지합니다.
  • B. 해당 소스 인스턴스를 중지하기 전에 각 읽기 전용 복제본을 삭제합니다. 최다 투표
  • C. 원본 인스턴스를 중지하기 전에 읽기 전용 복제본을 중지합니다.
  • D. AWS CLI를 사용하여 각 읽기 전용 복제본과 원본 인스턴스를 동시에 중지합니다.
