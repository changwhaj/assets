시험 AWS 공인 데이터베이스 - 전문 주제 1 질문 160 토론

Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 160
주제 #: 1

전자상거래 회사가 온프레미스 MongoDB 데이터베이스를 Amazon DocumentDB(MongoDB와 호환)로 마이그레이션합니다. 마이그레이션 후 데이터베이스 전문가는 Amazon DocumentDB 클러스터에 대해 유휴 암호화가 켜져 있지 않다는 것을 알게 됩니다.
Amazon DocumentDB 클러스터에 유휴 암호화를 활성화하려면 데이터베이스 전문가가 무엇을 해야 합니까?

  • A. Amazon DocumentDB 클러스터의 스냅샷을 찍습니다. 암호화 옵션을 지정하면서 암호화되지 않은 스냅샷을 새 클러스터로 복원하고 AWS Key Management Service(AWS KMS) 키를 제공합니다. 최다 투표
  • B. AWS Management Console에서 Amazon DocumentDB 클러스터에 대한 암호화를 활성화합니다. 클러스터를 재부팅합니다.
  • C. --storage-encrypted 파라미터가 true로 설정된 modify-db-cluster 명령을 사용하여 Amazon DocumentDB 클러스터를 수정합니다.
  • D. Amazon DocumentDB 클러스터에 암호화된 새 인스턴스를 추가한 다음 클러스터에서 암호화되지 않은 인스턴스를 삭제합니다. 모든 인스턴스가 암호화될 때까지 반복합니다.

제안된 답변: B 🗳️
AWS Management Console을 사용하여 클러스터를 프로비저닝할 때 Amazon DocumentDB 클러스터에서 유휴 암호화를 활성화하거나 비활성화할 수 있습니다.
