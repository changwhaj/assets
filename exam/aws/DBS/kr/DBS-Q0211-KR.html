Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 211
주제 #: 1

한 회사에서 맞춤형 애플리케이션의 데이터 스토어로 사용할 새로운 Amazon RDS for MySQL 다중 AZ DB 인스턴스를 출시하고 있습니다. 특정 시점 복구를 비활성화한 상태에서 일련의 테스트를 거친 후 회사는 프로덕션 데이터를 저장하기 위해 DB 인스턴스를 사용하기 전에 특정 시점 복구를 다시 활성화해야 한다고 결정했습니다.
특정 시점 복구가 성공하려면 데이터베이스 전문가는 어떻게 해야 합니까?

  • A. DB 인스턴스에서 사용하는 DB 파라미터 그룹에서 바이너리 로깅을 활성화합니다.
  • B. DB 인스턴스를 수정하고 감사 로그를 Amazon CloudWatch Logs로 푸시하도록 활성화합니다.
  • C. DB 인스턴스 수정 및 백업 보존 기간 설정 최다 투표
  • D. 예약된 작업을 설정하여 수동 DB 인스턴스 스냅샷을 생성합니다.
