Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 171
주제 #: 1

데이터베이스 전문가가 처음으로 Amazon Neptune을 사용하여 테스트 그래프 데이터베이스를 시작합니다. 데이터베이스 전문가는 Amazon S3에 저장된 .csv 파일에서 수백만 행의 테스트 관찰을 삽입해야 합니다. 데이터베이스 전문가는 일련의 API 호출을 사용하여 Neptune DB 인스턴스에 데이터를 업로드했습니다.
데이터베이스 전문가가 데이터를 더 빨리 업로드할 수 있는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. Amazon Cognito가 CSV 파일을 호스팅하는 S3 버킷에 대해 Neptune DB 인스턴스를 인증하기 위해 적절한 AWS STS 토큰을 반환하는지 확인합니다.
  • B. 꼭지점과 가장자리가 적절한 헤더 열 형식으로 다른 .csv 파일에 지정되었는지 확인합니다. 최다 투표
  • C. AWS DMS를 사용하여 Amazon S3에서 Neptune 로더로 데이터를 이동합니다.
  • D. Neptune DB 인스턴스 내부에서 S3 URI를 컬링한 다음 addVertex 또는 addEdge 명령을 실행합니다.
  • E. Neptune DB 인스턴스에 대한 IAM 역할이 S3 버킷의 파일에 대한 액세스를 허용하는 적절한 권한으로 구성되었는지 확인합니다. 최다 투표
  • F. S3 VPC 엔드포인트를 생성하고 데이터베이스의 로더 엔드포인트에 HTTP POST를 발행합니다. 최다 투표
