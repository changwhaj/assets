Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 78
주제 #: 1

금융 회사는 마이그레이션 작업을 수용하기 위해 스토리지 용량이 큰 Amazon RDS MariaDB DB 인스턴스를 할당했습니다. 마이그레이션 후 회사는 인스턴스에서 원치 않는 데이터를 제거했습니다. 이 회사는 이제 스토리지를 축소하여 비용을 절감하려고 합니다. 솔루션은 생산에 미치는 영향이 최소화되고 다운타임이 거의 없어야 합니다.
이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. 이전 데이터베이스의 스냅샷을 생성하고 필요한 스토리지로 스냅샷을 복원합니다.
  • B. 필요한 스토리지가 있는 새 RDS DB 인스턴스를 생성하고 AWS DMS를 사용하여 이전 인스턴스에서 새 인스턴스로 데이터베이스를 이동합니다. 최다 투표
  • C. 기본 백업 및 복원을 사용하여 새 데이터베이스 생성
  • D. 새 읽기 복제본을 생성하고 기존 기본 복제본을 종료하여 기본 복제본으로 만듭니다.
