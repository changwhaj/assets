질문 #: 27
주제 #: 1

회사에서 신제품 출시를 앞두고 있으며 프로덕션 데이터에서 테스트 데이터베이스를 다시 생성해야 합니다.
이 회사는 Amazon Aurora MySQL DB 클러스터 에서 프로덕션 데이터베이스를 실행합니다 . 데이터베이스 전문가는 최소한의 관리 노력으로 가능한 한 빨리 이러한 테스트 데이터베이스를 생성하는 솔루션을 배포해야 합니다.
데이터베이스 전문가는 이러한 요구 사항을 충족하기 위해 무엇을 해야 합니까?

  • A. 프로덕션 클러스터에서 테스트 클러스터로 스냅샷 복원
  • B. 프로덕션 클러스터의 논리적 덤프를 생성하고 이를 새 테스트 클러스터로 복원합니다.
  • C. 데이터베이스 복제를 사용하여 프로덕션 클러스터의 복제본 생성 최다 투표
  • D. 프로덕션 클러스터에 읽기 전용 복제본을 추가하고 해당 노드를 테스트용으로 사용
