주제 #: 1

데이터베이스 전문가가 승차 호출 애플리케이션을 위한 새로운 데이터베이스 인프라를 설계하고 있습니다. 애플리케이션 데이터에는
모든 라이딩에 대한 GPS 좌표를 저장하는 라이딩 추적 시스템이 포함되어 있습니다. 실시간 통계 및 메타데이터 조회는 높은 처리량과 마이크로초 대기 시간으로 수행되어야 합니다. 데이터베이스는 최소한의 운영 오버헤드와 개발 노력으로 내결함성이 있어야 합니다.
가장 효율적인 방식으로 이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. MySQL용 Amazon RDS를 데이터베이스로 사용하고 Amazon ElastiCache를 사용합니다.
  • B. Amazon DynamoDB를 데이터베이스로 사용하고 DynamoDB Accelerator 사용 최다 투표
  • C. Amazon Aurora MySQL을 데이터베이스로 사용하고 Aurora의 버퍼 캐시를 사용합니다.
  • D. Amazon DynamoDB를 데이터베이스로 사용하고 Amazon API Gateway 사용
