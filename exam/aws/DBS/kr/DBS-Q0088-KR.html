Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 88
주제 #: 1

데이터베이스 전문가는 하나의 읽기 전용 복제본이 있는 Amazon RDS for MySQL DB 인스턴스를 담당합니다. DB 인스턴스와 읽기 전용 복제본은 기본 파라미터 그룹에 할당됩니다. 데이터베이스 팀은 현재 읽기 복제본에 대해 테스트 쿼리를 실행합니다. 데이터베이스 팀은 테스트에 도움이 되도록 읽기 전용 복제본에서만 액세스할 수 있는 추가 테이블을 읽기 전용 복제본에 생성하려고 합니다.
데이터베이스 팀이 테스트 테이블을 만들 수 있도록 하려면 데이터베이스 전문가는 무엇을 해야 합니까?

  • A. 읽기 전용 복제본에서 읽기 전용 모드를 비활성화하려면 AWS Support에 문의하십시오. 읽기 전용 복제본을 재부팅합니다. 읽기 전용 복제본에 연결하고 테이블을 생성합니다.
  • B. 읽기 전용 복제본의 기본 파라미터 그룹에서 read_only 파라미터를 false(read_only=0)로 변경합니다. 장애 조치 없이 재부팅을 수행합니다. 읽기 전용 복제본에 연결하고 local_only MySQL 옵션을 사용하여 테이블을 생성합니다.
  • C. 기본 파라미터 그룹에서 read_only 파라미터를 false(read_only=0)로 변경합니다. 읽기 전용 복제본을 재부팅합니다. 읽기 전용 복제본에 연결하고 테이블을 생성합니다.
  • D. 새 DB 파라미터 그룹을 생성합니다. read_only 매개변수를 false로 변경합니다(read_only=0). 읽기 복제본을 새 그룹과 연결합니다. 읽기 전용 복제본을 재부팅합니다. 읽기 전용 복제본에 연결하고 테이블을 생성합니다. 최다 투표
