Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 37
주제 #: 1

온라인 게임 회사는 Amazon DynamoDB를 데이터 스토어로 사용하여 새로운 게임을 출시할 계획입니다. 다음 사용 사례를 지원하도록 데이터베이스를 지정해야 합니다.
✑ 플레이어가 게임을 할 때마다 실시간으로 점수를 업데이트합니다.
✑ 특정 게임 세션에 대한 플레이어의 점수 세부 정보를 검색합니다.
데이터베이스 전문가가 DynamoDB 테이블을 구현하기로 결정합니다. 각 플레이어에는 고유한 user_id가 있고 각 게임에는 고유한 game_id가 있습니다.
DynamoDB 테이블에 권장되는 키 선택은 무엇입니까?

  • A. game_id를 파티션 키로 사용하여 글로벌 보조 인덱스 생성
  • B. user_id를 파티션 키로 사용하여 글로벌 보조 인덱스 생성
  • C. game_id를 파티션 키로, user_id를 정렬 키로 사용하여 복합 기본 키를 생성합니다.
  • D. user_id를 파티션 키로, game_id를 정렬 키로 사용하여 복합 기본 키를 생성합니다. 최다 투표
