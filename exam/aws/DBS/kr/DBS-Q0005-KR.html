Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 5
주제 #: 1

회사는 데이터가 삭제되기 전에 2일 동안만 데이터를 저장하면 되는 Amazon DynamoDB를 사용하는 대규모 트랜잭션 애플리케이션의 비용에 대해 우려하고 있습니다. 데이터베이스 전문가는 테이블을 살펴보면서 많은 데이터가 몇 달 전이고 응용 프로그램이 처음 배포된 시점으로 거슬러 올라간다는 것을 알게 됩니다.
데이터베이스 전문가는 전체 비용을 줄이기 위해 무엇을 할 수 있습니까?

  • A. 각 테이블에 새 속성을 생성하여 만료 시간을 추적하고 AWS Glue 변환을 생성하여 2일이 지난 항목을 삭제합니다.
  • B. 각 테이블에 새 속성을 생성하여 만료 시간을 추적하고 각 테이블에서 DynamoDB 스트림을 활성화합니다.
  • C. 각 테이블에 새 속성을 생성하여 만료 시간을 추적하고 각 테이블에서 TTL(Time to Live)을 활성화합니다. 최다 투표
  • D. AWS Data Pipeline을 사용하여 매일 Amazon S3로 데이터를 내보내는 Amazon CloudWatch Events 이벤트를 생성한 다음 Amazon DynamoDB 테이블을 자릅니다.
