시험 AWS 공인 데이터베이스 - 전문 분야 주제 1 질문 21 토론

Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 21
주제 #: 1

한 회사에서 MySQL 및 PostgreSQL용 Amazon RDS를 사용하여 여러 데이터베이스를 유지 관리합니다. 각 RDS 데이터베이스는 보존 기간이 기본값으로 설정된 로그 파일을 생성합니다. 회사는 이제 실시간 및 사후 분석을 용이하게 하기 위해 데이터베이스 로그를 중앙 집중식 저장소에서 최대 90일 동안 유지하도록 의무화했습니다.
데이터베이스 전문가가 최소한의 노력으로 이러한 요구 사항을 충족하려면 어떻게 해야 합니까?

  • A. RDS 데이터베이스에서 로그를 가져오고 로그 파일을 Amazon S3 버킷에 통합하는 AWS Lambda 함수를 생성합니다. 90일 후에 개체가 만료되도록 수명 주기 정책을 설정합니다.
  • B. Amazon CloudWatch Logs에 로그를 게시하도록 RDS 데이터베이스를 수정합니다. 90일 후에 이벤트가 만료되도록 각 로그 그룹에 대한 로그 보존 정책을 변경합니다. 최다 투표
  • C. 각 RDS 데이터베이스에 저장 프로시저를 작성하여 로그를 다운로드하고 로그 파일을 Amazon S3 버킷에 통합합니다. 90일 후에 개체가 만료되도록 수명 주기 정책을 설정합니다.
  • D. RDS 데이터베이스에서 로그를 다운로드하고 Amazon CloudWatch Logs에 로그를 게시하는 AWS Lambda 함수를 생성합니다. 90일 후에 이벤트가 만료되도록 로그 그룹에 대한 로그 보존 정책을 변경합니다.
