Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 255
주제 #: 1

은행은 개념 증명에서 MySQL DB 인스턴스용 Amazon RDS를 사용하고 있습니다. 데이터베이스 전문가는 이 개념 증명의 일부로 자동화된 데이터베이스 스냅샷 및 교차

리전 스냅샷 복사본을 평가하고 있습니다.
세 개의 자동 스냅샷을 성공적으로 검증한 후 데이터베이스 전문가는 네 번째 스냅샷이 생성되지 않았음을 알게 됩니다.
다음 중 스냅샷이 생성되지 않은 이유는 무엇입니까? (두 가지를 선택하세요.)

  • A. 이 DB 인스턴스에 대한 자동 스냅샷의 복사본이 동일한 AWS 리전 내에서 진행 중입니다. 최다 투표
  • B. 이 DB 인스턴스의 수동 스냅샷 복사는 DB 인스턴스 내의 특정 데이터베이스에 대해서만 진행 중입니다.
  • C. RDS 유지 관리 기간이 지정되지 않았습니다.
  • D. DB 인스턴스가 STORAGE_FULL 상태입니다. 최다 투표
  • E. RDS 이벤트 알림이 활성화되지 않았습니다.
