Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 117
주제 #: 1

한 회사가 여러 가용 영역(AZ)에 배포된 Amazon EC2 인스턴스에서 웹 사이트를 실행하고 있습니다. 사이트는 범용 SSD(gp2) 스토리지가 있는 Amazon RDS for MySQL 다중 AZ DB 인스턴스에서 초당 많은 수의 반복 읽기 및 쓰기를 수행합니다. 포괄적인 테스트 및 분석 후 데이터베이스 전문가는 DB 인스턴스에서 높은 읽기 지연 시간과 높은 CPU 사용률을 발견했습니다.
애플리케이션을 변경하지 않고 이 문제를 해결하려면 데이터베이스 전문가가 어떤 접근 방식을 취해야 합니까?

  • A. 샤딩을 구현하여 MySQL용 여러 RDS 데이터베이스에 로드를 분산합니다.
  • B. 프로비저닝된 IOPS(PIOPS) 스토리지와 동일한 RDS for MySQL 인스턴스 클래스를 사용합니다.
  • C. RDS for MySQL 읽기 전용 복제본을 추가합니다.
  • D. RDS for MySQL 데이터베이스 클래스를 더 큰 크기로 수정하고 프로비저닝된 IOPS(PIOPS)를 구현합니다.
