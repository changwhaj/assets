Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 77
주제 #: 1

데이터베이스 전문가가 AWS DMS를 사용하여 PostgreSQL용 Amazon RDS로 온프레미스 Microsoft SQL Server 애플리케이션 데이터베이스를 마이그레이션하고 있습니다. 애플리케이션은 RDS DB 인스턴스가 활성화될 때 가동 중지 시간을 최소화해야 합니다.
데이터베이스 전문가는 마이그레이션을 활성화하기 위해 어떤 변경을 수행해야 합니까?

  • A. 지속적인 변경 데이터 캡처(CDC)와 함께 AWS DMS 전체 로드의 소스 역할을 하도록 온프레미스 애플리케이션 데이터베이스를 구성합니다. 최다 투표
  • B. 전체 로드와 지속적인 변경 데이터 캡처(CDC)를 모두 허용하도록 AWS DMS 복제 인스턴스를 구성합니다.
  • C. 지속적인 변경 데이터 캡처(CDC)를 허용하기 위해 전체 로그를 생성하도록 AWS DMS 작업을 구성합니다.
  • D. 지속적인 변경 데이터 캡처(CDC)를 허용하기 위해 양방향 통신을 허용하도록 AWS DMS 연결을 구성합니다.
