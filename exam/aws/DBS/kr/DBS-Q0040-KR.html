Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 40
주제 #: 1

데이터베이스 전문가가 온프레미스 Oracle 데이터베이스를 Amazon Aurora PostgreSQL로 마이그레이션했습니다. 스키마와 데이터가 성공적으로 마이그레이션되었습니다.
온프레미스 데이터베이스 서버는 또한 데이터 제거 및 데이터 내보내기 생성을 포함한 작업을 수행하기 위해 Python으로 작성된 데이터베이스 유지 관리 cron 작업을 실행하는 데 사용되었습니다. 이러한 작업에 대한 로그는 대부분의 경우 작업이 5분 이내에 완료되었지만 일부 작업은 완료하는 데 최대 10분이 걸렸음을 보여줍니다. 이러한 유지 관리 작업은 Aurora PostgreSQL에 대해 설정해야 합니다.
데이터베이스 전문가는 유지 관리를 최소화하고 고가용성을 제공하도록 이러한 작업을 어떻게 예약할 수 있습니까?

  • A. Amazon EC2 인스턴스에서 cron 작업을 생성하여 필요한 일정에 따라 유지 관리 작업을 실행합니다.
  • B. Aurora 호스트에 연결하고 cron 작업을 생성하여 필요한 일정에 따라 유지 관리 작업을 실행합니다.
  • C. 유지 관리 작업을 실행하고 Amazon CloudWatch Events로 예약하는 AWS Lambda 함수를 생성합니다. 최다 투표
  • D. Amazon CloudWatch 작업 예약 플러그인을 사용하여 유지 관리 작업을 생성합니다.
