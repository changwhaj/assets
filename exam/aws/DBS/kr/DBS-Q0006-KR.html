시험 AWS 공인 데이터베이스 - 전문 분야 주제 1 질문 6 토론

Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 6
주제 #: 1

회사에는 데이터베이스 종료, 삭제, 생성 및 백업을 포함하여 데이터베이스 수명 동안 발생하는 다양한 데이터베이스 작업을 추적하는 온프레미스 시스템이 있습니다.
이 회사는 최근 두 개의 데이터베이스를 Amazon RDS로 옮겼으며 이러한 요구 사항을 충족할 솔루션을 찾고 있습니다. 데이터는 회사 내의 다른 시스템에서 사용될 수 있습니다.
최소한의 노력으로 이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. Amazon RDS에서 추적해야 하는 작업으로 Amazon CloudWatch Events 규칙을 생성합니다. 이러한 규칙에 따라 작동하고 추적 시스템에 출력을 쓰는 AWS Lambda 함수를 생성합니다.
  • B. AWS CloudTrail API 호출에서 트리거할 AWS Lambda 함수를 생성합니다. 특정 RDS API 호출을 필터링하고 추적 시스템에 출력을 씁니다.
  • C. RDS 이벤트 구독을 생성합니다. 추적 시스템이 특정 RDS 이벤트 시스템 알림을 구독하도록 합니다.
  • D. Amazon Kinesis Data Firehose에 RDS 로그를 씁니다. 이러한 규칙에 따라 작동하고 추적 시스템에 출력을 쓰는 AWS Lambda 함수를 생성합니다.
