Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 41
주제 #: 1

한 회사에 크기가 200GB이고 RPO가 6시간인 Amazon RDS 다중 AZ DB 인스턴스가 있습니다. 회사의 재해 복구 정책을 충족하려면 데이터베이스 백업을 다른 리전에 복사해야 합니다. 이 회사는 솔루션이 비용 효율적이고 운영상 효율적이어야 한다고 요구합니다.
데이터베이스 전문가가 데이터베이스 백업을 다른 지역으로 복사하려면 어떻게 해야 합니까?

  • A. Amazon RDS 자동 스냅샷을 사용하고 AWS Lambda를 사용하여 스냅샷을 다른 지역으로 복사
  • B. 6시간마다 Amazon RDS 자동 스냅샷을 사용하고 Amazon S3 교차 리전 복제를 사용하여 스냅샷을 다른 리전으로 복사합니다.
  • C. 6시간마다 Amazon RDS 스냅샷을 생성하는 AWS Lambda 함수를 생성하고 두 번째 Lambda 함수를 사용하여 스냅샷을 다른 리전에 복사합니다.
  • D. 다른 리전에서 Amazon RDS에 대한 리전 간 읽기 전용 복제본을 생성하고 읽기 전용 복제본의 자동 스냅샷을 생성합니다.
