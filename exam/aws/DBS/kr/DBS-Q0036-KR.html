Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 36
주제 #: 1

한 회사에서 읽기 전용 워크로드 조정을 위해 Aurora 복제본과 함께 Amazon을 사용하고 있습니다. 데이터베이스 전문가는 각 애플리케이션이 항상 전용 복제본에 연결되도록 두 개의 읽기 전용 애플리케이션을 분할해야 합니다. 데이터베이스 전문가는 읽기 전용 애플리케이션에 대한 로드 밸런싱 및 고가용성을 구현하려고 합니다.
어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. 각 복제본에 대해 특정 인스턴스 엔드포인트를 사용하고 각 읽기 전용 애플리케이션 연결 문자열에 인스턴스 엔드포인트를 추가합니다.
  • B. 읽기 전용 워크로드 애플리케이션 모두에 대해 리더 엔드포인트를 사용합니다.
  • C. 하나의 읽기 전용 애플리케이션에는 리더 엔드포인트를 사용하고 다른 읽기 전용 애플리케이션에는 인스턴스 엔드포인트를 사용합니다.
  • D. 두 개의 읽기 전용 응용 프로그램에 대해 사용자 지정 끝점을 사용합니다. 최다 투표
