시험 AWS 공인 데이터베이스 - 전문 분야 주제 1 질문 14 토론

Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 14
주제 #: 1

회사는 Amazon Redshift 클러스터에서 중요한 비즈니스 데이터를 호스팅하고 있습니다. 데이터의 민감한 특성으로 인해 클러스터는 AWS
KMS를 사용하여 유휴 상태에서 암호화됩니다. 재해 복구 요구 사항의 일부로 회사는 Amazon Redshift 스냅샷을 다른 리전에 복사해야 합니다.
재해 복구 요구 사항을 충족하려면 AWS Management Console에서 어떤 단계를 수행해야 합니까?

  • A. 소스 리전에서 새 KMS 고객 마스터 키를 생성합니다. 대상 리전으로 전환하고 Amazon Redshift 교차 리전 스냅샷을 활성화하고 원본 리전의 KMS 키를 사용합니다.
  • B. KMS 키에 대한 액세스 권한이 있는 새 IAM 역할을 생성합니다. 새 IAM 역할을 사용하여 Amazon Redshift 교차 리전 복제를 활성화하고 원본 리전의 KMS 키를 사용합니다.
  • C. 원본 리전에서 Amazon Redshift 교차 리전 스냅샷을 활성화하고 스냅샷 복사 권한을 생성하고 대상 리전에서 KMS 키를 사용합니다. 최다 투표
  • D. 대상 리전에서 새 KMS 고객 마스터 키를 생성하고 새 KMS 키에 액세스할 수 있는 새 IAM 역할을 생성합니다. 원본 리전에서 Amazon Redshift 교차 리전 복제를 활성화하고 대상 리전의 KMS 키를 사용합니다.
