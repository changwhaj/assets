질문 #: 24
주제 #: 1

금융 회사의 보안 팀은 3주 전에 발생한 내부 보안 침해에 대해 통보 받았습니다. 데이터베이스 전문가는 보안 팀이 모니터링 및 알림에 사용할 프로덕션 Amazon Aurora PostgreSQL 클러스터에서 감사 로그 생성을 시작해야 합니다. 보안 팀은 Aurora DB 클러스터 외부에서 실시간 경고 및 모니터링을 수행해야 하며 클러스터가 선택한 솔루션에 암호화된 파일을 푸시하도록 하려고 합니다.
이러한 요구 사항을 충족하는 접근 방식은 무엇입니까?

  • A. pg_audit를 사용하여 감사 로그를 생성하고 보안 팀에 로그를 보냅니다.
  • B. AWS CloudTrail을 사용하여 DB 클러스터를 감사하면 보안 팀이 Amazon S3에서 데이터를 가져옵니다.
  • C. 데이터베이스 활동 스트림을 설정하고 Amazon Kinesis의 데이터 스트림을 소비자 애플리케이션에 연결합니다. 최다 투표
  • D. 세부 정보 로깅을 켜고 보안 팀을 위해 로그를 덤프할 일정을 설정합니다.
