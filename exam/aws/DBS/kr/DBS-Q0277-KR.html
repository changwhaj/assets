Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 277
주제 #: 1

한 회사가 최근 지난주 인기가 높아진 모바일 앱을 출시했습니다. 이 회사는 클라우드에서 개발을 시작했으며 처음에는 모바일 앱 개발 중에 보안 모범 사례를 따르지 않았습니다. 모바일 앱은 고객에게 플랫폼을 익명으로 사용할 수 있는 기능을 제공합니다. 플랫폼 설계자는 VPC에서 Redis용 Amazon ElastiCache를 사용하여 고객을 위한 세션 선호도(고정 세션) 및 쿠키를 관리합니다.

회사의 보안 팀은 이제 모든 트래픽에 대해 전송 중 암호화 및 미사용 암호화를 의무화합니다. 데이터베이스 전문가는 AWS CLI를 사용하여 이 의무를 준수하고 있습니다.

데이터베이스 전문가는 이러한 요구 사항을 충족하기 위해 어떤 단계 조합을 수행해야 합니까? (3개를 선택하세요.)

  • A. create-snapshot 명령을 사용하여 기존 Redis 복제 그룹의 수동 백업을 생성합니다. create-replication-group 명령을 사용하여 백업에서 복원
  • B. 새 Redis 복제 그룹에서 --transit-encryption-enabled 파라미터 사용
  • C. 기존 Redis 복제 그룹에서 --at-rest-encryption-enabled 파라미터 사용
  • D. 기존 Redis 복제 그룹에서 --transit-encryption-enabled 파라미터 사용
  • E. 새 Redis 복제 그룹에서 --at-rest-encryption-enabled 매개변수 사용
  • F. CreateBackupSelection 명령을 사용하여 기존 Redis 복제 그룹의 수동 백업을 생성합니다. StartRestoreJob 명령을 사용하여 백업에서 복원
