Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 93
주제 #: 1

소프트웨어 개발 회사는 개발 및 보고를 포함한 여러 사용 사례에 Amazon Aurora MySQL DB 클러스터를 사용하고 있습니다. 이러한 사용 사례는 Aurora DB 클러스터에 대해 예측할 수 없고 다양한 수요를 발생시키며 지연 시간이 일시적으로 급증할 수 있습니다. 시스템 사용자는 일주일 내내 산발적으로 임시 쿼리를 실행합니다. 비용은 회사의 주요 관심사이며 상당한 재작업이 필요하지 않은 솔루션이 필요합니다.
어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. 개발 및 보고를 위해 새로운 Aurora Serverless DB 클러스터를 생성한 다음 이 새로운 DB 클러스터로 마이그레이션합니다. 최다 투표
  • B. DB 클러스터 중 하나를 더 큰 크기로 업그레이드하고 이 더 큰 DB 클러스터에서 개발 및 보고 활동을 통합합니다.
  • C. 기존 DB 클러스터를 사용하고 스케줄링 도구를 사용하여 정기적으로 데이터베이스를 중지/시작합니다.
  • D. DB 클러스터를 버스트 가능한 인스턴스 패밀리로 변경합니다.
