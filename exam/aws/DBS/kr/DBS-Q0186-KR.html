Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 186
주제 #: 1

금융 서비스 회사는 중요한 애플리케이션을 위해 온프레미스 MySQL 데이터베이스를 실행합니다. 회사는 현재 데이터베이스 재해 복구(DR) 솔루션에 만족하지 않습니다. 애플리케이션은 데이터베이스가 DR 시설로 장애 조치될 때마다 상당한 가동 중지 시간을 경험합니다. 또한 보고서가 동일한 데이터베이스에서 처리될 때 응용 프로그램의 응답 시간이 느려집니다. DR 상황에서 가동 중지 시간을 최소화하기 위해 회사는 데이터베이스를 AWS로 마이그레이션하기로 결정했습니다. 이 회사는 가용성이 높고 가장 비용 효율적인 솔루션이 필요합니다.
어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. Amazon RDS for MySQL 다중 AZ DB 인스턴스를 생성하고 다른 가용 영역에서 읽기 전용 복제본을 구성합니다. 복제본 인스턴스 엔드포인트를 참조하도록 애플리케이션을 구성하고 기본 DB 인스턴스 엔드포인트를 참조하도록 쿼리를 보고합니다.
  • B. Amazon RDS for MySQL 다중 AZ DB 인스턴스를 생성하고 다른 가용 영역에서 읽기 전용 복제본을 구성합니다. 기본 DB 인스턴스 엔드포인트를 참조하도록 애플리케이션을 구성하고 복제본 인스턴스 엔드포인트를 참조하도록 쿼리를 보고합니다.
  • C. Amazon Aurora DB 클러스터를 생성하고 다른 가용 영역에서 Aurora 복제본을 구성합니다. 클러스터 엔드포인트를 참조하도록 애플리케이션을 구성하고 리더 엔드포인트를 참조하도록 쿼리를 보고합니다. 최다 투표
  • D. Amazon Aurora DB 클러스터를 생성하고 다른 가용 영역에서 Aurora 복제본을 구성합니다. 기본 DB 인스턴스 엔드포인트를 참조하도록 애플리케이션을 구성하고 복제본 인스턴스 엔드포인트를 참조하도록 쿼리를 보고합니다.
