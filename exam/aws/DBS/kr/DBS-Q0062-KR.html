Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 62
주제 #: 1

회사의 보안 부서는 내부 사용자가 회사 Active Directory(AD) 자격 증명을 사용하여 SQL Server DB 인스턴스용 기존 Amazon RDS에 연결해야 한다고 명시하는 새로운 요구 사항을 설정했습니다. 데이터베이스 전문가는 이 요구 사항을 충족하는 데 필요한 수정 작업을 수행해야 합니다.
데이터베이스 전문가는 어떤 조합의 조치를 취해야 합니까? (3개를 선택하세요.)

  • A. RDS SQL Server DB 인스턴스에서 TDE(투명한 데이터 암호화)를 비활성화합니다.
  • B. Windows 인증에 디렉터리를 사용하도록 RDS SQL Server DB 인스턴스를 수정합니다. 적절한 새 로그인을 만듭니다. 최다 투표
  • C. AWS Management Console을 사용하여 AWS Managed Microsoft AD를 생성합니다. 회사 AD와 트러스트 관계를 만듭니다. 최다 투표
  • D. RDS SQL Server DB 인스턴스를 중지하고 Windows 인증에 디렉터리를 사용하도록 수정한 다음 다시 시작합니다. 적절한 새 로그인을 만듭니다.
  • E. AWS Management Console을 사용하여 AD 커넥터를 생성합니다. 회사 AD와 트러스트 관계를 만듭니다.
  • F. AWS Managed Microsoft AD 도메인 컨트롤러 보안 그룹을 구성합니다. 최다 투표
