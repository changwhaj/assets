Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 242
주제 #: 1

회사에서 Amazon Redshift를 데이터 웨어하우스 솔루션으로 사용하고 있습니다. Redshift 클러스터는 다음 유형의 워크로드를 처리합니다.
✑ Amazon Kinesis Data Firehose를 통한 실시간 삽입
✑ Amazon S3의 COPY 명령을 통한 대량 삽입
✑ SQL 쿼리를 통한 분석
최근 클러스터에 성능 문제가 발생하기 시작했습니다.
클러스터의 성능을 향상시키기 위해 데이터베이스 전문가가 취해야 할 조치 조합은 무엇입니까? (3개를 선택하세요.)

  • A. 버퍼 크기가 큰 Amazon S3로 데이터를 스트리밍하고 COPY 명령을 사용하여 데이터를 Amazon Redshift로 로드하도록 Kinesis Data Firehose 전송 스트림을 수정합니다.
  • B. 데이터를 영구 테이블로 로드하기 전에 실시간 데이터를 Redshift 임시 테이블로 스트리밍합니다. 최다 투표
  • C. 대량 삽입의 경우 Amazon S3의 입력 파일을 여러 파일로 분할하여 Amazon Redshift의 조각 수와 일치시킵니다. 그런 다음 COPY 명령을 사용하여 데이터를 Amazon Redshift로 로드합니다. 최다 투표
  • D. 대량 삽입의 경우 COPY 명령에서 병렬 매개변수를 사용하여 다중 스레딩을 활성화합니다.
  • E. 정렬 키를 사용하도록 분석 SQL 쿼리를 최적화합니다. 최다 투표
  • F. 분석 SQL 쿼리에서 임시 테이블을 사용하지 마십시오.
