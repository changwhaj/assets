Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 87
주제 #: 1

회사에서 데이터베이스 라이선스를 갱신해야 합니다. 이 회사는 80TB 트랜잭션 데이터베이스 시스템을 온프레미스에서 AWS 클라우드로 마이그레이션하려고 합니다.
마이그레이션은 다운스트림 데이터베이스 애플리케이션에서 가동 중지 시간을 최소화해야 합니다. 회사의 네트워크 인프라에는 다른 애플리케이션과 공유되는 네트워크 대역폭이 제한되어 있습니다.
데이터베이스 전문가는 시기적절한 마이그레이션을 위해 어떤 솔루션을 사용해야 합니까?

  • A. 소스 데이터베이스의 전체 백업을 AWS Snowball Edge 어플라이언스로 수행하고 Amazon S3에 로드하도록 배송합니다. AWS DMS를 사용하여 원본 데이터베이스에서 Amazon S3로 변경 데이터 캡처(CDC) 데이터를 마이그레이션합니다. 두 번째 AWS DMS 작업을 사용하여 모든 S3 데이터를 대상 데이터베이스로 마이그레이션합니다. 최다 투표
  • B. 소스 데이터베이스의 전체 백업을 AWS Snowball Edge 어플라이언스로 수행하고 Amazon S3에 로드하도록 배송합니다. 다른 Snowball Edge 어플라이언스로 배송할 소스 데이터베이스의 증분 백업을 주기적으로 수행하여 소스에서 대상 데이터베이스로 변경 데이터 캡처(CDC) 데이터 동기화를 처리합니다.
  • C. AWS DMS를 사용하여 기본 연결에 인터넷을 사용하여 VPN 터널을 통해 원본 데이터베이스의 전체 로드를 마이그레이션합니다. AWS DMS가 소스에서 대상 데이터베이스로 변경 데이터 캡처(CDC) 데이터 동기화를 처리하도록 허용합니다.
  • D. AWS Schema Conversion Tool(AWS SCT)을 사용하여 기본 연결에 인터넷을 사용하는 VPN 터널을 통해 원본 데이터베이스의 전체 로드를 마이그레이션합니다. AWS SCT가 원본에서 대상 데이터베이스로 변경 데이터 캡처(CDC) 데이터 동기화를 처리하도록 허용합니다.
