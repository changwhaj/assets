Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 264
주제 #: 1

회사에서 AWS Database Migration Service(AWS DMS)를 사용하여 온프레미스 Oracle 데이터베이스를 PostgreSQL용 Amazon RDS로 마이그레이션하려고 합니다. 데이터베이스 전문가는 AWS DMS 작업에서 마이그레이션 작업 설정 및 데이터 유형 변환을 평가해야 합니다.

데이터베이스 전문가는 최적의 마이그레이션 방법을 식별하기 위해 무엇을 해야 합니까?

  • A. AWS Schema Conversion Tool(AWS SCT) 데이터베이스 마이그레이션 평가 보고서 사용 최다 투표
  • B. AWS Schema Conversion Tool(AWS SCT) 다중 서버 평가자를 사용합니다.
  • C. AWS DMS 사전 마이그레이션 평가 사용 최다 투표
  • D. AWS DMS 데이터 검증 도구 사용
