Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 45
주제 #: 1

한 전자상거래 회사는 데이터베이스 전문가에게 Amazon Aurora에서 실행되는 핵심 프로덕션 데이터베이스에서 가져올 중요한 비즈니스 지표를 시각화하는 보고 대시보드를 만드는 작업을 맡겼습니다. 대시보드에서 읽은 데이터는 업데이트 후 100밀리초 이내에 사용할 수 있어야 합니다.
데이터베이스 전문가는 Aurora DB 클러스터의 현재 구성을 검토하고 비용 효율적인 솔루션을 개발해야 합니다. 이 솔루션은 DB 클러스터의 쓰기 가용성 및 성능에 영향을 미치지 않으면서 보고 대시보드에서 예측할 수 없는 읽기 워크로드를 수용해야 합니다.
어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. 수요에 따라 자동으로 확장할 수 있도록 DB 클러스터에서 서버리스 옵션을 켭니다.
  • B. 새 애플리케이션 팀을 위해 기존 DB 클러스터의 복제본을 프로비저닝합니다.
  • C. 새 워크로드에 대해 별도의 DB 클러스터를 생성하고 원본 DB 클러스터에서 새로 고침하고 AWS DMS CDC(변경 데이터 캡처)를 사용하여 지속적인 복제를 설정합니다.
  • D. DB 클러스터에 자동 조정 정책을 추가하여 CPU 사용량에 따라 Aurora 복제본을 클러스터에 추가합니다. 최다 투표
