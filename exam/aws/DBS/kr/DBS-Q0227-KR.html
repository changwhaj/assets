Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 227
주제 #: 1

데이터베이스 전문가는 Amazon Aurora DB 클러스터가 항상 사용 가능한 최신 마이너 버전으로 자동 업그레이드되기를 원합니다. 사용 가능한 새로운 마이너 버전이 있음을 알게 된 데이터베이스 전문가는 자동 마이너 버전 업데이트를 활성화하는 AWS CLI 명령을 발행합니다. 명령이 성공적으로 실행되지만 Aurora DB 클러스터를 확인하면 Aurora 버전이 업데이트되지 않은 것으로 나타납니다.
그 이유는 무엇입니까? (두 가지를 선택하세요.)

  • A. 새 마이너 버전은 아직 기본 버전으로 지정되지 않았으며 수동 업그레이드가 필요합니다. 최다 투표
  • B. AWS CLI를 사용한 자동 업그레이드 구성은 지원되지 않습니다. 이는 AWS Management Console을 사용하여 명시적으로 활성화해야 합니다.
  • C. 마이너 버전 업그레이드를 적용하려면 충분한 여유 공간이 필요합니다.
  • D. AWS CLI 명령에 apply-immediately 파라미터가 포함되지 않았습니다. 최다 투표
  • E. Aurora가 새 마이너 버전에서 주요 변경 사항을 감지하고 자동으로 업그레이드를 거부했습니다.
