Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 230
주제 #: 1

한 회사의 데이터베이스 전문가는 변경 데이터 캡처(CDC)를 위한 AWS Database Migration Service(AWS DMS) 작업을 구현하여 온프레미스 Oracle 데이터베이스에서 Amazon S3로 데이터를 복제합니다. 회사 애플리케이션의 사용량이 증가하면 데이터베이스 전문가는
CDC에 몇 시간의 대기 시간이 있음을 발견했습니다.
어떤 솔루션이 이 지연 시간을 줄여줄까요? (2개를 선택하세요.)

  • A. 전체 LOB(대형 이진 개체) 모드에서 실행되도록 DMS 작업을 구성합니다.
  • B. 제한된 LOB(Large Binary Object) 모드에서 실행되도록 DMS 작업을 구성합니다. 최다 투표
  • C. 다중 AZ 복제 인스턴스를 생성합니다.
  • D. 공통 트랜잭션에 참여하는 테이블 집합에 대해 여러 복제 인스턴스를 생성하여 테이블을 병렬로 로드합니다.
  • E. 공통 트랜잭션에 참여하지 않는 테이블 세트에 대해 여러 DMS 작업을 생성하여 테이블을 병렬로 복제합니다. 최다 투표

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 230
Topic #: 1

A company's database specialist implements an AWS Database Migration Service (AWS DMS) task for change data capture (CDC) to replicate data from an on- premises Oracle database to Amazon S3. When usage of the company's application increases, the database specialist notices multiple hours of latency with the
CDC.
Which solutions will reduce this latency? (Choose two.)

  • A. Configure the DMS task to run in full large binary object (LOB) mode.
  • B. Configure the DMS task to run in limited large binary object (LOB) mode. Most Voted
  • C. Create a Multi-AZ replication instance.
  • D. Load tables in parallel by creating multiple replication instances for sets of tables that participate in common transactions.
  • E. Replicate tables in parallel by creating multiple DMS tasks for sets of tables that do not participate in common transactions. Most Voted
Show Suggested Answer Hide Answer
