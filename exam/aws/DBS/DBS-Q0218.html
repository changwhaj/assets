Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 218
주제 #: 1

한 회사는 Amazon Aurora MySQL을 많은 애플리케이션의 기본 데이터베이스 엔진으로 사용합니다. 데이터베이스 전문가는 데이터베이스에 대한 사용자 연결에 대한 정보를 회사에 제공하기 위해 대시보드를 만들어야 합니다. 규정 준수 요구 사항에 따라 회사는 모든 연결 로그를 최소 7년 동안 보관해야 합니다.
이러한 요구 사항을 가장 비용 효율적으로 충족하는 솔루션은 무엇입니까?

  • A. Aurora 클러스터에서 고급 감사를 활성화하여 CONNECT 이벤트를 기록합니다. Amazon EventBridge(Amazon CloudWatch Events) 예약 이벤트에 의해 호출되는 AWS Lambda 함수를 사용하여 Amazon CloudWatch에서 Amazon S3로 감사 로그를 내보냅니다. Amazon QuickSight를 사용하여 대시보드를 구축하십시오. 최다 투표
  • B. DescriptionEvents API 작업을 사용하여 AWS Cloud Trail을 통해 Aurora 클러스터에 대한 연결 시도를 캡처합니다. CloudTrail 추적을 생성하여 연결 로그를 Amazon S3로 내보냅니다. Amazon QuickSight를 사용하여 대시보드를 구축하십시오.
  • C. Aurora 클러스터에 대한 데이터베이스 활동 스트림을 시작합니다. 활동 레코드를 Amazon Kinesis 데이터 스트림으로 푸시합니다. AWS Lambda를 사용하여 동적 대시보드를 구축하세요.
  • D. Aurora DB 인스턴스에 대한 DatabaseConnections 지표를 Amazon CloudWatch에 게시합니다. CloudWatch 대시보드를 사용하여 대시보드를 구축합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 218
Topic #: 1

A company uses Amazon Aurora MySQL as the primary database engine for many of its applications. A database specialist must create a dashboard to provide the company with information about user connections to databases. According to compliance requirements, the company must retain all connection logs for at least 7 years.
Which solution will meet these requirements MOST cost-effectively?

  • A. Enable advanced auditing on the Aurora cluster to log CONNECT events. Export audit logs from Amazon CloudWatch to Amazon S3 by using an AWS Lambda function that is invoked by an Amazon EventBridge (Amazon CloudWatch Events) scheduled event. Build a dashboard by using Amazon QuickSight. Most Voted
  • B. Capture connection attempts to the Aurora cluster with AWS Cloud Trail by using the DescribeEvents API operation. Create a CloudTrail trail to export connection logs to Amazon S3. Build a dashboard by using Amazon QuickSight.
  • C. Start a database activity stream for the Aurora cluster. Push the activity records to an Amazon Kinesis data stream. Build a dynamic dashboard by using AWS Lambda.
  • D. Publish the DatabaseConnections metric for the Aurora DB instances to Amazon CloudWatch. Build a dashboard by using CloudWatch dashboards.

Suggested Answer: C 🗳️
