Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 181
주제 #: 1

금융 서비스 회사에는 Amazon Aurora PostgreSQL DB 클러스터를 사용하는 애플리케이션이 AWS에 배포되어 있습니다. 최근 감사에서는 데이터베이스 관리자 활동이 포함된 로그 파일이 없는 것으로 나타났습니다. 데이터베이스 전문가는 데이터베이스 액세스 및 활동 로그를 제공하는 솔루션을 추천해야 합니다. 솔루션은 최소한의 노력을 기울여야 하며 성능에 최소한의 영향을 주어야 합니다.
데이터베이스 전문가는 어떤 솔루션을 추천해야 합니까?

  • A. 동기 모드의 데이터베이스에서 Aurora 데이터베이스 활동 스트림을 활성화합니다. Amazon Kinesis 데이터 스트림을 Kinesis Data Firehose에 연결합니다. Kinesis Data Firehose 대상을 Amazon S3 버킷으로 설정합니다.
  • B. 데이터베이스가 실행되는 지역에 AWS CloudTrail 추적을 생성합니다. 데이터베이스 활동 로그를 추적과 연결합니다.
  • C. 비동기 모드로 데이터베이스에서 Aurora 데이터베이스 활동 스트림을 활성화합니다. Amazon Kinesis 데이터 스트림을 Kinesis Data Firehose에 연결합니다. Firehose 대상을 Amazon S3 버킷으로 설정합니다. 최다 투표
  • D. 배스천 호스트를 통해서만 DB 클러스터에 대한 연결을 허용합니다. 배스천 호스트 및 애플리케이션 서버에 대한 데이터베이스 액세스를 제한합니다. CloudWatch Logs 에이전트를 사용하여 배스천 호스트 로그를 Amazon CloudWatch Logs로 푸시합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 181
Topic #: 1

A financial services company has an application deployed on AWS that uses an Amazon Aurora PostgreSQL DB cluster. A recent audit showed that no log files contained database administrator activity. A database specialist needs to recommend a solution to provide database access and activity logs. The solution should use the least amount of effort and have a minimal impact on performance.
Which solution should the database specialist recommend?

  • A. Enable Aurora Database Activity Streams on the database in synchronous mode. Connect the Amazon Kinesis data stream to Kinesis Data Firehose. Set the Kinesis Data Firehose destination to an Amazon S3 bucket.
  • B. Create an AWS CloudTrail trail in the Region where the database runs. Associate the database activity logs with the trail.
  • C. Enable Aurora Database Activity Streams on the database in asynchronous mode. Connect the Amazon Kinesis data stream to Kinesis Data Firehose. Set the Firehose destination to an Amazon S3 bucket. Most Voted
  • D. Allow connections to the DB cluster through a bastion host only. Restrict database access to the bastion host and application servers. Push the bastion host logs to Amazon CloudWatch Logs using the CloudWatch Logs agent.
