From here: https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/USER_WorkingWithAutomatedBackups.html Automated backups follow these rules: Your DB instance must be in the AVAILABLE state for automated backups to occur. Automated backups don't occur while your DB instance is in a state other than AVAILABLE, for example STORAGE_FULL. Automated backups don't occur while a DB snapshot copy is running in the same AWS Region for the same DB instance.
