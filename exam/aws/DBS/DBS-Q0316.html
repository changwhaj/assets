Amazon의 AWS 공인 데이터베이스 - 전문 분야 시험 문제
질문 #: 316
주제 #: 1

미국에 위치한 한 회사가 아시아에서 사업을 확장하려고 합니다. us-west-2 지역의 회사 데이터는 Amazon DynamoD8 테이블에 저장됩니다. ap-northeast-1 지역의 회사 개발 팀은 us-west-2의 프로덕션 데이터 사본을 사용하여 UAT(사용자 승인 테스트) 및 기타 여러 성능 타당성 테스트를 수행해야 합니다. 실시간으로 업데이트되는 데이터에 대해서는 타당성 테스트를 실행할 필요가 없습니다.

us-west-2에서 ap-northeast-1까지 가장 비용 효과적으로 데이터를 사용할 수 있게 해주는 솔루션은 무엇입니까?

  • A. ap-northeast-1에 새 DynamoDB 테이블을 생성합니다. us-west-2의 DynamoDB 테이블에서 데이터 내보내기를 수행하는 AWS Glue 작업을 생성합니다. 동일한 데이터를 ap-northeast-1의 DynamoDB 테이블로 가져옵니다.
  • B. us-west-2의 DynamoDB 테이블에서 DynamoDB 스트림을 활성화합니다. ap-northeast-1에 새 DynamoDB 테이블을 생성합니다. us-west-2에서 DynamoDB 테이블 스트림을 폴링하고 스트림의 배치 레코드를 ap-northeast-1의 새 DynamoDB 테이블로 전달하는 AWS Lambda 함수를 생성합니다.
  • C. 특정 시점 복구를 사용하여 us-west-2 lo ap-northeast-1에서 DynamoDB 테이블을 복원합니다.
  • D. us-west-2의 DynamoDB 테이블에서 DynamoDB 스트림을 활성화합니다. us-west-2의 DynamoDB 전역 테이블 설정에 ap-northeast-1을 추가합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 316
Topic #: 1

A company that is located in the United States wants to expand its operations in Asia. The company’s data in the us-west-2 Region is stored in an Amazon DynamoD8 table. The company’s development team in the ap-northeast-1 Region needs to perform user acceptance testing (UAT) and several other performance feasibility tests with a copy of production data from us-west-2. The feasibility tests do not need to be run on data that is updated in real time.

Which solution will make data available from us-west-2 to ap-northeast-1 MOST cost-effectively?

  • A. Create a new DynamoDB table in ap-northeast-1. Create an AWS Glue job to perform a data export from the DynamoDB table in us-west-2. Import the same data into the DynamoDB table in ap-northeast-1.
  • B. Enable DynamoDB Streams on the DynamoDB table in us-west-2. Create a new DynamoDB table in ap-northeast-1. Create an AWS Lambda function to poll the DynamoDB table stream in us-west-2 and to deliver batch records from the stream to the new DynamoDB table in ap-northeast-1.
  • C. Use point-in-time recovery to restore the DynamoDB table from us-west-2 lo ap-northeast-1.
  • D. Enable DynamoDB Streams on the DynamoDB table in us-west-2. Add ap-northeast-1 to the DynamoDB global tables setting in us-west-2.

선택한 답변: D
us-west-2의 DynamoDB 테이블에서 DynamoDB 스트림을 활성화하고 DynamoDB 전역 테이블 구성에서 ap-northeast-1을 리전으로 추가하면 us-west-2에서 ap-northeast-1로 데이터를 효율적으로 복제할 수 있습니다. 실시간. DynamoDB Streams는 테이블에 대한 변경 사항을 캡처하고 글로벌 테이블 구성은 여러 리전에 대한 자동 복제를 보장합니다. 이 솔루션을 사용하면 수동 데이터 내보내기 및 가져오기 프로세스가 필요하지 않으며 us-west-2에서 ap-northeast-1로 데이터를 지속적으로 복제할 수 있습니다. 이를 통해 ap-northeast-1 개발팀은 UAT(사용자 승인 테스트) 및 성능 타당성 테스트를 위한 최신 데이터에 액세스할 수 있습니다. 따라서 us-west-2에서 ap-northeast-1까지의 데이터를 가장 비용 효율적으로 활용하는 솔루션은 옵션 D입니다.

Selected Answer: D
By enabling DynamoDB Streams on the DynamoDB table in us-west-2 and adding ap-northeast-1 as a region in the DynamoDB Global Table configuration, you can efficiently replicate the data from us-west-2 to ap-northeast-1 in real-time. DynamoDB Streams captures changes made to the table, and the Global Table configuration ensures automatic replication to multiple regions. This solution eliminates the need for manual data export and import processes and allows for continuous replication of the data from us-west-2 to ap-northeast-1. It ensures that the ap-northeast-1 development team has access to up-to-date data for user acceptance testing (UAT) and performance feasibility testing. Therefore, the solution that provides the most cost-efficient utilization of data from us-west-2 to ap-northeast-1 is option D.
