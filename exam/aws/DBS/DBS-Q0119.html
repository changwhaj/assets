질문 #: 119
주제 #: 1

한 온라인 쇼핑 회사에는 매일 많은 쇼핑 요청이 유입되고 있습니다. 결과적으로 회사의 Amazon RDS 데이터베이스에 지속적인 로드가 발생합니다. 데이터베이스 전문가는 데이터베이스가 항상 가동되어 실행되고 있는지 확인해야 합니다. 데이터베이스 전문가는 데이터베이스 가동 중지 시간을 유발할 수 있는 문제나 데이터베이스 구성 변경 사항에 대한 자동 알림 시스템을 원합니다.
이를 달성하기 위해 데이터베이스 전문가는 무엇을 해야 합니까? (2개를 선택하세요.)

  • A. Amazon CloudTrail에 기록된 모든 API 호출에 대해 Amazon SNS를 사용하여 알림을 보내려면 Amazon CloudWatch Events 이벤트를 생성하십시오.
  • B. RDS 이벤트 구독을 구독하고 Amazon SNS 주제를 사용하여 알림을 보내도록 구성합니다. 최다 투표
  • C. Amazon SES를 사용하여 구성된 Amazon CloudWatch Events 이벤트를 기반으로 알림을 보냅니다.
  • D. RDS 인스턴스용 FreeStorageSpace와 같은 다양한 지표에 대해 Amazon CloudWatch 경보를 구성합니다. 최다 투표
  • E. AWS Trusted Advisor에 대한 이메일 알림을 활성화합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 119
Topic #: 1

An online shopping company has a large inflow of shopping requests daily. As a result, there is a consistent load on the company's Amazon RDS database. A database specialist needs to ensure the database is up and running at all times. The database specialist wants an automatic notification system for issues that may cause database downtime or for configuration changes made to the database.
What should the database specialist do to achieve this? (Choose two.)

  • A. Create an Amazon CloudWatch Events event to send a notification using Amazon SNS on every API call logged in AWS CloudTrail.
  • B. Subscribe to an RDS event subscription and configure it to use an Amazon SNS topic to send notifications. Most Voted
  • C. Use Amazon SES to send notifications based on configured Amazon CloudWatch Events events.
  • D. Configure Amazon CloudWatch alarms on various metrics, such as FreeStorageSpace for the RDS instance. Most Voted
  • E. Enable email notifications for AWS Trusted Advisor.
