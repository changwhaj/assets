한 미디어 회사는 Amazon RDS for PostgreSQL을 사용하여 사용자 데이터를 저장하고 있습니다. RDS DB 인스턴스는 현재 공개적으로 액세스 가능한 설정이 활성화되어 있으며 퍼블릭 서브넷에서 호스팅됩니다. 최근 AWS Well-Architected 프레임워크 검토에 이어 데이터베이스 전문가에게 새로운 보안 요구 사항이 부여되었습니다.
✑ 특정 온프레미스 회사 네트워크 IP만 DB 인스턴스에 연결해야 합니다.
✑ 회사 네트워크에서만 연결이 허용됩니다.
이러한 새로운 요구 사항을 충족하기 위해 데이터베이스 전문가가 수행해야 하는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. pg_hba.conf 파일을 수정합니다. 필요한 회사 네트워크 IP를 추가하고 원치 않는 IP를 제거합니다.
  • B. 연결된 보안 그룹을 수정합니다. 필요한 회사 네트워크 IP를 추가하고 원치 않는 IP를 제거합니다. 최다 투표
  • C. AWS DMS를 사용하여 DB 인스턴스를 프라이빗 서브넷으로 이동합니다.
  • D. 회사 네트워크에서 실행되는 애플리케이션 호스트와 DB 인스턴스에 연결된 VPC 간에 VPC 피어링을 활성화합니다.
  • E. 공개적으로 액세스할 수 있는 설정을 비활성화합니다. 최다 투표
  • F. 사설 IP와 VPN을 사용하여 DB 인스턴스에 연결합니다. 최다 투표

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 26
Topic #: 1

A media company is using Amazon RDS for PostgreSQL to store user data. The RDS DB instance currently has a publicly accessible setting enabled and is hosted in a public subnet. Following a recent AWS Well-Architected Framework review, a Database Specialist was given new security requirements.
✑ Only certain on-premises corporate network IPs should connect to the DB instance.
✑ Connectivity is allowed from the corporate network only.
Which combination of steps does the Database Specialist need to take to meet these new requirements? (Choose three.)

  • A. Modify the pg_hba.conf file. Add the required corporate network IPs and remove the unwanted IPs.
  • B. Modify the associated security group. Add the required corporate network IPs and remove the unwanted IPs. Most Voted
  • C. Move the DB instance to a private subnet using AWS DMS.
  • D. Enable VPC peering between the application host running on the corporate network and the VPC associated with the DB instance.
  • E. Disable the publicly accessible setting. Most Voted
  • F. Connect to the DB instance using private IPs and a VPN. Most Voted
