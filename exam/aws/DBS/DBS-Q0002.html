주제 #: 1

데이터베이스 전문가는 향후 우발적인 인스턴스 데이터 손실 가능성을 줄이기 위해 CloudFormation 템플릿에 RDS 설정을 추가해야 합니다 .
이 요구 사항을 충족하는 설정은 무엇입니까? (3개를 선택하세요.)

  • A. DeletionProtection을 True로 설정 최다 투표
  • B. MultiAZ를 True로 설정
  • C. TerminationProtection을 True로 설정
  • D. DeleteAutomatedBackups를 False로 설정 최다 투표
  • E. DeletionPolicy를 삭제로 설정
  • F. DeletionPolicy를 Retain으로 설정 최다 투표

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 2
Topic #: 1

An AWS CloudFormation stack that included an Amazon RDS DB instance was accidentally deleted and recent data was lost. A Database Specialist needs to add
RDS settings to the CloudFormation template to reduce the chance of accidental instance data loss in the future.
Which settings will meet this requirement? (Choose three.)

  • A. Set DeletionProtection to True Most Voted
  • B. Set MultiAZ to True
  • C. Set TerminationProtection to True
  • D. Set DeleteAutomatedBackups to False Most Voted
  • E. Set DeletionPolicy to Delete
  • F. Set DeletionPolicy to Retain Most Voted
