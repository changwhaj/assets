질문 #: 256
주제 #: 1

한 회사가 최근 LOB(기간 업무) 애플리케이션을 AWS로 마이그레이션했습니다. 이 애플리케이션은 SQL Server DB 인스턴스용 Amazon RDS를 데이터베이스 엔진으로 사용합니다.
회사는 애플리케이션에 대해 지역 간 재해 복구를 설정해야 합니다. 이 회사는 RPO 및 RTO가 가장 낮은 솔루션이 필요합니다.
이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. DB 인스턴스의 리전 간 읽기 전용 복제본을 생성합니다. 장애 조치 시 읽기 전용 복제본을 승격합니다. 최다 투표
  • B. DB 인스턴스에서 재해 복구 리전의 Amazon EC2 인스턴스로 SQL 복제를 설정합니다. EC2 인스턴스를 기본 서버로 승격합니다.
  • C. 재해 복구 리전에서 DB 인스턴스의 지속적인 복제를 위해 AWS Database Migration Service(AWS KMS)를 사용합니다. 최다 투표
  • D. 기본 리전에서 DB 인스턴스의 수동 스냅샷을 만듭니다. 재해 복구 지역에 스냅샷을 복사합니다.

Exam question from Amazon's AWS Certified Database - Specialty
Question #: 256
Topic #: 1

A company recently migrated its line-of-business (LOB) application to AWS. The application uses an Amazon RDS for SQL Server DB instance as its database engine.
The company must set up cross-Region disaster recovery for the application. The company needs a solution with the lowest possible RPO and RTO.
Which solution will meet these requirements?

  • A. Create a cross-Region read replica of the DB instance. Promote the read replica at the time of failover. Most Voted
  • B. Set up SQL replication from the DB instance to an Amazon EC2 instance in the disaster recovery Region. Promote the EC2 instance as the primary server.
  • C. Use AWS Database Migration Service (AWS KMS) for ongoing replication of the DB instance in the disaster recovery Region. Most Voted
  • D. Take manual snapshots of the DB instance in the primary Region. Copy the snapshots to the disaster recovery Region.
