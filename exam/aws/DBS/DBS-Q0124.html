주제 #: 1

Amazon Aurora MySQL DB 인스턴스의 FreeLocalStorage에 대한 Amazon CloudWatch 지표는 로컬 스토리지의 양이 10MB 미만임을 보여줍니다. 데이터베이스 엔지니어는 Aurora DB 인스턴스에서 사용 가능한 로컬 스토리지를 늘려야 합니다.
데이터베이스 엔지니어는 이 요구 사항을 어떻게 충족해야 합니까?

  • A. 더 많은 로컬 SSD 스토리지를 제공하는 인스턴스 클래스를 사용하도록 DB 인스턴스를 수정합니다. 최다 투표
  • B. 자동 볼륨 크기 조정을 활성화하도록 Aurora DB 클러스터를 수정합니다.
  • C. 데이터베이스 엔진 버전을 업그레이드하여 로컬 스토리지를 늘리십시오.
  • D. DB 인스턴스를 수정하고 구성 섹션에서 필요한 스토리지 볼륨을 구성합니다.

How should the database engineer meet this requirement?

  • B. Modify the Aurora DB cluster to enable automatic volume resizing.
  • C. Increase the local storage by upgrading the database engine version.
  • D. Modify the DB instance and configure the required storage volume in the configuration section.
