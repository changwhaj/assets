이러한 요구 사항을 가장 비용 효율적으로 충족하는 작업 조합은 무엇입니까? (2개를 선택하세요.)

  • A. 인스턴스당 허용 가능한 백로그를 목표로 하는 대상 조정 정책을 사용하여 스팟 집합을 생성합니다. 최소 용량을 위해 온디맨드 인스턴스 2개를 요청하세요. 추가 용량을 원하면 스팟 인스턴스를 사용하십시오. 최다 투표
  • B. 대상 AWS 지역의 대상 플랫폼 및 인스턴스 유형에 대해 두 개의 예약 인스턴스를 구매합니다. 최다 투표 최다 투표

  • D. 스팟 인스턴스 요청을 사용하는 Auto Scaling 그룹을 생성합니다. SQS 대기열의 크기에 따라 조정되도록 조정 정책을 구성합니다. 최소값을 2로 설정합니다. 최다 투표
  • E. 두 개의 전용 호스트를 프로비저닝합니다. 추가 용량을 제공하기 위해 스팟 인스턴스를 사용하도록 AWS Batch를 구성합니다.

Topic #: 1

A solutions architect is designing a data processing system that will use Amazon EC2 instances. Data that needs to be processed will wait in an Amazon Simple
Queue Service (Amazon SQS) queue. At least two data processing instances must run at all times.
Which combination of actions will meet these requirements MOST cost-effectively? (Choose two.)

  • A. Create a Spot Fleet with a target scaling policy that targets the acceptable backlog per instance. Request two On-Demand Instances for minimum capacity. Use Spot Instances for additional capacity. Most Voted
  • B. Purchase two Reserved Instances for the target platform and instance type in the target AWS Region. Most Voted Most Voted
  • C. Create On-Demand Capacity Reservations for two instances for the target platform and instance type in the target AWS Region.
  • D. Create an Auto Scaling group that uses Spot Instance requests. Configure the scaling policy to scale with the size of the SQS queue. Set the minimum value to 2. Most Voted
  • E. Provision two Dedicated Hosts. Configure AWS Batch to use Spot Instances to supply additional capacity.
