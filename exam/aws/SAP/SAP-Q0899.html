주제 #: 1

한 언론사는 외부 API를 호출하여 10분마다 새로운 보도 자료를 수신하는 AWS Lambda 함수를 구현하려고 합니다. API 공급자는 IP 주소 허용 목록을 사용하여 API를 보호할 계획이므로 뉴스 회사는 API에 액세스하는 모든 공용 IP 주소를 제공해야 합니다. 회사의 현재 아키텍처에는 인터넷 게이트웨이와 NAT 게이트웨이가 있는 VPC가 포함되어 있습니다. 솔루션 아키텍트는 Lambda 함수에 대한 고정 IP 주소를 구현해야 합니다.
솔루션 설계자는 이러한 요구 사항을 충족하기 위해 어떤 단계 조합을 수행해야 합니까? (2개를 선택하세요.)

  • A. IP 주소 허용 목록에는 NAT 게이트웨이와 연결된 탄력적 IP 주소를 사용합니다. 최다 투표
  • B. Lambda 함수에 탄력적 IP 주소를 할당합니다. IP 주소 허용 목록에는 Lambda 함수의 탄력적 IP 주소를 사용합니다.
  • C. VPC의 프라이빗 서브넷에서 실행되도록 Lambda 함수를 구성합니다. 최다 투표
  • D. VPC의 퍼블릭 서브넷에서 실행되도록 Lambda 함수를 구성합니다.
  • E. 전송 게이트웨이를 생성합니다. VPC와 Lambda 함수를 전송 게이트웨이에 연결합니다.

A news company wants to implement an AWS Lambda function that calls an external API to receive new press releases every 10 minutes. The API provider is planning to use an IP address allow list to protect the API, so the news company needs to provide any public IP addresses that access the API. The company's current architecture includes a VPC with an internet gateway and a NAT gateway. A solutions architect must implement a static IP address for the Lambda function.
Which combination of steps should the solutions architect take to meet these requirements? (Choose two.)

  • A. Use the Elastic IP address that is associated with the NAT gateway for the IP address allow list. Most Voted
  • B. Assign an Elastic IP address to the Lambda function. Use the Lambda function's Elastic IP address for the IP address allow list.
  • C. Configure the Lambda function to launch in the private subnet of the VPC. Most Voted
  • D. Configure the Lambda function to launch in the public subnet of the VPC.
  • E. Create a transit gateway. Attach the VPC and the Lambda function to the transit gateway.
