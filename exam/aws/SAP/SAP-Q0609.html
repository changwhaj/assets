한 회사에서 인기 있는 공개 전자상거래 웹사이트를 운영하고 있습니다. 사용자 기반은 현지 시장에서 전국 시장으로 빠르게 성장하고 있습니다. 웹사이트는 웹 서버와 MySQL 데이터베이스를 갖춘 온프레미스 데이터 센터에서 호스팅됩니다. 회사는 워크로드를 AWS로 마이그레이션하려고 합니다. 솔루션 설계자는 다음을 위해 솔루션을 만들어야 합니다.
✑ 보안 향상
✑ 안정성 향상
✑ 가용성 향상
✑ 대기 시간 감소
✑ 유지 관리 감소
이러한 요구 사항을 충족하기 위해 솔루션 설계자가 수행해야 하는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. Application Load Balancer 뒤의 Auto Scaling 그룹에 있는 웹 서버용 2개의 가용 영역에서 Amazon EC2 인스턴스를 사용하십시오. 최다 투표
  • B. 데이터베이스를 다중 AZ Amazon Aurora MySQL DB 클러스터로 마이그레이션합니다. 최다 투표
  • C. 두 개의 가용 영역에서 Amazon EC2 인스턴스를 사용하여 고가용성 MySQL 데이터베이스 클러스터를 호스팅합니다.
  • D. Amazon S3에서 정적 웹사이트 콘텐츠를 호스팅합니다. S3 Transfer Acceleration을 사용하면 웹페이지를 제공하는 동안 지연 시간을 줄일 수 있습니다. AWS WAF를 사용하여 웹사이트 보안을 강화하세요.
  • E. Amazon S3에서 정적 웹사이트 콘텐츠를 호스팅합니다. Amazon CloudFront를 사용하면 웹페이지를 제공하는 동안 지연 시간을 줄일 수 있습니다. AWS WAF를 사용하여 웹사이트 보안을 강화하세요. 최다 투표
  • F. 데이터베이스를 단일 AZ Amazon RDS for MySQL DB 인스턴스로 마이그레이션합니다.

Exam question from Amazon's AWS Certified Solutions Architect - Professional
Question #: 609
Topic #: 1

A company runs a popular public-facing ecommerce website. Its user base is growing quickly from a local market to a national market. The website is hosted in an on-premises data center with web servers and a MySQL database. The company wants to migrate its workload to AWS. A solutions architect needs to create a solution to:
✑ Improve security
✑ Improve reliability
✑ Improve availability
✑ Reduce latency
✑ Reduce maintenance
Which combination of steps should the solutions architect take to meet these requirements? (Choose three.)

  • A. Use Amazon EC2 instances in two Availability Zones for the web servers in an Auto Scaling group behind an Application Load Balancer. Most Voted
  • B. Migrate the database to a Multi-AZ Amazon Aurora MySQL DB cluster. Most Voted
  • C. Use Amazon EC2 instances in two Availability Zones to host a highly available MySQL database cluster.
  • D. Host static website content in Amazon S3. Use S3 Transfer Acceleration to reduce latency while serving webpages. Use AWS WAF to improve website security.
  • E. Host static website content in Amazon S3. Use Amazon CloudFront to reduce latency while serving webpages. Use AWS WAF to improve website security. Most Voted
  • F. Migrate the database to a single-AZ Amazon RDS for MySQL DB instance.
