회사에서는 솔루션 설계자에게 솔루션 비용 최적화를 요청합니다. 이 솔루션은 여러 고객의 요청을 처리합니다. 이 솔루션에는 Amazon API Gateway, AWS Lambda, AWS Fargate, Amazon Simple Queue Service(Amazon SQS) 및 Amazon EC2를 사용하는 다중 계층 아키텍처가 포함되어 있습니다.
현재 설정에서 요청은 API 게이트웨이를 통해 Lambda로 이동하고 Fargate에서 컨테이너를 시작하거나 SQS 대기열에 메시지를 푸시합니다. EC2 집합은 SQS 대기열의 작업자 역할을 하는 EC2 인스턴스를 제공합니다. EC2 집합은 SQS 대기열의 항목 수에 따라 확장됩니다.
솔루션 설계자는 비용을 가장 많이 절감하기 위해 어떤 단계 조합을 권장해야 합니까? (3개를 선택하세요.)

  • A. 하루에 필요한 최소 EC2 인스턴스 수를 결정하십시오. 이 수만큼의 인스턴스를 3년 요금제로 예약하고 모두 선불로 결제하세요.
  • B. 지난 6개월간 서비스 전체의 컴퓨팅 사용률을 조사합니다. 이 정보를 사용하여 솔루션에 필요한 컴퓨팅을 결정합니다. 이 금액에 대해 저축 계획을 세우십시오. 최다 투표
  • C. 하루 동안 필요한 평균 EC2 인스턴스 수를 결정합니다. 이 수만큼의 인스턴스를 3년 요금제로 예약하고 모두 선불로 결제하세요.
  • D. 솔루션 및 솔루션 인프라에서 SQS 대기열을 제거합니다. 최다 투표
  • E. EC2 인스턴스 대신 컨테이너로 실행되도록 솔루션을 변경합니다. 환경 변수를 사용하여 솔루션에 메시지를 제공함으로써 Fargate에서 솔루션을 시작하도록 Lambda를 구성합니다. 최다 투표
  • F. Application Load Balancer를 통해 EC2 인스턴스에 직접 메시지를 게시하도록 Lambda 함수를 변경합니다.

Exam question from Amazon's AWS Certified Solutions Architect - Professional
Question #: 825
Topic #: 1

A company asks a solution architect to optimize the cost of a solution. The solution handles requests from multiple customers. The solution includes a multi-tier architecture that uses Amazon API Gateway, AWS Lambda, AWS Fargate, Amazon Simple Queue Service (Amazon SQS), and Amazon EC2.
In the current setup, requests go through API Gateway to Lambda and either start a container in Fargate or push a message to an SQS queue. An EC2 Fleet provides EC2 instances that serve as workers for the SQS queue. The EC2 Fleet scales based on the number of items in the SQS queue.
Which combination of steps should the solutions architect recommend to reduce cost the MOST? (Choose three.)

  • A. Determine the minimum number of EC2 instances that are needed during a day. Reserve this number of instances in a 3-year plan with payment all upfront.
  • B. Examine the last 6 months of compute utilization across the services. Use this information to determine the needed compute for the solution. Commit to a Savings Plan for this amount. Most Voted
  • C. Determine the average number of EC2 instances that are needed during a day. Reserve this number of instances in a 3-year plan with payment all upfront.
  • D. Remove the SQS queue from the solution and from the solution infrastructure. Most Voted
  • E. Change the solution so that it runs as a container instead of on EC2 instances. Configure Lambda to start up the solution in Fargate by using environment variables to give the solution the message. Most Voted
  • F. Change the Lambda function so that it posts the message directly to the EC2 instances through an Application Load Balancer.
