주제 #: 1

한 회사에서 객체 스토리지로 Amazon S3를 사용하려고 합니다. 사용자는 온프레미스 프라이빗 네트워크 또는 Amazon EC2 인스턴스에 연결된 디바이스에서 객체에 액세스할 수 있어야 합니다. 회사는 AWS Direct Connect 및 AWS Site-to-Site VPN을 백업으로 구성했습니다. 회사는 공용 인터넷을 통해 S3 트래픽을 라우팅하는 것을 원하지 않습니다. 또한 회사는 S3 버킷에 저장된 모든 데이터를 DataClassification이라는 태그를 사용하여 데이터 유형별로 적절하게 분류하도록 요구합니다.

이러한 요구 사항을 충족하려면 솔루션 설계자가 수행해야 하는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. 온프레미스에서 S3 버킷으로 트래픽을 안전하게 라우팅하도록 게이트웨이 VPC 엔드포인트를 구성합니다. AWS 프라이빗 네트워크를 통해 S3 버킷과 EC2 인스턴스 간에 트래픽을 라우팅하도록 인터페이스 VPC 엔드포인트를 구성합니다.
  • B. 온프레미스에서 S3 버킷으로 트래픽을 안전하게 라우팅하도록 인터페이스 VPC 엔드포인트를 구성합니다. AWS 프라이빗 네트워크를 통해 S3 버킷과 EC2 인스턴스 간에 트래픽을 라우팅하도록 게이트웨이 VPC 엔드포인트를 구성합니다. 최다 투표 최다 투표
  • C. DataClassification 태그가 누락된 S3 버킷을 식별하도록 Amazon GuardDuty를 구성합니다. Amazon Simple 알림 서비스(Amazon SNS) 주제를 생성합니다. 태그가 지정되지 않은 S3 버킷이 식별될 때마다 해당 주제에 알림을 전달합니다.
  • D. DataClassification 태그가 누락된 S3 버킷을 식별하도록 AWS Security Hub를 구성합니다. Amazon Simple 알림 서비스(Amazon SNS) 주제를 생성합니다. 태그가 지정되지 않은 S3 버킷이 식별될 때마다 해당 주제에 알림을 전달합니다. 최다 투표
  • E. DataClassification 태그가 누락된 S3 버킷을 식별하도록 AWS Config를 구성합니다. AWS Config가 태그 누락으로 식별한 모든 리소스에 대한 보고서를 생성합니다. 최다 투표
  • F. 일정에 따라 계정의 모든 S3 버킷을 스캔하도록 Amazon Macie를 구성합니다. Macie를 Amazon EventBridge(Amazon CloudWatch Events)와 통합합니다. Macie가 추론한 데이터 분류를 검증하고 누락된 태그를 추가하는 AWS Lambda 함수를 생성합니다. 최다 투표 최다 투표

Topic #: 1

A company wants to use Amazon S3 for object storage. Users must be able to access the objects from devices that are connected to their on-premises private network or Amazon EC2 instances. The company has configured AWS Direct Connect and AWS Site-to-Site VPN as a backup. The company does not want to route S3 traffic over the public Internet. The company also requires all data that is stored in S3 buckets to be appropriately classified by data type with a tag named DataClassification.

Which combination of steps should a solutions architect take to meet these requirements? (Choose three.)

  • A. Configure a gateway VPC endpoint to securely route traffic from on premises to the S3 buckets. Configure an interface VPC endpoint to route traffic between the S3 buckets and EC2 instances over the AWS private network.
  • B. Configure an interface VPC endpoint to securely route traffic from on premises to the S3 buckets. Configure a gateway VPC endpoint to route traffic between the S3 buckets and EC2 instances over the AWS private network. Most Voted Most Voted
  • C. Configure Amazon GuardDuty to identify S3 buckets that are missing the DataClassification tag. Create an Amazon Simple Notification Service (Amazon SNS) topic. Deliver notifications to the topic whenever an untagged S3 bucket is identified.
  • D. Configure AWS Security Hub to identify S3 buckets that are missing the DataClassification tag. Create an Amazon Simple Notification Service (Amazon SNS) topic. Deliver notifications to the topic whenever an untagged S3 bucket is identified. Most Voted
  • E. Configure AWS Config to identify S3 buckets that are missing the DataClassification tag. Generate a report of all resources that AWS Config identifies as missing the tag. Most Voted
  • F. Configure Amazon Macie to scan all S3 buckets in the account on a scheduled basis. Integrate Macie with Amazon EventBridge (Amazon CloudWatch Events). Create an AWS Lambda function to validate the data classification inferred by Macie and to add the missing tag. Most Voted Most Voted
