When using an IAM role to grant permissions to applications running on Amazon EC2 instances, only one IAM role can be assigned to an EC2 instance at a time. Each EC2 instance can be associated with a single IAM role, and all applications running on that instance will share the same role and have the same set of permissions granted by that role. Option A is incorrect because although all applications on the instance share the same role, they have the same permissions, not different permissions. Option B is incorrect because all applications on the instance share a single role, not multiple roles. Option C is incorrect because only one role can be assigned to an EC2 instance at a time.
