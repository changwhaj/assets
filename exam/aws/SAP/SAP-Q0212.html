  • A. 인스턴스의 모든 애플리케이션은 동일한 역할을 공유하지만 권한은 다릅니다.
  • B. 인스턴스의 모든 애플리케이션은 여러 역할과 권한을 공유합니다.
  • C. 한 번에 EC2 인스턴스에 여러 역할이 할당됩니다.
  • D. EC2 인스턴스에는 한 번에 하나의 역할만 할당할 수 있습니다.

Question #: 212
Topic #: 1

Which of the following is true while using an IAM role to grant permissions to applications running on Amazon EC2 instances?

  • A. All applications on the instance share the same role, but different permissions.
  • B. All applications on the instance share multiple roles and permissions.
  • C. Multiple roles are assigned to an EC2 instance at a time.
  • D. Only one role can be assigned to an EC2 instance at a time.
Show Suggested Answer Hide Answer
Suggested Answer: D 🗳️
Only one role can be assigned to an EC2 instance at a time, and all applications on the instance share the same role and permissions.
Reference:
http://docs.aws.amazon.com/IAM/latest/UserGuide/role-usecase-ec2app.html

Selected Answer: D
The correct option is D. Only one role can be assigned to an EC2 instance at a time. When using an IAM role to grant permissions to applications running on Amazon EC2 instances, only one IAM role can be assigned to an EC2 instance at a time. Each EC2 instance can be associated with a single IAM role, and all applications running on that instance will share the same role and have the same set of permissions granted by that role. Option A is incorrect because although all applications on the instance share the same role, they have the same permissions, not different permissions. Option B is incorrect because all applications on the instance share a single role, not multiple roles. Option C is incorrect because only one role can be assigned to an EC2 instance at a time.
