  • A. 작동 유효성을 확인하려면 스택 생성을 시도해야 합니다. 최다 투표
  • B. AWS CloudFormation 템플릿의 작동 유효성을 확인할 수 있는 방법이 없습니다.
  • C. 작동 유효성을 확인하려면 AWS CloudFormation 스택에 대한 샌드박스 또는 테스트 영역이 필요합니다.
  • D. 작동 유효성을 확인하려면 aws cloudformation verify-template 명령을 사용해야 합니다.
제안된 답변 표시 답변 숨기기
제안 답변: A 🗳️

How can you check the operational validity of your AWS CloudFormation template?

  • A. To check the operational validity, you need to attempt to create the stack. Most Voted
  • B. There is no way to check the operational validity of your AWS CloudFormation template.
  • C. To check the operational validity, you need a sandbox or test area for AWS CloudFormation stacks.
  • D. To check the operational validity, you need to use the aws cloudformation validate-template command.
Show Suggested Answer Hide Answer
Suggested Answer: A 🗳️
In AWS CloudFormation, to check the operational validity, you need to attempt to create the stack. There is no sandbox or test area for AWS CloudFormation stacks, so you are charged for the resources you create during testing.
Reference:
http://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/using-cfn-validate-template.html

Selected Answer: A
A. To check the operational validity, you need to attempt to create the stack. By creating a stack from your CloudFormation template, AWS will attempt to provision the specified resources and configurations. During this process, AWS CloudFormation will perform various checks and validations to ensure the template's operational validity. If there are any issues or errors in the template, AWS CloudFormation will provide feedback and fail to create the stack. This feedback can help identify and address any problems before attempting to deploy the template in a production environment. Using the "aws cloudformation validate-template" command (option D) can help catch syntax and structure errors in your template but may not cover all aspects of operational validity. A sandbox or test area (option C) can be useful for testing the behavior of your resources but may not fully validate the operational viability of the template.
