주제 #: 1

회사는 두 개의 개별 애플리케이션을 온프레미스에서 AWS로 마이그레이션해야 합니다.

• 첫 번째 애플리케이션은 물리적 Windows 서버에서 호스팅되는 레거시 사용자 지정 애플리케이션입니다. 애플리케이션 소스 코드는 더 이상 사용할 수 없습니다. 애플리케이션에는 문서가 거의 없고 운영 체제 구성 설정이 하드코딩되어 있으며 외부 제3자가 사용합니다.
• 두 번째 애플리케이션은 NAS(Network Attached Storage)를 사용하여 데이터베이스 데이터를 저장하는 단일 Linux VM에서 호스팅되는 IBM Db2 데이터베이스입니다. 회사는 직원 기록을 위해 내부적으로 이 데이터베이스를 사용합니다.

애플리케이션은 회사가 90일 후에 폐기할 데이터 센터에서 호스팅됩니다. 가능한 경우 회사는 관리형 AWS 서비스를 사용해야 합니다.

솔루션 설계자는 이러한 요구 사항을 충족하기 위해 어떤 마이그레이션 작업을 권장해야 합니까? (2개를 선택하세요.)

  • A. AWS Application Migration Service(CloudEndure Migration)를 사용하여 레거시 애플리케이션이 포함된 Windows 서버를 Amazon EC2로 마이그레이션합니다. 최다 투표
  • B. AWS Application Migration Service(CloudEndure Migration)를 사용하여 IBM Db2 데이터베이스 서비스가 포함된 Linux VM을 Amazon EC2 인스턴스로 마이그레이션합니다.
  • C. AWS Server Migration Service(AWS SMS)를 사용하여 레거시 애플리케이션이 포함된 Windows 서버를 Amazon EC2로 마이그레이션합니다.
  • D. AWS Database Migration Service(AWS DMS) 및 AWS Schema Conversion Tool 복제 에이전트를 사용하여 IBM Db2 데이터베이스 데이터를 MySQL용 Amazon RDS로 마이그레이션합니다. 최다 투표
  • E. AWS DataSync 및 AWS Schema Conversion Tool 데이터 추출 에이전트를 사용하여 IBM Db2 데이터베이스 데이터를 MySQL용 Amazon RDS로 마이그레이션합니다.

Exam question from Amazon's AWS Certified Solutions Architect - Professional
Question #: 998
Topic #: 1

A company needs to migrate two individual applications from on premises to AWS:

• The first application is a legacy custom application that is hosted on a physical Windows server. The application source code is no longer available. The application has little documentation, has hardcoded operating system configuration settings, and is used by an external third party.
• The second application is an IBM Db2 database that is hosted on a single Linux VM that uses network-attached storage (NAS) to store the database data. The company uses this database internally for employee records.

The applications are hosted in a data center that the company plans to decommission in 90 days. Where possible, the company must use managed AWS services.

Which actions for migration should a solutions architect recommend to meet these requirements? (Choose two.)

  • A. Migrate the Windows server with the legacy application to Amazon EC2 by using AWS Application Migration Service (CloudEndure Migration). Most Voted
  • B. Migrate the Linux VM with the IBM Db2 database service to an Amazon EC2 instance by using AWS Application Migration Service (CloudEndure Migration).
  • C. Migrate the Windows server with the legacy application to Amazon EC2 by using AWS Server Migration Service (AWS SMS).
  • D. Migrate the IBM Db2 database data to Amazon RDS for MySQL by using AWS Database Migration Service (AWS DMS) and the AWS Schema Conversion Tool replication agent. Most Voted
  • E. Migrate the IBM Db2 database data to Amazon RDS for MySQL by using AWS DataSync and the AWS Schema Conversion Tool data extraction agent.
