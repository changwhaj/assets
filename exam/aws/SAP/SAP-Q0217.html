주제 #: 1

다음 중 AWS Data Pipeline을 사용하여 수행할 수 없는 것은 무엇입니까?

  • A. 내결함성, 반복 가능 및 고가용성을 갖춘 복잡한 데이터 처리 워크로드를 생성합니다.
  • B. 저장된 데이터에 정기적으로 액세스하고, 이를 대규모로 변환 및 처리하고, 결과를 다른 AWS 서비스로 효율적으로 전송합니다.
  • C. 저장된 데이터에 대한 보고서를 생성합니다.
  • D. 지정된 간격으로 다양한 AWS 컴퓨팅 및 스토리지 서비스는 물론 온프레미스 데이터 소스 간에 데이터를 이동합니다.
제안된 답변 표시 답변 숨기기
추천 답변: C 🗳️
AWS Data Pipeline은 지정된 간격으로 다양한 AWS 컴퓨팅 및 스토리지 서비스는 물론 온프레미스 데이터 소스 간에 데이터를 안정적으로 처리하고 이동하는 데 도움이 되는 웹 서비스입니다. AWS Data Pipeline을 사용하면 저장된 데이터에 정기적으로 액세스하여 대규모로 변환 및 처리하고 결과를 다른 AWS로 효율적으로 전송할 수 있습니다.
AWS Data Pipeline을 사용하면 내결함성, 반복 가능 및 가용성이 뛰어난 복잡한 데이터 처리 워크로드를 쉽게 생성할 수 있습니다. AWS Data Pipeline을 사용하면 이전에 온프레미스 데이터 사일로에 갇혀 있던 데이터를 이동하고 처리할 수도 있습니다.
참조:
http://aws.amazon.com/datapipeline/

Which of the following cannot be done using AWS Data Pipeline?

  • A. Create complex data processing workloads that are fault tolerant, repeatable, and highly available.
  • B. Regularly access your data where it's stored, transform and process it at scale, and efficiently transfer the results to another AWS service.
  • C. Generate reports over data that has been stored.
  • D. Move data between different AWS compute and storage services as well as on premise data sources at specified intervals.
Show Suggested Answer Hide Answer
Suggested Answer: C 🗳️
AWS Data Pipeline is a web service that helps you reliably process and move data between different AWS compute and storage services as well as on premise data sources at specified intervals. With AWS Data Pipeline, you can regularly access your data where it's stored, transform and process it at scale, and efficiently transfer the results to another AWS.
AWS Data Pipeline helps you easily create complex data processing workloads that are fault tolerant, repeatable, and highly available. AWS Data Pipeline also allows you to move and process data that was previously locked up in on premise data silos.

옵션 C, "저장된 데이터에 대한 보고서 생성"은 AWS Data Pipeline을 사용하여 직접 수행할 수 없습니다. AWS Data Pipeline은 다양한 AWS 서비스와 온프레미스 데이터 소스 간의 데이터 이동 및 처리를 조율하고 자동화할 수 있는 웹 서비스입니다. 복잡한 데이터 처리 워크로드를 관리하고 조정하도록 설계되었습니다. 그러나 저장된 데이터에 대한 보고서를 생성하려면 일반적으로 데이터 쿼리 및 분석이 필요하며 이는 AWS Data Pipeline에서 제공하는 직접적인 기능이 아닙니다. AWS에 저장된 데이터에 대한 보고서를 생성하려면 Amazon Athena, Amazon Redshift 또는 Amazon QuickSight와 같은 다른 서비스를 사용할 수 있습니다. 이러한 서비스는 저장된 데이터를 기반으로 보고서를 생성하는 데 사용할 수 있는 쿼리, 분석 및 시각화 기능을 제공합니다.

Selected Answer: C
Option C, "Generate reports over data that has been stored," cannot be done directly using AWS Data Pipeline. AWS Data Pipeline is a web service that allows you to orchestrate and automate the movement and processing of data between different AWS services and on-premises data sources. It is designed for managing and orchestrating complex data processing workloads. However, generating reports over stored data typically involves querying and analyzing the data, which is not a direct functionality provided by AWS Data Pipeline. To generate reports over stored data in AWS, you can use other services such as Amazon Athena, Amazon Redshift, or Amazon QuickSight. These services provide querying, analysis, and visualization capabilities that can be used to generate reports based on the stored data.
