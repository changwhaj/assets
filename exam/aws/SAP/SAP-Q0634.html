이 문제를 해결하려면 어떤 전략이 필요합니까? (2개를 선택하세요.)

  • A. 아웃바운드 Amazon Route 53 Resolver를 정의합니다. Active Directory 도메인에 대한 조건부 전달 규칙을 Active Directory 서버로 설정합니다. AmazonProvidedDNS로 설정된 VPC DHCP 옵션을 업데이트합니다. 최다 투표
  • B. 권한이 없는 모든 쿼리를 VPC 확인자에 전달하도록 Active Directory 서버의 DNS 서비스를 업데이트합니다. 최다 투표
  • C. 인바운드 Amazon Route 53 Resolver를 정의합니다. Active Directory 도메인에 대한 조건부 전달 규칙을 Active Directory 서버로 설정합니다. AmazonProvidedDNS로 설정된 VPC DHCP 옵션을 업데이트합니다.
  • D. 클라이언트 인스턴스의 DNS 서비스를 업데이트하여 Active Directory 서버와 VPC 확인자 간에 DNS 쿼리를 분할합니다.
  • E. 모든 쿼리를 VPC 확인자로 전달하도록 Active Directory 서버의 DNS 서비스를 업데이트합니다.

Question #: 634
Topic #: 1

A company has a VPC with two domain controllers running Active Directory in the default configuration. The VPC DHCP options set is configured to use the IP addresses of the two domain controllers. There is a VPC interface endpoint defined; but instances within the VPC are not able to resolve the private endpoint addresses.
Which strategies would resolve this issue? (Choose two.)

  • A. Define an outbound Amazon Route 53 Resolver. Set a conditional forward rule for the Active Directory domain to the Active Directory servers. Update the VPC DHCP options set to AmazonProvidedDNS. Most Voted
  • B. Update the DNS service on the Active Directory servers to forward all non-authoritative queries to the VPC Resolver. Most Voted
  • C. Define an inbound Amazon Route 53 Resolver. Set a conditional forward rule for the Active Directory domain to the Active Directory servers. Update the VPC DHCP options set to AmazonProvidedDNS.
  • D. Update the DNS service on the client instances to split DNS queries between the Active Directory servers and the VPC Resolver.
  • E. Update the DNS service on the Active Directory servers to forward all queries to the VPC Resolver.
