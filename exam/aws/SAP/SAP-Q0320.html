AWS CloudFormation ______ are special actions you use in your template to assign values to properties that are not available until runtime.

  • A. intrinsic functions
  • B. properties declarations
  • C. output functions
  • D. conditions declarations
Show Suggested Answer Hide Answer
Suggested Answer: A 🗳️
AWS CloudFormation intrinsic functions are special actions you use in your template to assign values to properties not available until runtime. Each function is declared with a name enclosed in quotation marks (""), a single colon, and its parameters.
Reference:
http://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/intrinsic-fuctions-structure.html

정답은 다음과 같습니다. A. 내장 함수 AWS CloudFormation 내장 함수는 런타임까지 알려지지 않았거나 사용할 수 없는 속성에 값을 할당하기 위해 CloudFormation 템플릿에서 사용할 수 있는 특수 작업입니다. 내장 함수를 사용하면 CloudFormation 템플릿 내에서 변환, 계산 및 조건부 논리를 수행할 수 있습니다. 내장 함수의 예로는 지정된 매개변수 또는 리소스의 값을 검색하는 Fn::Ref, 템플릿의 리소스에서 속성 값을 검색하는 Fn::GetAtt, 내부 변수를 대체하는 Fn::Sub가 있습니다. 해당 값이 포함된 문자열입니다. 내장 함수를 사용하면 속성에 값을 동적으로 할당하고 CloudFormation 템플릿을 더욱 유연하고 재사용 가능하게 만들 수 있습니다.

Selected Answer: A
The correct answer is: A. intrinsic functions AWS CloudFormation intrinsic functions are special actions that you can use in your CloudFormation templates to assign values to properties that are not known or available until runtime. Intrinsic functions allow you to perform transformations, calculations, and conditional logic within your CloudFormation templates. Examples of intrinsic functions include Fn::Ref, which retrieves the value of the specified parameter or resource, Fn::GetAtt, which retrieves the value of an attribute from a resource in the template, and Fn::Sub, which substitutes variables within a string with their corresponding values. By using intrinsic functions, you can dynamically assign values to properties and make your CloudFormation templates more flexible and reusable.
