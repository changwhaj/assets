주제 #: 1

회사는 AWS Organizations에 조직을 가지고 있습니다. 회사는 조직과 AWS Resource Access Manager
(AWS RAM) 간의 신뢰할 수 있는 액세스를 활성화했습니다. 조직에는 공유 서비스, 개발 및 생산용으로 각각 하나씩 총 3개의 AWS 계정이 있습니다. 공유 서비스 계정에는
VPC가 있습니다.
솔루션 아키텍트는 다음 요구 사항을 충족해야 합니다.
* 공유 서비스 VPC와 개발 및 프로덕션 계정 간의 액세스를 구성합니다.
* 각 계정의 워크로드가 3개 이상의 가용 영역에 배포되었는지 확인하세요.
* 개발 워크로드와 프로덕션 워크로드 간에 직접적인 통신이 없는지 확인하세요.
이러한 요구 사항을 충족하는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. 공유 서비스 VPC에서 3개의 가용 영역에 대해 3개의 서브넷을 생성합니다. 각 가용 영역에 하나의 서브넷을 생성합니다.
  • B. 공유 서비스 VPC에서 3개의 가용 영역에 대해 6개의 서브넷을 생성합니다. 각 가용 영역에 2개의 서브넷을 생성합니다. 최다 투표
  • C. 개발 계정과 프로덕션 계정의 서브넷 간의 연결을 방지하도록 네트워크 ACL을 구성합니다. 최다 투표
  • D. 개발 계정과 프로덕션 계정의 서브넷 간의 연결을 방지하도록 VPC 기본 보안 그룹 아웃바운드 규칙을 구성합니다.
  • E. AWS RAM을 사용하여 개발 계정과 서로 다른 가용 영역에 있는 서브넷 3개를 공유합니다. 또한 AWS RAM을 사용하여 동일한 3개의 서브넷을 프로덕션 계정과 공유합니다.
  • F. AWS RAM을 사용하여 개발 계정과 서로 다른 가용 영역에 있는 서브넷 3개를 공유합니다. 또한 AWS RAM을 사용하여 서로 다른 가용 영역에 있는 다른 서브넷 3개를 프로덕션 계정과 공유합니다. 최다 투표

Exam question from Amazon's AWS Certified Solutions Architect - Professional
Question #: 946
Topic #: 1

A company has an organization in AWS Organizations. The company has enabled trusted access between Organizations and AWS Resource Access Manager
(AWS RAM). The organization includes three AWS accounts, one each for shared services, development, and production. The shared services account has a
VPC.
A solutions architect needs to meet the following requirements:
* Configure access between the shared services VPC and the development and production accounts.
* Ensure that workloads in each account are deployed to at least three Availability Zones.
* Ensure that there is no direct communication between the development and production workloads.
Which combination of steps will meet these requirements? (Choose three.)

  • A. In the shared services VPC, create three subnets for three Availability Zones. Create one subnet in each Availability Zone.
  • B. In the shared services VPC, create six subnets for three Availability Zones. Create two subnets in each Availability Zone. Most Voted
  • C. Configure network ACLs to prevent connectivity between the subnets in the development account and the production account. Most Voted
  • D. Configure VPC default security group outbound rules to prevent connectivity between the subnets in the development account and the production account.
  • E. Use AWS RAM to share three subnets in different Availability Zones with the development account. Additionally, use AWS RAM to share the same three subnets with the production account.
  • F. Use AWS RAM to share three subnets in different Availability Zones with the development account. Additionally, use AWS RAM to share three other subnets in different Availability Zones with the production account. Most Voted
