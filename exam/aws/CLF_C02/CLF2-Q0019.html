A. AWS CodePipeline: Focuses on continuous integration and delivery. B. AWS CodeDeploy: Automates application deployments. C. AWS Direct Connect: Provides dedicated network connections. D. AWS CloudFormation: AWS CloudFormation is a service that allows you to define and provision AWS infrastructure as code. It enables you to use a template to describe and provision AWS resources in a repeatable and automated manner. With CloudFormation, you can manage your infrastructure as code, making it easier to version, replicate, and share infrastructure configurations.
