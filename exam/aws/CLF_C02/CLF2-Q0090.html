선택한 답변: C
A. AWS DataSync: 온프레미스 스토리지 시스템과 AWS 클라우드 스토리지 간에 대량의 데이터 이동을 단순화, 자동화 및 가속화하는 데이터 전송 서비스입니다. 효율적인 데이터 전송에 중점을 두고 있습니다. B. Amazon S3 Glacier: 자주 액세스하지 않는 데이터의 보관 및 장기 백업을 위해 설계된 스토리지 서비스입니다. 거의 액세스하지 않고 저렴한 스토리지가 필요한 데이터에 적합합니다. 클라우드 스토리지 서비스이기는 하지만 온프레미스 사용자가 거의 무제한의 클라우드 스토리지에 액세스할 수 있도록 특별히 설계되지 않았습니다. C. AWS Storage Gateway: 온프레미스 애플리케이션이 클라우드 스토리지를 원활하게 사용할 수 있도록 지원하는 하이브리드 클라우드 스토리지 서비스입니다. 온프레미스 환경을 AWS 클라우드 스토리지와 통합하기 위한 파일, 볼륨 및 테이프 게이트웨이 인터페이스를 제공합니다. 이 서비스를 통해 온프레미스 사용자는 하이브리드 스토리지 인프라를 유지하면서 사실상 무제한의 클라우드 스토리지에 액세스할 수 있습니다. D. Amazon Elastic Block Store(Amazon EBS): Amazon EC2 인스턴스와 함께 사용할 수 있는 블록 수준 스토리지 볼륨을 제공합니다. 주로 EC2 인스턴스에 확장 가능한 고성능 블록 스토리지를 제공하는 데 중점을 두고 있습니다.

Selected Answer: C
A. AWS DataSync: A data transfer service that simplifies, automates, and accelerates moving large amounts of data between on-premises storage systems and AWS Cloud storage. It is focused on efficient data transfer. B. Amazon S3 Glacier: A storage service designed for archiving and long-term backup of infrequently accessed data. It is suitable for data that is rarely accessed and requires low-cost storage. While it is a cloud storage service, it is not specifically designed for on-premises users to access virtually unlimited cloud storage. C. AWS Storage Gateway: A hybrid cloud storage service that enables on-premises applications to use cloud storage seamlessly. It provides file, volume, and tape gateway interfaces to integrate on-premises environments with AWS Cloud storage. This service allows on-premises users to access virtually unlimited cloud storage while maintaining a hybrid storage infrastructure. D. Amazon Elastic Block Store (Amazon EBS): Provides block-level storage volumes for use with Amazon EC2 instances. It is primarily focused on providing scalable and high-performance block storage for EC2 instances.
