B. Security groups Security groups act as virtual firewalls for your Amazon EC2 instances, controlling inbound and outbound traffic. You can configure security groups to allow or deny traffic based on rules that you define. By associating a security group with an EC2 instance, you can control the traffic to and from that instance at the network level.
