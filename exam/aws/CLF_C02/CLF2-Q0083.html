A. S3 Multi-Region Access Points: This service is designed for multi-region access to S3 data. B. S3 Storage Lens: This service provides organization-wide visibility into object storage usage and activity. C. Access Analyzer for S3: This service is designed to review and analyze access policies for S3 buckets, providing insights into who has access to your buckets and how that access is granted. It is the most suitable option for the specified requirement.
