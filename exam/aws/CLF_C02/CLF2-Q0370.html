Aurora offers storage resiliency by making data durable across 3 AZs (customers only pay for 1 copy). Aurora has up to 99.99% availability, and customers can access local read performance with Global Database when deployed across AWS Regions. With serverless, Aurora can scale-up to hundreds of thousands of transactions in a fraction of a second. Aurora zero-ETL integration with Amazon Redshift delivers analytics on transactional data in near real-time. https://aws.amazon.com/rds/aurora/
