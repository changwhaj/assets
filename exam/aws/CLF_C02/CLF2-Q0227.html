키 - "데이터베이스" AWS 데이터베이스 마이그레이션 서비스 AWS 클라우드 준비 - 자세히 요약된 6가지 AWS CAF(클라우드 채택 프레임워크) 관점에 대한 응답 및 점수를 확인하세요. AWS Migration Hub는 마이그레이션 평가, 계획 및 추적을 위해 서버 및 애플리케이션 인벤토리 데이터를 수집하는 중앙 위치를 제공합니다. AWS Database Migration Service(AWS DMS)는 데이터베이스 및 분석 워크로드를 AWS로 이동하는 데 도움이 되는 관리형 마이그레이션 및 복제 서비스입니다. 빠르고 안전하게. AWS Application Migration Service(AWS MGN)는 AWS로의 마이그레이션에 권장되는 서비스입니다. AWS Application Migration Service는 AWS에서 기본적으로 실행되도록 물리적, 가상 또는 클라우드 인프라의 소스 서버를 자동으로 변환하여 AWS로의 마이그레이션을 단순화하고 신속하게 처리합니다.

AWS Application Migration Service (AWS MGN) is the recommended service for migrations to AWS.
