A. AWS Glue: AWS Glue는 분석용 데이터를 쉽게 검색, 준비 및 로드할 수 있게 해주는 완전 관리형 ETL(추출, 변환 및 로드) 서비스입니다. 데이터 검색, 변환, 작업 예약 등 시간이 많이 소요되는 작업을 자동화하여 사용자가 데이터 분석에 집중할 수 있도록 합니다. B. Amazon Elastic File System(Amazon EFS): 파일 기반 데이터에 대한 공유 액세스를 위한 완전 관리형 파일 시스템입니다. 주로 데이터 검색, 변환 또는 시각화에 사용되지 않습니다. C. Amazon Redshift: 대규모 데이터 세트에 대한 복잡한 쿼리를 실행하기 위한 완전 관리형 데이터 웨어하우스 서비스입니다. 데이터 검색, 변환 또는 시각화에는 사용되지 않습니다. D. Amazon QuickSight: Amazon QuickSight는 사용자가 대화형 대시보드 및 보고서를 생성하고 시각화할 수 있는 완전 관리형 비즈니스 인텔리전스(BI) 서비스입니다. 다양한 데이터 소스에 연결되므로 AWS Glue와 같은 서비스에서 준비한 데이터를 시각화하는 데 적합합니다. E. Amazon Quantum Ledger Database(Amazon QLDB): 데이터 무결성을 보장하기 위한 완전 관리형 원장 데이터베이스 서비스입니다. 데이터 검색, 변환 또는 시각화용으로 설계되지 않았습니다.

A. AWS Glue: AWS Glue is a fully managed extract, transform, and load (ETL) service that makes it easy to discover, prepare, and load data for analysis. It automates the time-consuming tasks of data discovery, transformation, and job scheduling, allowing users to focus on analyzing the data. B. Amazon Elastic File System (Amazon EFS): Fully managed file system for shared access to file-based data. Not primarily used for data discovery, transformation, or visualization. C. Amazon Redshift: Fully managed data warehouse service for running complex queries on large datasets. Not used for data discovery, transformation, or visualization. D. Amazon QuickSight: Amazon QuickSight is a fully managed business intelligence (BI) service that enables users to create and visualize interactive dashboards and reports. It connects to various data sources, making it suitable for visualizing data prepared by services like AWS Glue. E. Amazon Quantum Ledger Database (Amazon QLDB): Fully managed ledger database service for ensuring data integrity. Not designed for data discovery, transformation, or visualization.
