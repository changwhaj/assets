질문 #: 381
주제 #: 1

개발자가 기본 Elastic Beanstalk 인스턴스 프로파일을 사용하여 AWS Elastic Beanstalk에서 실행되는 Java 애플리케이션을 생성했습니다. 개발자는 애플리케이션의 문제를 식별하고 디버깅하는 데 도움이 되도록 AWS 서비스와 애플리케이션의 상호 작용 맵을 시각화해야 합니다.

최소한의 운영 노력으로 이 요구 사항을 충족하기 위해 개발자가 수행해야 하는 단계 조합은 무엇입니까? (두 가지를 선택하세요.)

  • A. Java용 AWS X-Ray 소프트웨어 개발 키트(SDK)를 사용하여 코드를 계측합니다. 최다 투표
  • B. Elastic Beanstalk 구성 파일을 생성하여 기본 Amazon EC2 인스턴스에 AWS X-Ray 데몬을 다운로드하고 설치합니다.
  • C. Elastic Beanstalk 콘솔에서 AWS X-Ray 데몬을 활성화합니다. 최다 투표
  • D. Elastic Beanstalk 향상된 상태 보고를 활성화합니다.
  • E. 서비스 맵을 시각화하도록 AWS CloudTrail을 구성합니다.

A developer has created a Java application that runs on AWS Elastic Beanstalk with the default Elastic Beanstalk instance profile. The developer needs to visualize a map of the application’s interactions with AWS services to help identify and debug issues with the application.

Which combination of steps should the developer take to meet this requirement with the LEAST operational effort? (Choose two.)

  • A. Instrument the code by using the AWS X-Ray software development kit (SDK) for Java. Most Voted
  • B. Create an Elastic Beanstalk configuration file to download and install the AWS X-Ray daemon on the underlying Amazon EC2 instances.
  • C. Enable the AWS X-Ray daemon in the Elastic Beanstalk console. Most Voted
  • D. Enable Elastic Beanstalk enhanced health reporting.
  • E. Configure AWS CloudTrail to visualize the services map.
