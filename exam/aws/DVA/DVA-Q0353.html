What steps should the developer take to access the RDS DB instance from the code? (Choose two.)

  • A. Modify the endpoint name using either the AWS Management Console or AWS CLI. Most Voted
  • B. Upload the driver to Amazon S3 and reference it in the code.
  • C. Download the appropriate database driver and include it with the application. Most Voted
  • D. Construct a connection string using the Elastic Beanstalk environment variables. Most Voted Most Voted
  • E. Create a CNAME record referencing database instances ALIAS.
