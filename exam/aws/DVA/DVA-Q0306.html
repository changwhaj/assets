A developer uses server-side encryption with Amazon S3 managed encryption keys (SSE-S3) to store data in Amazon S3. The developer needs to decrypt and download the encrypted objects by using the GetObject API call.

What is the LEAST amount of information that the developer must provide in the API call to meet this requirement?

  • A. The S3 object key only Most Voted
  • B. The S3 object key and the encryption key
  • C. The S3 object key and the Amazon Resource Name (ARN) of the AWS Key Management Service (AWS KMS) key
  • D. The S3 object key and a randomly salted Hash-based Message Authentication Code (HMAC) value of the encryption key

Selected Answer: A
The correct answer is the S3 object key only. When using SSE-S3 to encrypt objects in Amazon S3, the encryption keys are managed by Amazon S3. To decrypt and download encrypted objects using the GetObject API call, the developer only needs to provide the S3 object key. Amazon S3 will use the encryption keys that it manages to decrypt the object before returning it to the developer. Providing the encryption key or the ARN of the AWS KMS key would not be necessary because the encryption keys are managed by Amazon S3. Providing a salted HMAC value of the encryption key would not be possible because the developer does not have access to the encryption key.
