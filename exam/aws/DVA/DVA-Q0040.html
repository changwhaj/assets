D. https://docs.aws.amazon.com/kms/latest/APIReference/API_GenerateDataKey.html Use the following pattern to encrypt data locally in your application: 1. Use the GenerateDataKey operation to get a data encryption key. 2. Use the plaintext data key (returned in the Plaintext field of the response) to encrypt data locally, then erase the plaintext data key from memory. 3. Store the encrypted data key (returned in the CiphertextBlob field of the response) alongside the locally encrypted data.
