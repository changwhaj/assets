주제 #: 1

개발자가 사진 웹사이트를 만들고 있습니다. Amazon Route 53은 웹 사이트의 도메인을 호스팅합니다. 개발자는 Amazon S3 버킷에 애플리케이션 코드와 이미지를 저장하려고 합니다. 또한 개발자는 Amazon CloudFront를 사용하여 사용자에게 이미지를 전달하려고 합니다.

개발자가 S3 버킷과 CloudFront 배포를 생성했습니다. 개발자는 웹사이트 도메인을 통해서만 이미지에 액세스할 수 있기를 원합니다. 사용자는 S3 URL을 사용해서는 안 됩니다.

어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. CloudFront OAI(원본 액세스 ID)를 생성합니다. OAI를 CloudFront 배포와 연결합니다. OAI에서만 액세스를 허용하도록 S3 버킷 정책을 수정합니다. 웹 사이트 도메인이 S3 버킷을 가리키는 별칭을 Route 53에 생성합니다.
  • B. CloudFront 원본 액세스 ID(OAI)를 생성합니다. OAI를 CloudFront 배포와 연결합니다. OAI에서만 액세스를 허용하도록 S3 버킷 정책을 수정합니다. 웹 사이트 도메인이 CloudFront 도메인 이름을 가리키도록 Route 53 레코드를 업데이트합니다. 최다 투표
  • C. S3 버킷 정책에서 공개 액세스를 차단합니다. S3 버킷 엔드포인트를 사용하도록 CloudFront를 구성합니다. 웹 사이트 도메인이 S3 버킷을 가리키는 별칭을 Route 53에 생성합니다.
  • D. S3 버킷 정책에서 공개 액세스를 차단합니다. S3 버킷 엔드포인트를 사용하도록 CloudFront를 구성합니다. 웹 사이트 도메인을 CloudFront 도메인 이름으로 가리키는 Route 53의 별칭을 생성합니다.

Topic #: 1

A developer is creating a photo website. Amazon Route 53 hosts the website’s domain. The developer wants to store the application code and images in an Amazon S3 bucket. The developer also wants to use Amazon CloudFront to deliver the images to users.

The developer has created the S3 bucket and a CloudFront distribution. The developer wants the images to be accessed only through the website’s domain. Users must not use the S3 URLs.

Which solution will meet these requirements?

  • A. Create a CloudFront origin access identity (OAI). Associate the OAI with the CloudFront distribution. Modify the S3 bucket policy to allow access from only the OAI. Create an alias in Route 53 that points the website domain to the S3 bucket.
  • B. Create a CloudFront origin access identity (OAI). Associate the OAI with the CloudFront distribution. Modify the S3 bucket policy to allow access from only the OAI. Update the Route 53 records to point the website domain to the CloudFront domain name. Most Voted
  • C. Block public access in the S3 bucket policy. Configure CloudFront to use the S3 bucket endpoint. Create an alias in Route 53 that points the website domain to the S3 bucket.
  • D. Block public access in the S3 bucket policy. Configure CloudFront to use the S3 bucket endpoint. Create an alias in Route 53 that points the website domain to the CloudFront domain name.
