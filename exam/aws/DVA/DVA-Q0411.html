주제 #: 1

한 전자상거래 회사가 공개적으로 액세스할 수 있는 새로운 애플리케이션을 출시하고 있습니다. 이 회사는 us-east-1 지역의 ALB(Application Load Balancer) 뒤에 있는 Amazon EC2 인스턴스에 애플리케이션을 배포했습니다. EC2 인스턴스는 Auto Scaling 그룹에 있습니다.

개발자는 AWS에서 제공하는 TLS/SSL 인증서를 사용하여 애플리케이션을 안전하게 보호해야 합니다. 또한 개발자는 Amazon Route 53에서 기존 도메인 www.example.com을 구성해야 합니다.

이러한 요구 사항을 충족하려면 개발자가 수행해야 하는 단계 조합은 무엇입니까? (2개를 선택하세요.)

  • A. ALB를 대상으로 하여 Route 53에서 A 레코드를 구성합니다. IAM 서버 인증서에서 www.example.com TLS/SSL 인증서를 요청합니다.
  • B. ALB를 대상으로 하여 Route 53에서 A 레코드를 구성합니다. AWS Certificate Manager(ACM)에서 www.example.com TLS/SSL 인증서를 요청합니다. 최다 투표
  • C. www.example.com 인증서를 HTTP 수신기에 추가합니다. www.example.com을 호스트로 사용하여 규칙을 추가합니다. Auto Scaling 그룹이 포함된 대상 그룹으로 트래픽을 전달합니다.
  • D. ALB를 대상으로 하여 Route 53에서 A 레코드를 구성합니다. Route 53에서 www.example.com TLS/SSL 인증서를 요청합니다.
  • E. ALB에 HTTPS 리스너를 생성합니다. www.example.com 인증서를 리스너에 추가합니다. www.example.com을 호스트로 사용하여 규칙을 추가합니다. Auto Scaling 그룹이 포함된 대상 그룹으로 트래픽을 전달합니다. 최다 투표

Question #: 411
Topic #: 1

An ecommerce company is launching a new application that will be publicly accessible. The company deployed the application on Amazon EC2 instances behind an Application Load Balancer (ALB) in the us-east-1 Region. The EC2 instances are in an Auto Scaling group.

A developer needs to make the application secure by using TLS/SSL certificates provisioned by AWS. The developer also needs to configure the existing domain www.example.com in Amazon Route 53.

Which combination of steps should the developer take to meet these requirements? (Choose two.)

  • A. Configure an A record in Route 53 with the ALB as the target. Request the www.example.com TLS/SSL certificate in IAM server certificates.
  • B. Configure an A record in Route 53 with the ALB as the target. Request the www.example.com TLS/SSL certificate in AWS Certificate Manager (ACM). Most Voted
  • C. Add the www.example.com certificate to the HTTP listener. Add a rule with www.example.com as the host. Forward the traffic to a target group that contains the Auto Scaling group.
  • D. Configure an A record in Route 53 with the ALB as the target. Request the www.example.com TLS/SSL certificate in Route 53.
  • E. Create an HTTPS listener in the ALB. Add the www.example.com certificate to the listener. Add a rule with www.example.com as the host. Forward the traffic to a target group that contains the Auto Scaling group. Most Voted
