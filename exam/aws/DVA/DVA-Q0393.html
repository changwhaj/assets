이러한 요구 사항을 충족하는 단계 조합은 무엇입니까? (두 가지를 선택하세요.)

  • A. Apache HTTP 서버에 Amazon CloudWatch 에이전트를 설치합니다. 로그를 Amazon Simple Queue Service(Amazon SQS) 대기열로 푸시하도록 CloudWatch 에이전트를 구성합니다.
  • B. Apache HTTP 서버에 Amazon CloudWatch 에이전트를 설치합니다. 로그를 Amazon Kinesis Data Firehose 전송 스트림으로 푸시하도록 CloudWatch 에이전트를 구성합니다. 최다 투표
  • C. Apache HTTP 서버에 Amazon Kinesis 에이전트를 설치합니다. 로그를 Amazon Kinesis Data Firehose 전송 스트림으로 푸시하도록 Kinesis 에이전트를 구성합니다. 최다 투표
  • D. 로그를 JSON 형식으로 변환하고 결과를 OpenSearch Service 클러스터로 푸시하는 AWS Lambda 함수를 생성합니다. Lambda 함수를 사용하여 Amazon Simple Queue Service(Amazon SQS) 대기열에서 로그를 사용합니다.
  • E. Amazon Kinesis Data Firehose 전송 스트림을 생성합니다. 소스를 Direct PUT으로 설정합니다. 로그를 JSON 형식으로 변환하는 AWS Lambda 함수를 구현합니다. Lambda 함수에 대한 Kinesis Data Firehose 전송 스트림에서 소스 레코드 변환을 활성화합니다. OpenSearch Service 클러스터를 대상으로 설정합니다. 최다 투표 최다 투표

Exam question from Amazon's AWS Certified Developer Associate
Question #: 393
Topic #: 1

A company needs an application that consumes logs from Apache HTTP servers at a large scale with near real-time processing. The logs will vary in size from 300 KB to 500 KB. As part of processing, the company needs to convert the logs to JSON format and then upload the logs to an Amazon OpenSearch Service cluster.

Which combination of steps will meet these requirements? (Choose two.)

  • A. Install the Amazon CloudWatch agent on the Apache HTTP servers. Configure the CloudWatch agent to push the logs to an Amazon Simple Queue Service (Amazon SQS) queue.
  • B. Install the Amazon CloudWatch agent on the Apache HTTP servers. Configure the CloudWatch agent to push the logs to the Amazon Kinesis Data Firehose delivery stream. Most Voted
  • C. Install the Amazon Kinesis agent on the Apache HTTP servers. Configure the Kinesis agent to push the logs to the Amazon Kinesis Data Firehose delivery stream. Most Voted
  • D. Create an AWS Lambda function that converts the logs to JSON format and pushes the results to the OpenSearch Service cluster. Consume the logs from the Amazon Simple Queue Service (Amazon SQS) queue by using the Lambda function.
  • E. Create an Amazon Kinesis Data Firehose delivery stream. Set the source as Direct PUT. Implement an AWS Lambda function to convert the logs to JSON format. Enable source record transformation on the Kinesis Data Firehose delivery stream for the Lambda function. Set the OpenSearch Service cluster as the destination. Most Voted Most Voted
