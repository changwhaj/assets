이 기능을 제공하기 위해 인스턴스 역할을 생성할 때 어떤 AWS IAM 변경이 필요합니까?

  • D. DynamoDB에 대한 액세스를 허용하는 역할에 연결된 IAM 권한 정책을 생성합니다. Amazon EC2가 역할을 맡을 수 있도록 역할에 신뢰 정책을 추가합니다. 개발자가 역할에 대해 iam:GetRole 권한을 사용할 수 있도록 허용하는 권한 정책을 AWS IAM의 개발 그룹에 연결합니다.
