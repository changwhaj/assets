C If you update an existing file in your origin with a newer version that has the same name, an edge location won't get that new version from your origin until both of the following occur: The old version of the file in the cache expires. There's a user request for the file at that edge location. https://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/UpdatingExistingObjects.html#ReplacingObjectsSameName
