  • C. Launch an AWS Control Tower landing zone. Create OUs and attach SCPs that deny access to run services outside of the approved Regions. Most Voted
  • D. Enable AWS Security Hub in each account. Create controls to specify the Regions where an account can deploy infrastructure.
