주제 #: 1

클라우드 팀은 AWS Organizations 및 AWS IAM Identity Center(AWS Single Sign-On)를 사용하여 회사의 AWS 계정을 관리합니다. 회사는 최근 연구팀을 꾸렸다. 연구팀은 계정의 리소스를 완벽하게 관리할 수 있는 능력이 필요합니다. 연구팀은 IAM 사용자를 생성할 수 없어야 합니다.

클라우드 팀은 연구 팀을 위해 IAM ID 센터에 연구 관리자 권한 세트를 생성합니다. 권한 세트에는 AdministratorAccess AWS 관리형 정책이 연결되어 있습니다. 클라우드 팀은 연구팀의 누구도 IAM 사용자를 생성할 수 없도록 해야 합니다.

어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. iam:CreateUser 작업을 거부하는 IAM 정책을 생성합니다. 연구 관리자 권한 집합에 IAM 정책을 연결합니다. 최다 투표
  • B. iam:CreateUser 작업을 제외한 모든 작업을 허용하는 IAM 정책을 생성합니다. IAM 정책을 사용하여 연구 관리자 권한 집합에 대한 권한 경계를 설정합니다.
  • C. iam:CreateUser 작업을 거부하는 SCP를 생성합니다. SCP를 연구팀의 AWS 계정에 연결합니다. 최다 투표
  • D. IAM 사용자를 삭제하는 AWS Lambda 함수를 생성합니다. IAM CreateUser 이벤트를 감지하는 Amazon EventBridge 규칙을 생성합니다. Lambda 함수를 호출하도록 규칙을 구성합니다.

Exam question from Amazon's AWS Certified DevOps Engineer - Professional DOP-C02
Question #: 215
Topic #: 1

A cloud team uses AWS Organizations and AWS IAM Identity Center (AWS Single Sign-On) to manage a company's AWS accounts. The company recently established a research team. The research team requires the ability to fully manage the resources in its account. The research team must not be able to create IAM users.

The cloud team creates a Research Administrator permission set in IAM Identity Center for the research team. The permission set has the AdministratorAccess AWS managed policy attached. The cloud team must ensure that no one on the research team can create IAM users.

Which solution will meet these requirements?

  • A. Create an IAM policy that denies the iam:CreateUser action. Attach the IAM policy to the Research Administrator permission set. Most Voted
  • B. Create an IAM policy that allows all actions except the iam:CreateUser action. Use the IAM policy to set the permissions boundary for the Research Administrator permission set.
  • C. Create an SCP that denies the iam:CreateUser action. Attach the SCP to the research team's AWS account. Most Voted
  • D. Create an AWS Lambda function that deletes IAM users. Create an Amazon EventBridge rule that detects the IAM CreateUser event. Configure the rule to invoke the Lambda function.
