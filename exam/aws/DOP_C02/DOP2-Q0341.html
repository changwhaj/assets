주제 #: 1

한 회사가 동일한 AWS 계정에서 여러 애플리케이션을 실행합니다. 애플리케이션은 Amazon CloudWatch에 로그를 보냅니다.

데이터 분석 팀은 애플리케이션에서 성능 지표와 사용자 지정 지표를 수집해야 합니다. 분석 팀은 Amazon S3 버킷에 데이터를 저장하기 전에 지표 데이터를 변환해야 합니다. 분석 팀은 CloudWatch 네임스페이스에 추가된 모든 새 지표를 자동으로 수집해야 합니다.

어떤 솔루션이 이러한 요구 사항을 가장 적은 운영 오버헤드로 충족할까요?

  • A. 애플리케이션과 CloudWatch 네임스페이스의 메트릭을 포함하도록 CloudWatch 메트릭 스트림을 구성합니다. 메트릭 스트림을 구성하여 메트릭을 Amazon Data Firehose 전달 스트림으로 전달합니다. Firehose 전달 스트림을 구성하여 AWS Lambda 함수를 호출하여 데이터를 변환합니다. 전달 스트림을 구성하여 변환된 데이터를 S3 버킷으로 보냅니다. 가장 많이 투표된
  • B. 모든 메트릭을 포함하고 메트릭을 Amazon Data Firehose 전달 스트림에 전달하도록 CloudWatch 메트릭 스트림을 구성합니다. Firehose 전달 스트림을 구성하여 AWS Lambda 함수를 호출하여 데이터를 변환합니다. 전달 스트림을 구성하여 변환된 데이터를 S3 버킷으로 보냅니다.
  • C. CloudWatch 로그에 대한 메트릭 필터를 구성하여 사용자 지정 메트릭을 만듭니다. CloudWatch 메트릭 스트림을 구성하여 애플리케이션 메트릭을 S3 버킷에 전달합니다.
  • D. 애플리케이션 로그 그룹에서 구독 필터를 구성하여 Amazon Data Firehose 전송 스트림을 대상으로 합니다. Firehose 전송 스트림을 구성하여 AWS Lambda 함수를 호출하여 데이터를 변환합니다. 전송 스트림을 구성하여 변환된 데이터를 S3 버킷으로 보냅니다.

Exam question from Amazon's AWS Certified DevOps Engineer - Professional DOP-C02
Question #: 341
Topic #: 1

A company runs several applications in the same AWS account. The applications send logs to Amazon CloudWatch.

A data analytics team needs to collect performance metrics and custom metrics from the applications. The analytics team needs to transform the metrics data before storing the data in an Amazon S3 bucket. The analytics team must automatically collect any new metrics that are added to the CloudWatch namespace.

Which solution will meet these requirements with the LEAST operational overhead?

  • A. Configure a CloudWatch metric stream to include metrics from the application and the CloudWatch namespace. Configure the metric stream to deliver the metrics to an Amazon Data Firehose delivery stream. Configure the Firehose delivery stream to invoke an AWS Lambda function to transform the data. Configure the delivery stream to send the transformed data to the S3 bucket. Most Voted
  • B. Configure a CloudWatch metrics stream to include all the metrics and to deliver the metrics to an Amazon Data Firehose delivery stream. Configure the Firehose delivery stream to invoke an AWS Lambda function to transform the data. Configure the delivery stream to send the transformed data to the S3 bucket.
  • C. Configure metric filters for the CloudWatch logs to create custom metrics. Configure a CloudWatch metric stream to deliver the application metrics to the S3 bucket.
  • D. Configure subscription filters on the application log groups to target an Amazon Data Firehose delivery stream. Configure the Firehose delivery stream to invoke an AWS Lambda function to transform the data. Configure the delivery stream to send the transformed data to the S3 bucket.
