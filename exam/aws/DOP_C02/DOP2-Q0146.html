회사는 AWS를 사용하며 예측 가능한 트래픽 패턴을 갖춘 중요한 컴퓨팅 인프라가 포함된 VPC를 보유하고 있습니다. 회사는 Amazon CloudWatch Logs의 로그 그룹에 게시되는 VPC 흐름 로그를 구성했습니다.

회사의 DevOps 팀은 시간이 지남에 따라 VPC에 대한 네트워크 트래픽의 이상 현상을 식별하기 위해 VPC 흐름 로그에 대한 모니터링 솔루션을 구성해야 합니다. 모니터링 솔루션이 이상 징후를 감지하면 회사는 이상 징후에 대한 대응을 시작할 수 있는 능력이 필요합니다.

DevOps 팀은 이러한 요구 사항을 충족하기 위해 모니터링 솔루션을 어떻게 구성해야 합니까?

  • A. Amazon Kinesis 데이터 스트림을 생성합니다. 로그 그룹을 데이터 스트림에 등록합니다. 데이터 스트림에서 로그 변칙을 감지하도록 Amazon Kinesis Data Analytics를 구성합니다. 데이터 스트림의 출력으로 사용할 AWS Lambda 함수를 생성합니다. 이상 징후가 발견될 경우 기본 Amazon EventBridge 이벤트 버스에 쓰도록 Lambda 함수를 구성합니다. 최다 투표
  • B. Amazon S3 버킷에 이벤트를 전달하는 Amazon Kinesis Data Firehose 전송 스트림을 생성합니다. 로그 그룹을 전송 스트림에 등록합니다. S3 버킷의 데이터에 이상이 있는지 모니터링하도록 Amazon Lookout for Metrics를 구성합니다. Lookout for Metrics 이상 징후 발견에 대한 응답으로 실행할 AWS Lambda 함수를 생성합니다. 기본 Amazon EventBridge 이벤트 버스에 게시하도록 Lambda 함수를 구성합니다. 최다 투표
  • C. 이상 징후를 탐지하는 AWS Lambda 함수를 생성합니다. Lambda 함수가 이상 현상을 감지한 경우 기본 Amazon EventBridge 이벤트 버스에 이벤트를 게시하도록 Lambda 함수를 구성합니다. 로그 그룹에 Lambda 함수를 구독합니다.
  • D. Amazon Kinesis 데이터 스트림을 생성합니다. 로그 그룹을 데이터 스트림에 등록합니다. 로그 이상을 감지하는 AWS Lambda 함수를 생성합니다. Lambda 함수가 이상 현상을 감지하면 기본 Amazon EventBridge 이벤트 버스에 쓰도록 Lambda 함수를 구성합니다. Lambda 함수를 데이터 스트림의 프로세서로 설정합니다.

Topic #: 1

A company uses AWS and has a VPC that contains critical compute infrastructure with predictable traffic patterns. The company has configured VPC flow logs that are published to a log group in Amazon CloudWatch Logs.

The company's DevOps team needs to configure a monitoring solution for the VPC flow logs to identify anomalies in network traffic to the VPC over time. If the monitoring solution detects an anomaly, the company needs the ability to initiate a response to the anomaly.

How should the DevOps team configure the monitoring solution to meet these requirements?

  • A. Create an Amazon Kinesis data stream. Subscribe the log group to the data stream. Configure Amazon Kinesis Data Analytics to detect log anomalies in the data stream. Create an AWS Lambda function to use as the output of the data stream. Configure the Lambda function to write to the default Amazon EventBridge event bus in the event of an anomaly finding. Most Voted
  • B. Create an Amazon Kinesis Data Firehose delivery stream that delivers events to an Amazon S3 bucket. Subscribe the log group to the delivery stream. Configure Amazon Lookout for Metrics to monitor the data in the S3 bucket for anomalies. Create an AWS Lambda function to run in response to Lookout for Metrics anomaly findings. Configure the Lambda function to publish to the default Amazon EventBridge event bus. Most Voted
  • C. Create an AWS Lambda function to detect anomalies. Configure the Lambda function to publish an event to the default Amazon EventBridge event bus if the Lambda function detects an anomaly. Subscribe the Lambda function to the log group.
  • D. Create an Amazon Kinesis data stream. Subscribe the log group to the data stream. Create an AWS Lambda function to detect log anomalies. Configure the Lambda function to write to the default Amazon EventBridge event bus if the Lambda function detects an anomaly. Set the Lambda function as the processor for the data stream.
