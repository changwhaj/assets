주제 #: 1

회사는 Amazon EC2 및 온프레미스 구성으로 애플리케이션을 실행합니다. DevOps 엔지니어는 두 환경 모두에서 패치를 표준화해야 합니다. 회사 정책에 따르면 패치는 업무 시간 외에만 수행됩니다.
이러한 요구 사항을 충족하는 작업 조합은 무엇입니까? (3개를 선택하세요.)

  • A. Systems Manager 하이브리드 활성화를 사용하여 AWS Systems Manager에 물리적 시스템을 추가합니다. 최다 투표
  • B. EC2 인스턴스에 IAM 역할을 연결하여 AWS Systems Manager에서 관리할 수 있도록 합니다. 최다 투표
  • C. 온프레미스 머신이 AWS 시스템 관리자와 상호 작용할 수 있도록 IAM 액세스 키를 생성합니다.
  • D. AWS 시스템 관리자 자동화 문서를 실행하여 매시간 시스템을 패치합니다.
  • E. Amazon EventBridge 예약 이벤트를 사용하여 패치 기간을 예약합니다.
  • F. AWS 시스템 관리자 유지 관리 기간을 사용하여 패치 기간을 예약합니다. 최다 투표

  • D. Run an AWS Systems Manager Automation document to patch the systems every hour
  • E. Use Amazon EventBridge scheduled events to schedule a patch window.
  • F. Use AWS Systems Manager Maintenance Windows to schedule a patch window. Most Voted
