Amazon의 AWS Certified DevOps Engineer - Professional DOP-C02 의 시험 문제
질문 #: 325
주제 #: 1

한 회사가 Amazon Redshift를 데이터 웨어하우스 솔루션으로 사용합니다. 이 회사는 Redshift 사용자와 사용자가 수행하는 쿼리에 대한 변경 사항을 볼 수 있는 대시보드를 만들고자 합니다.

이 요구 사항을 충족하는 단계 조합은 무엇입니까? (두 가지를 선택하세요.)

  • A. Amazon CloudWatch 로그 그룹을 만듭니다. CloudWatch 로그 그룹에 쓰는 AWS CloudTrail 트레일을 만듭니다.
  • B. 새 Amazon S3 버킷을 만듭니다. Redshift 클러스터에서 기본 감사 로깅을 구성합니다. S3 버킷을 대상으로 구성합니다.
  • C. Redshift 클러스터 데이터베이스 감사 로깅을 구성하여 사용자 활동 로그를 포함합니다. Amazon CloudWatch를 대상으로 구성합니다. 가장 많이 투표된
  • D. 로그 위젯이 있는 Amazon CloudWatch 대시보드를 만듭니다. Redshift 로그에서 사용자 세부 정보를 표시하도록 위젯을 구성합니다. 가장 많이 투표된
  • E. Amazon Athena를 사용하여 Redshift 로그를 쿼리하는 AWS Lambda 함수를 만듭니다. Lambda 함수를 사용하는 사용자 지정 위젯 유형이 있는 Amazon CloudWatch 대시보드를 만듭니다.

Topic #: 1

A company uses Amazon Redshift as its data warehouse solution. The company wants to create a dashboard to view changes to the Redshift users and the queries the users perform.

Which combination of steps will meet this requirement? (Choose two.)

  • A. Create an Amazon CloudWatch log group. Create an AWS CloudTrail trail that writes to the CloudWatch log group.
  • B. Create a new Amazon S3 bucket. Configure default audit logging on the Redshift cluster. Configure the S3 bucket as the target.
  • C. Configure the Redshift cluster database audit logging to include user activity logs. Configure Amazon CloudWatch as the target. Most Voted
  • D. Create an Amazon CloudWatch dashboard that has a log widget. Configure the widget to display user details from the Redshift logs. Most Voted
  • E. Create an AWS Lambda function that uses Amazon Athena to query the Redshift logs. Create an Amazon CloudWatch dashboard that has a custom widget type that uses the Lambda function.
