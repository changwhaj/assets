Amazon의 AWS Certified DevOps Engineer - Professional DOP-C02 의 시험 문제
질문 #: 334
주제 #: 1

한 회사가 AWS Organizations에서 조직을 사용하여 여러 AWS 계정을 관리합니다. 이 회사는 조직에서 사용할 수 있는 모든 기능을 활성화했습니다. 이 회사는 루트 OU 아래의 OU 계층으로 조직을 구성했습니다. 이 회사는 최근에 모든 OU를 등록하고 모든 AWS 계정을 AWS Control Tower에 등록했습니다.

이 회사는 각 회사의 AWS 계정에서 AWS Control Tower 관리 AWS Config 구성 레코더를 사용자 지정해야 합니다. 이 회사는 기존 AWS 계정과 회사가 앞으로 AWS Control Tower에 등록하는 모든 새 AWS 계정에 사용자 지정을 적용해야 합니다.

이러한 요구 사항을 충족하는 단계 조합은 무엇입니까? (세 가지를 선택하십시오.)

  • A. 새 AWS 계정을 만듭니다. 조직의 각 AWS 계정에서 AWS Config 구성 레코더에 사용자 정의를 적용하기 위해 새 계정에서 AWS Lambda 함수를 만듭니다. 가장 많이 투표된
  • B. AWS Config 위임 관리자로 새 AWS 계정을 만듭니다. 위임 관리자 계정에서 AWS Lambda 함수를 만들어 위임 관리자 계정의 AWS Config 구성 레코더에 사용자 지정을 적용합니다.
  • C. AWS Control Tower 관리 계정에서 Amazon EventBridge 규칙을 구성하여 Organizations OU가 등록되거나 재등록될 때 AWS Lambda 함수를 호출합니다. 루트 Organizations OU를 재등록합니다.
  • D. 조직의 각 AWS 계정에서 AWSControlTowerExecution IAM 역할을 AWS Lambda 함수에서 맡을 수 있도록 구성합니다. Lambda 함수가 AWSControlTowerExecution IAM 역할을 맡도록 구성합니다.
  • E. AWS Lambda 함수가 맡을 수 있는 AWS Control Tower 관리 계정에서 IAM 역할을 만듭니다. 조직의 모든 계정에서 AWSControlTowerExecution IAM 역할을 맡을 수 있는 IAM 역할 권한을 부여합니다. Lambda 함수가 새 IAM 역할을 사용하도록 구성합니다. 가장 많이 투표된
  • F. AWS Control Tower 관리 계정에서 Amazon EventBridge 규칙을 구성하여 AWS 계정이 업데이트되거나 AWS Control Tower에 등록되거나 랜딩 존이 업데이트될 때 AWS Lambda 함수를 호출합니다. 조직의 각 Organizations OU를 다시 등록합니다. 가장 많이 투표된

Exam question from Amazon's AWS Certified DevOps Engineer - Professional DOP-C02
Question #: 334
Topic #: 1

A company use an organization in AWS Organizations to manage multiple AWS accounts. The company has enabled all features enabled for the organization. The company configured the organization as a hierarchy of OUs under the root OU. The company recently registered all its OUs and enrolled all its AWS accounts in AWS Control Tower.

The company needs to customize the AWS Control Tower managed AWS Config configuration recorder in each of the company's AWS accounts. The company needs to apply the customizations to both the existing AWS accounts and to any new AWS accounts that the company enrolls in AWS Control Tower in the future.

Which combination of steps will meet these requirements? (Choose three.)

  • A. Create a new AWS account. Create an AWS Lambda function in the new account to apply the customizations to the AWS Config configuration recorder in each AWS account in the organization. Most Voted
  • B. Create a new AWS account as an AWS Config delegated administrator. Create an AWS Lambda function in the delegated administrator account to apply the customizations to the AWS Config configuration recorder in the delegated administrator account.
  • C. Configure an Amazon EventBridge rule in the AWS Control Tower management account to invoke an AWS Lambda function when the Organizations OU is registered or reregistered. Re-register the root Organizations OU.
  • D. Configure the AWSControlTowerExecution IAM role in each AWS account in the organization to be assumable by an AWS Lambda function. Configure the Lambda function to assume the AWSControlTowerExecution IAM role.
  • E. Create an IAM role in the AWS Control Tower management account that an AWS Lambda function can assume. Grant the IAM role permission to assume the AWSControlTowerExecution IAM role in any account in the organization. Configure the Lambda function to use the new IAM role. Most Voted
  • F. Configure an Amazon EventBridge rule in the AWS Control Tower management account to invoke an AWS Lambda function when an AWS account is updated or enrolled in AWS Control Tower or when the landing zone is updated. Re-register each Organizations OU in the organization. Most Voted

B, E, and F. B: AWS Config supports delegated administrators, allowing a central account to manage configurations across the organization. By creating a Lambda function in the delegated administrator account, you can apply the customizations to the AWS Config configuration recorder in all member accounts centrally. E: The AWSControlTowerExecution IAM role exists in each enrolled account and allows centralized operations. The IAM role in the management account needs permissions to assume the AWSControlTowerExecution role in member accounts. F: AWS Control Tower emits events when an account is enrolled or updated, or when the landing zone is updated. An EventBridge rule can trigger the Lambda function to ensure that any new or updated accounts automatically receive the customizations. Re-registering each OU ensures that Control Tower applies its governance to all accounts and OUs consistently.
