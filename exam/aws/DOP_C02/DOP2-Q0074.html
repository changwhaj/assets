주제 #: 1

회사는 AWS CodeBuild 프로젝트를 사용하여 애플리케이션을 구축하고 패키징하고 있습니다. 패키지는 여러 AWS 계정에 배포되기 전에 공유 Amazon S3 버킷에 복사됩니다.
buildspec.yml 파일에는 다음 내용이 포함되어 있습니다.

DevOps 엔지니어는 AWS 계정이 있는 사람은 누구나 아티팩트를 다운로드할 수 있음을 확인했습니다.
이를 중지하려면 DevOps 엔지니어가 어떤 조치를 취해야 합니까?

  • A. --acl public-read를 사용하도록 post_build 명령을 수정하고 관련 AWS 계정에만 읽기 액세스 권한을 부여하는 버킷 정책을 구성합니다.
  • B. 인증된 사용자 집합을 관련 AWS 계정으로만 정의하고 읽기 전용 액세스 권한을 부여하는 S3 버킷에 대한 기본 ACL을 구성합니다.
  • C. 관련 AWS 계정에 읽기 액세스 권한을 부여하고 보안 주체 "*"에 대한 읽기 액세스를 거부하는 S3 버킷 정책을 생성합니다.
  • D. post_build 명령을 수정하여 --acl authenticated-read를 제거하고 관련 AWS 계정에만 읽기 액세스를 허용하는 버킷 정책을 구성합니다. 최다 투표
