What should the solutions architect recommend to meet these requirements?

  • A. Enable termination protection for the EC2 instance.
  • B. Configure the EC2 instance for Multi-AZ deployment.
  • C. Create an Amazon CloudWatch alarm to recover the EC2 instance in case of failure.
  • D. Launch the EC2 instance with two Amazon Elastic Block Store (Amazon EBS) volumes that use RAID configurations for storage redundancy. Most Voted

선택한 답변: C
A. EC2 인스턴스에 대한 종료 방지 기능을 활성화합니다. 아니요. 종료 방지는 실수로 인스턴스를 삭제하는 것을 방지하는 것입니다. B. 다중 AZ 배포를 위해 EC2 인스턴스를 구성합니다. 아니요. 질문에 "둘 이상의 인스턴스에서 실행할 수 없습니다"라고 나와 있습니다. C. 오류가 발생한 경우 EC2 인스턴스를 복구하기 위해 Amazon CloudWatch 경보를 생성합니다. 예. CloudWatch를 사용하여 인스턴스를 복구할 수 있습니다: https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/UsingAlarmActions.html#AddingRecoverActions D. 두 개의 Amazon Elastic Block Store(Amazon EBS) 볼륨을 사용하여 EC2 인스턴스를 시작합니다. 스토리지 중복성을 위해 RAID 구성을 사용하십시오. 아니요. Raid는 회복력을 높이는 데 도움이 될 수 있지만 "복구 시간 개선"에는 도움이 되지 않습니다.
