  • D. Use On-Demand Instances in an Amazon Elastic Kubernetes Service (Amazon EKS) managed node group.

Selected Answer: B
The key reasons are: Using Spot Instances reduces EC2 costs significantly compared to On-Demand. EKS managed node groups simplify running and scaling containerized applications vs self-managed Kubernetes.
