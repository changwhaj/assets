The reasons are: CloudFront is a content delivery network (CDN) that caches content at edge locations around the world. Caching the video content globally brings it closer to viewers, reducing latency. This improves performance for both live streaming and on-demand playback for the global audience. Route 53 provides DNS resolution but does not cache content locally. Global Accelerator improves TCP traffic routing performance but is not a caching CDN. S3 Transfer Acceleration optimizes uploads to S3 over long distances but does not help with content delivery.
