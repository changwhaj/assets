정답은 A입니다. AWS Certificate Manager(ACM)를 사용하여 SSL/TLS 인증서를 발급합니다. 인증서를 ALB에 적용합니다. 관리형 갱신 기능을 사용하여 인증서를 자동으로 교체합니다. AWS Certificate Manager(ACM)는 AWS 리소스에서 사용할 Secure Sockets Layer/Transport Layer Security(SSL/TLS) 인증서를 쉽게 프로비저닝, 관리 및 배포할 수 있는 서비스입니다. ACM은 SSL/TLS 인증서에 대한 관리형 갱신을 제공하므로 ACM이 만료되기 전에 인증서를 자동으로 갱신합니다. 웹 애플리케이션의 요구 사항을 충족하려면 ACM을 사용하여 SSL/TLS 인증서를 발급하고 이를 Application Load Balancer(ALB)에 적용해야 합니다. 그런 다음 관리형 갱신 기능을 사용하여 만료되기 전에 매년 인증서를 자동으로 교체할 수 있습니다. 이렇게 하면 웹 애플리케이션이 항상 유효한 SSL/TLS 인증서로 에지에서 암호화됩니다.
