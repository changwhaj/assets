Which solution will meet these requirements with the LEAST operational overhead?

App2Container: A tool to automatically containerize .NET applications. Fargate: A serverless compute engine for containers, eliminating the need to manage underlying EC2 instances. It integrates with CloudWatch for scaling based on custom metrics. App Runner: A fully managed service designed for simple application deployment and scaling, but it also deploys applications *directly*, which means it doesn't deploy to ECS or EKS.
