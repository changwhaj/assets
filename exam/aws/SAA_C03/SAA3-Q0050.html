AWS Systems Manager의 기능인 Patch Manager의 주요 초점은 관리되는 노드에 운영 체제 보안 관련 업데이트를 설치하는 것입니다. 기본적으로 Patch Manager는 사용 가능한 모든 패치를 설치하지 않고 보안에 초점을 맞춘 소규모 패치 세트를 설치합니다. (참조 https://docs.aws.amazon.com/systems-manager/latest/userguide/execute-remote-commands.html) Patch Manager는 타사 애플리케이션이 아닌 OS 수준 패치를 위한 것 같습니다. 그리고 이는 대규모로 일회성 구성 변경(3번째 부분 애플리케이션 업데이트)을 수행하기 위한 run command 휠하우스에 속합니다.
