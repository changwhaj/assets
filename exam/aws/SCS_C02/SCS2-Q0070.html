Amazon AWS Certified Security - Specialty SCS-C02 시험문제
질문 #: 70
주제 #: 1

회사에는 인터넷 게이트웨이가 연결되지 않은 VPC의 단일 프라이빗 서브넷에 Amazon EC2 인스턴스 그룹이 있습니다. 보안 엔지니어는 특정 애플리케이션에서 로그를 캡처하기 위해 해당 서브넷의 모든 인스턴스에 Amazon CloudWatch 에이전트를 설치했습니다. 로그가 안전하게 흐르도록 하기 위해 회사의 네트워킹 팀은 CloudWatch 모니터링 및 CloudWatch 로그를 위한 VPC 엔드포인트를 생성했습니다. 네트워킹 팀이 엔드포인트를 VPC에 연결했습니다.

애플리케이션이 로그를 생성하고 있습니다. 그러나 보안 엔지니어가 CloudWatch에 쿼리하면 로그가 표시되지 않습니다.

이 문제를 해결하려면 보안 엔지니어가 수행해야 하는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. EC2 인스턴스에 연결된 EC2 인스턴스 프로필에 로그 스트림을 생성하고 로그를 쓸 수 있는 권한이 있는지 확인하세요. 최다 투표
  • B. AWS Management Console에서 볼 수 있도록 로그에 지표 필터를 생성합니다.
  • C. 각 EC2 인스턴스의 CloudWatch 에이전트 구성 파일을 확인하여 CloudWatch 에이전트가 적절한 로그 파일을 수집하고 있는지 확인하세요. 최다 투표
  • D. 두 VPC 엔드포인트의 VPC 엔드포인트 정책을 확인하여 EC2 인스턴스에 이를 사용할 권한이 있는지 확인하십시오. 최다 투표
  • E. EC2 인스턴스가 CloudWatch와 통신할 수 있도록 서브넷에 NAT 게이트웨이를 생성합니다.
  • F. 보안 그룹이 모든 EC2 인스턴스가 전송하기 전에 로그를 집계하기 위해 서로 통신할 수 있도록 허용하는지 확인하십시오.

Exam question from Amazon's AWS Certified Security - Specialty SCS-C02
Question #: 70
Topic #: 1

A company has a group of Amazon EC2 instances in a single private subnet of a VPC with no internet gateway attached. A security engineer has installed the Amazon CloudWatch agent on all instances in that subnet to capture logs from a specific application. To ensure that the logs flow securely, the company's networking team has created VPC endpoints for CloudWatch monitoring and CloudWatch logs. The networking team has attached the endpoints to the VPC.

The application is generating logs However, when the security engineer queries CloudWatch, the logs do not appear.

Which combination of steps should the security engineer take to troubleshoot this issue? (Choose three.)

  • A. Ensure that the EC2 instance profile that is attached to the EC2 instances has permissions to create log streams and write logs. Most Voted
  • B. Create a metric filter on the logs so that they can be viewed in the AWS Management Console.
  • C. Check the CloudWatch agent configuration file on each EC2 instance to make sure that the CloudWatch agent is collecting the proper log files. Most Voted
  • D. Check the VPC endpoint policies of both VPC endpoints to ensure that the EC2 instances have permissions to use them. Most Voted
  • E. Create a NAT gateway in the subnet so that the EC2 instances can communicate with CloudWatch.
  • F. Ensure that the security groups allow all the EC2 instances to communicate with each other to aggregate logs before sending.
