Amazon AWS Certified Security - Specialty SCS-C02 시험문제
질문 #: 56
주제 #: 1

하이브리드 클라우드 환경에서 운영되는 회사는 엄격한 규정 준수 요구 사항을 충족해야 합니다. 회사는 AWS 리소스의 증거와 함께 온프레미스 워크로드의 증거를 포함하는 보고서를 생성하려고 합니다. 보안 엔지니어는 회사 정책 준수를 입증하기 위해 증거를 수집, 검토 및 관리하는 솔루션을 구현해야 합니다.
어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. 사전 구축된 프레임워크 또는 사용자 지정 프레임워크를 통해 AWS Audit Manager에서 평가를 생성합니다. 온프레미스 워크로드에서 수동 증거를 업로드합니다. 평가에 증거를 추가합니다. Audit Manager가 AWS 리소스에서 필요한 증거를 수집한 후 평가 보고서를 생성합니다. 최다 투표
  • B. 온프레미스 워크로드에 Amazon CloudWatch 에이전트를 설치합니다. AWS Config를 사용하여 샘플 적합성 팩 템플릿 또는 사용자 지정 YAML 템플릿에서 적합성 팩을 배포합니다. AWS Config가 규정을 준수하지 않는 워크로드 및 리소스를 식별한 후 평가 보고서를 생성합니다.
  • C. AWS Security Hub에서 적절한 보안 표준을 설정합니다. 온프레미스 워크로드에서 수동 증거를 업로드합니다. Security Hub가 AWS 리소스에서 증거를 수집할 때까지 기다립니다. 컨트롤 목록을 .csv 파일로 다운로드합니다.
  • D. 온프레미스 워크로드에 Amazon CloudWatch 에이전트를 설치합니다. 온프레미스 워크로드와 AWS 리소스를 모니터링하는 CloudWatch 대시보드를 생성합니다. 워크로드 및 리소스에 대해 쿼리를 실행합니다. 결과를 다운로드하세요.

Exam question from Amazon's AWS Certified Security - Specialty SCS-C02
Question #: 56
Topic #: 1

A company that operates in a hybrid cloud environment must meet strict compliance requirements. The company wants to create a report that includes evidence from on-premises workloads alongside evidence from AWS resources. A security engineer must implement a solution to collect, review, and manage the evidence to demonstrate compliance with company policy.
Which solution will meet these requirements?

  • A. Create an assessment in AWS Audit Manager from a prebuilt framework or a custom framework. Upload manual evidence from the on-premises workloads. Add the evidence to the assessment. Generate an assessment report after Audit Manager collects the necessary evidence from the AWS resources. Most Voted
  • B. Install the Amazon CloudWatch agent on the on-premises workloads. Use AWS Config to deploy a conformance pack from a sample conformance pack template or a custom YAML template. Generate an assessment report after AWS Config identifies noncompliant workloads and resources.
  • C. Set up the appropriate security standard in AWS Security Hub. Upload manual evidence from the on-premises workloads. Wait for Security Hub to collect the evidence from the AWS resources. Download the list of controls as a .csv file.
  • D. Install the Amazon CloudWatch agent on the on-premises workloads. Create a CloudWatch dashboard to monitor the on-premises workloads and the AWS resources. Run a query on the workloads and resources. Download the results.
