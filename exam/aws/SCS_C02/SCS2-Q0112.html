Amazon AWS Certified Security - Specialty SCS-C02 시험문제
질문 #: 112
주제 #: 1

회사는 AWS Organizations의 조직에 AWS 계정을 가지고 있습니다. 조직에는 전용 보안 계정이 포함되어 있습니다.

모든 회원 계정의 모든 AWS 계정 활동은 기록되어 전용 보안 계정에 보고되어야 합니다. 회사는 모든 활동 로그를 전용 보안 계정 내의 안전한 저장 위치에 2년 동안 보관해야 합니다. 로그의 변경이나 삭제는 허용되지 않습니다.

최소한의 운영 오버헤드로 이러한 요구 사항을 충족하는 단계 조합은 무엇입니까? (2개를 선택하세요.)

  • A. 전용 보안 계정에서 Amazon S3 버킷을 생성하십시오. 규정 준수 모드에서 S3 객체 잠금을 구성하고 S3 버킷에 보존 기간을 2년으로 설정합니다. 조직의 마스터 계정이 S3 버킷에 쓸 수 있도록 버킷 정책을 설정합니다.
  • B. 전용 보안 계정에서 Amazon S3 버킷을 생성합니다. 규정 준수 모드에서 S3 객체 잠금을 구성하고 S3 버킷에 보존 기간을 2년으로 설정합니다. 조직의 회원 계정이 S3 버킷에 쓸 수 있도록 버킷 정책을 설정합니다. 최다 투표
  • C. 전용 보안 계정에서 2년 후에 객체가 만료되는 S3 수명 주기 구성이 포함된 Amazon S3 버킷을 생성합니다. 조직의 회원 계정이 S3 버킷에 쓸 수 있도록 버킷 정책을 설정합니다.
  • D. 조직을 위한 AWS CloudTrail 추적을 생성합니다. 전용 보안 계정의 로깅 Amazon S3 버킷으로 전달되도록 로그를 구성합니다. 최다 투표
  • E. 각 계정에서 AWS CloudTrail을 활성화합니다. 조직의 마스터 계정에서 생성된 Amazon S3 버킷으로 전달되도록 로그를 구성합니다. AWS Lambda 및 Amazon Kinesis Data Firehose를 사용하여 전용 보안 계정의 S3 버킷으로 로그를 전달합니다.

Exam question from Amazon's AWS Certified Security - Specialty SCS-C02
Question #: 112
Topic #: 1

A company has AWS accounts in an organization in AWS Organizations. The organization includes a dedicated security account.

All AWS account activity across all member accounts must be logged and reported to the dedicated security account. The company must retain all the activity logs in a secure storage location within the dedicated security account for 2 years. No changes or deletions of the logs are allowed.

Which combination of steps will meet these requirements with the LEAST operational overhead? (Choose two.)

  • A. In the dedicated security account, create an Amazon S3 bucket. Configure S3 Object Lock in compliance mode and a retention period of 2 years on the S3 bucket. Set the bucket policy to allow the organization's management account to write to the S3 bucket.
  • B. In the dedicated security account, create an Amazon S3 bucket. Configure S3 Object Lock in compliance mode and a retention period of 2 years on the S3 bucket. Set the bucket policy to allow the organization's member accounts to write to the S3 bucket. Most Voted
  • C. In the dedicated security account, create an Amazon S3 bucket that has an S3 Lifecycle configuration that expires objects after 2 years. Set the bucket policy to allow the organization's member accounts to write to the S3 bucket.
  • D. Create an AWS CloudTrail trail for the organization. Configure logs to be delivered to the logging Amazon S3 bucket in the dedicated security account. Most Voted
  • E. Turn on AWS CloudTrail in each account. Configure logs to be delivered to an Amazon S3 bucket that is created in the organization's management account. Forward the logs to the S3 bucket in the dedicated security account by using AWS Lambda and Amazon Kinesis Data Firehose.
