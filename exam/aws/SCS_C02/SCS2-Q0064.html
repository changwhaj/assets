Amazon AWS Certified Security - Specialty SCS-C02 시험문제
질문 #: 64
주제 #: 1

한 회사가 민감한 정보를 저장할 애플리케이션을 AWS에 구축하고 있습니다. 회사에는 데이터베이스를 포함한 IT 인프라에 대한 액세스 권한을 갖춘 지원 팀이 있습니다. 회사의 보안 엔지니어는 관리 오버헤드를 최소화하면서 데이터 침해로부터 민감한 데이터를 보호하기 위한 조치를 도입해야 합니다. 자격 증명은 정기적으로 교체되어야 합니다.
보안 엔지니어는 무엇을 추천해야 합니까?

  • A. Amazon RDS 암호화를 활성화하여 데이터베이스와 스냅샷을 암호화합니다. Amazon EC2 인스턴스에서 Amazon Elastic Block Store(Amazon EBS) 암호화를 활성화합니다. EC2 사용자 데이터 필드에 데이터베이스 자격 증명을 포함합니다. AWS Lambda 함수를 사용하여 데이터베이스 자격 증명을 교체합니다. 데이터베이스 연결을 위해 TLS를 설정합니다.
  • B. Amazon EC2 인스턴스에 데이터베이스를 설치합니다. Amazon Elastic Block Store(Amazon EBS) 볼륨을 암호화하려면 타사 디스크 암호화를 활성화하십시오. 자동 교체를 통해 AWS CloudHSM에 데이터베이스 자격 증명을 저장합니다. 데이터베이스 연결을 위해 TLS를 설정합니다.
  • C. Amazon RDS 암호화를 활성화하여 데이터베이스와 스냅샷을 암호화합니다. Amazon EC2 인스턴스에서 Amazon Elastic Black Store(Amazon EBS) 암호화를 활성화합니다. 자동 교체를 통해 AWS Secrets Manager에 데이터베이스 자격 증명을 저장합니다. RDS 호스팅 데이터베이스에 대한 연결을 위해 TLS를 설정합니다. 최다 투표
  • D. KMS 키를 저장하기 위해 AWS Key Management Service(AWS KMS)를 사용하여 AWS CloudHSM 클러스터를 설정합니다. AWS KMS를 사용하여 Amazon RDS 암호화를 설정하여 데이터베이스를 암호화합니다. 자동 교체를 통해 AWS Systems Manager Parameter Store에 데이터베이스 자격 증명을 저장합니다. RDS 호스팅 데이터베이스에 대한 연결을 위해 TLS를 설정합니다.

Exam question from Amazon's AWS Certified Security - Specialty SCS-C02
Question #: 64
Topic #: 1

A company is building an application on AWS that will store sensitive information. The company has a support team with access to the IT infrastructure, including databases. The company’s security engineer must introduce measures to protect the sensitive data against any data breach while minimizing management overhead. The credentials must be regularly rotated.
What should the security engineer recommend?

  • A. Enable Amazon RDS encryption to encrypt the database and snapshots. Enable Amazon Elastic Block Store (Amazon EBS) encryption on Amazon EC2 instances. Include the database credential in the EC2 user data field. Use an AWS Lambda function to rotate database credentials. Set up TLS for the connection to the database.
  • B. Install a database on an Amazon EC2 instance. Enable third-party disk encryption to encrypt the Amazon Elastic Block Store (Amazon EBS) volume. Store the database credentials in AWS CloudHSM with automatic rotation. Set up TLS for the connection to the database.
  • C. Enable Amazon RDS encryption to encrypt the database and snapshots. Enable Amazon Elastic Black Store (Amazon EBS) encryption on Amazon EC2 instances. Store the database credentials in AWS Secrets Manager with automatic rotation. Set up TLS for the connection to the RDS hosted database. Most Voted
  • D. Set up an AWS CloudHSM cluster with AWS Key Management Service (AWS KMS) to store KMS keys. Set up Amazon RDS encryption using AWS KMS to encrypt the database. Store database credentials in the AWS Systems Manager Parameter Store with automatic rotation. Set up TLS for the connection to the RDS hosted database.
