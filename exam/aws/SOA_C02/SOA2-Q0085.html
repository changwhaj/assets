Description You get the InstanceLimitExceeded error when you try to launch a new instance or restart a stopped instance. Cause If you get an InstanceLimitExceeded error when you try to launch a new instance or restart a stopped instance, you have reached the limit on the number of instances that you can launch in a Region. When you create your AWS account, we set default limits on the number of instances you can run on a per-Region basis. Solution You can request an instance limit increase on a per-region basis. https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/troubleshooting-launch.html#troubleshooting-launch-limit
