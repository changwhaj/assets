It's no longer call "Defined Duration". Now we would use "Persistent Request" not "one-time" request. https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/spot-requests.html Request type: The Spot Instance request type that you choose determines what happens if your Spot Instance is interrupted. One-time: Amazon EC2 places a one-time request for your Spot Instance. If your Spot Instance is interrupted, the request is not resubmitted. Persistent request: Amazon EC2 places a persistent request for your Spot Instance. If your Spot Instance is interrupted, the request is resubmitted to replenish the interrupted Spot Instance.
