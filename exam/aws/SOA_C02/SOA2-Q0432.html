Amazon AWS 공인 SysOps 관리자 – 어소시에이트 의 시험 문제
질문 #: 432
주제 #: 1

SysOps 관리자가 새로운 Amazon EC2 인스턴스를 퍼블릭 서브넷의 웹 서버로 설정했습니다. 인스턴스는 HTTP 포트 80 및 HTTPS 포트 443을 사용합니다.

SysOps 관리자는 공개 리포지토리에서 운영 체제 업데이트 및 소프트웨어를 다운로드하여 인터넷 연결을 확인했습니다. 그러나 SysOps 관리자는 인터넷의 웹 브라우저에서 인스턴스에 액세스할 수 없습니다.

이 문제를 해결하려면 SysOps 관리자가 수행해야 하는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. 인스턴스 보안 그룹의 인바운드 규칙이 포트 80 및 443의 트래픽을 허용하는지 확인하십시오. 최다 투표
  • B. 인스턴스 보안 그룹의 아웃바운드 규칙이 포트 80 및 443의 트래픽을 허용하는지 확인하십시오.
  • C. 인스턴스의 서브넷과 연결된 네트워크 ACL의 인바운드 규칙에서 임시 포트 1024-65535가 허용되는지 확인하십시오.
  • D. 인스턴스의 서브넷과 연결된 네트워크 ACL의 아웃바운드 규칙에서 임시 포트 1024-65535가 허용되는지 확인하십시오. 최다 투표
  • E. 인스턴스에서 실행 중인 모든 방화벽에 대한 필터링 규칙이 포트 80 및 443에서 인바운드 트래픽을 허용하는지 확인하십시오. 최다 투표
  • F. 인스턴스에 대해 AWS WAF가 켜져 있고 웹 트래픽을 차단하고 있는지 확인하십시오.

Exam question from Amazon's AWS Certified SysOps Administrator - Associate
Question #: 432
Topic #: 1

A SysOps administrator has set up a new Amazon EC2 instance as a web server in a public subnet. The instance uses HTTP port 80 and HTTPS port 443.

The SysOps administrator has confirmed internet connectivity by downloading operating system updates and software from public repositories. However, the SysOps administrator cannot access the instance from a web browser on the internet.

Which combination of steps should the SysOps administrator take to troubleshoot this issue? (Choose three.)

  • A. Ensure that the inbound rules of the instance’s security group allow traffic on ports 80 and 443. Most Voted
  • B. Ensure that the outbound rules of the instance’s security group allow traffic on ports 80 and 443.
  • C. Ensure that ephemeral ports 1024-65535 are allowed in the inbound rules of the network ACL that is associated with the instance's subnet.
  • D. Ensure that ephemeral ports 1024-65535 are allowed in the outbound rules of the network ACL that is associated with the instance’s subnet. Most Voted
  • E. Ensure that the filtering rules for any firewalls that are running on the instance allow inbound traffic on ports 80 and 443. Most Voted
  • F. Ensure that AWS WAF is turned on for the instance and is blocking web traffic.
