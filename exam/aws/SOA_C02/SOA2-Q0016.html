고객 게이트웨이 리소스를 생성하려면 어떤 주소를 사용해야 합니까?

  • A. 고객 게이트웨이 디바이스의 프라이빗 IP 주소

  • C. 고객 게이트웨이 디바이스의 퍼블릭 IP 주소
  • D. 고객 게이트웨이 디바이스 앞에 있는 NAT 디바이스의 퍼블릭 IP 주소 최다 투표

What address should be used to create the customer gateway resource?

  • A. The private IP address of the customer gateway device
  • B. The MAC address of the NAT device in front of the customer gateway device
  • C. The public IP address of the customer gateway device
  • D. The public IP address of the NAT device in front of the customer gateway device Most Voted
