이것이 어떻게 이루어질 수 있습니까?

  • A. 기본 모니터링을 통해 EC2 인스턴스에 대한 Amazon CloudWatch 경보를 생성합니다. 인스턴스를 다시 시작하는 작업을 추가합니다.
  • B. 자세한 모니터링을 통해 EC2 인스턴스에 대한 Amazon CloudWatch 경보를 생성합니다. 인스턴스를 다시 시작하는 작업을 추가합니다. 최다 투표
  • C. 2분마다 예약된 방식으로 호출되는 EC2 인스턴스를 다시 시작하는 AWS Lambda 함수를 생성합니다.
  • D. EC2 상태 확인에 의해 호출되는 EC2 인스턴스를 다시 시작하는 AWS Lambda 함수를 생성합니다.
