주제 #: 1

회사는 기록 데이터를 분석하고 Amazon S3에 저장된 데이터를 쿼리해야 합니다. 새로운 데이터는 Amazon S3에 저장되는 .csv 파일로 매일 생성됩니다.
이 회사의 분석가들은 Amazon Athena를 사용하여 전체 데이터의 최근 하위 집합에 대해 SQL 쿼리를 수행하고 있습니다. Amazon S3 에 수집되는 데이터의 양은
시간이 지남에 따라 크게 증가했으며 쿼리 지연 시간도 늘어났습니다.
쿼리 성능을 향상시키기 위해 회사는 어떤 솔루션을 구현할 수 있습니까? (2개를 선택하세요.)

  • A. Amazon EC2 인스턴스에서 MySQL Workbench를 사용하고 JDBC 또는 ODBC 커넥터를 사용하여 Athena에 연결합니다. Athena 대신 MySQL Workbench에서 쿼리를 직접 실행하세요.
  • B. Athena를 사용하여 매일 데이터를 추출하고 Apache Parquet 형식으로 저장합니다. 추출된 데이터를 쿼리합니다.
  • C. 매일 AWS Glue ETL 작업을 실행하여 데이터 파일을 Apache Parquet로 변환하고 변환된 파일을 분할합니다. 정기적인 AWS Glue 크롤러를 생성하여 매일 분할된 데이터를 자동으로 크롤링합니다. 최다 투표
  • D. 매일 AWS Glue ETL 작업을 실행하여 .gzip 형식을 사용하여 데이터 파일을 압축합니다. 압축된 데이터를 쿼리합니다.
  • E. 매일 AWS Glue ETL 작업을 실행하여 .lzo 형식을 사용하여 데이터 파일을 압축합니다. 압축된 데이터를 쿼리합니다. 최다 투표

Exam question from Amazon's AWS Certified Data Analytics - Specialty
Question #: 115
Topic #: 1

A company analyzes historical data and needs to query data that is stored in Amazon S3. New data is generated daily as .csv files that are stored in Amazon S3.
The company's analysts are using Amazon Athena to perform SQL queries against a recent subset of the overall data. The amount of data that is ingested into
Amazon S3 has increased substantially over time, and the query latency also has increased.
Which solutions could the company implement to improve query performance? (Choose two.)

  • A. Use MySQL Workbench on an Amazon EC2 instance, and connect to Athena by using a JDBC or ODBC connector. Run the query from MySQL Workbench instead of Athena directly.
  • B. Use Athena to extract the data and store it in Apache Parquet format on a daily basis. Query the extracted data.
  • C. Run a daily AWS Glue ETL job to convert the data files to Apache Parquet and to partition the converted files. Create a periodic AWS Glue crawler to automatically crawl the partitioned data on a daily basis. Most Voted
  • D. Run a daily AWS Glue ETL job to compress the data files by using the .gzip format. Query the compressed data.
  • E. Run a daily AWS Glue ETL job to compress the data files by using the .lzo format. Query the compressed data. Most Voted
