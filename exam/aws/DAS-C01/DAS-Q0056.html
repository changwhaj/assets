주제 #: 1

병원에서는 웨어러블 의료용 센서 장치를 사용하여 환자로부터 데이터를 수집합니다. 병원은 대규모로 데이터를 안전하게 수집할 수 있는 거의 실시간 솔루션을 설계하고 있습니다. 또한 솔루션은 스트리밍 데이터에서 환자의 보호 건강 정보(PHI)를 제거하고 해당 데이터를 내구성 있는 저장소에 저장할 수 있어야 합니다.
최소한의 운영 오버헤드로 이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. KCL(Kinesis 클라이언트 라이브러리)을 사용하여 AWS Lambda 함수를 호출하여 모든 PHI를 제거하는 Amazon Kinesis Data Streams를 사용하여 데이터를 수집합니다. Amazon S3에 데이터를 씁니다.
  • B. Amazon Kinesis Data Firehose를 사용하여 데이터를 수집하여 Amazon S3에 데이터를 씁니다. Amazon S3가 센서 데이터를 구문 분석하여 Amazon S3의 모든 PHI를 제거하는 AWS Lambda 함수를 트리거하도록 합니다.
  • C. Amazon Kinesis Data Streams를 사용하여 데이터를 수집하여 Amazon S3에 데이터를 씁니다. 데이터 스트림이 센서 데이터를 구문 분석하고 Amazon S3에서 모든 PHI를 제거하는 AWS Lambda 함수를 시작하도록 합니다.
  • D. Amazon Kinesis Data Firehose를 사용하여 데이터를 수집하여 Amazon S3에 데이터를 씁니다. 모든 PHI를 제거하기 위해 센서 데이터를 구문 분석하는 변환 AWS Lambda 함수를 구현합니다. 최다 투표

Exam question from Amazon's AWS Certified Data Analytics - Specialty
Question #: 56
Topic #: 1

A hospital uses wearable medical sensor devices to collect data from patients. The hospital is architecting a near-real-time solution that can ingest the data securely at scale. The solution should also be able to remove the patient's protected health information (PHI) from the streaming data and store the data in durable storage.
Which solution meets these requirements with the least operational overhead?

  • A. Ingest the data using Amazon Kinesis Data Streams, which invokes an AWS Lambda function using Kinesis Client Library (KCL) to remove all PHI. Write the data in Amazon S3.
  • B. Ingest the data using Amazon Kinesis Data Firehose to write the data to Amazon S3. Have Amazon S3 trigger an AWS Lambda function that parses the sensor data to remove all PHI in Amazon S3.
  • C. Ingest the data using Amazon Kinesis Data Streams to write the data to Amazon S3. Have the data stream launch an AWS Lambda function that parses the sensor data and removes all PHI in Amazon S3.
  • D. Ingest the data using Amazon Kinesis Data Firehose to write the data to Amazon S3. Implement a transformation AWS Lambda function that parses the sensor data to remove all PHI. Most Voted
Show Suggested Answer Hide Answer
Suggested Answer: C 🗳️
