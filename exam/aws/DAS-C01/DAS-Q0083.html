Correct answer is C as Job Bookmarks supports incremental data processing by persisting the state information. Option A is wrong as Glue Workflow helps create and visualize complex extract, transform, and load (ETL) activities involving multiple crawlers, jobs, and triggers. Option B is wrong a Glue trigger can start specified jobs and crawlers. A trigger fires on-demand, based on a schedule, or based on a combination of events. Option D is wrong as the Glue classifier reads the data in a data store. If it recognizes the format of the data, it generates a schema. The classifier also returns a certain number to indicate how certain the format recognition was.
