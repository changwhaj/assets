주제 #: 1

미디어 회사는 Amazon S3 데이터 레이크에 있는 데이터에 대해 기계 학습 및 분석을 수행하려고 합니다. 회사 내 소비자가 보고서를 생성할 수 있게 하는 두 가지 데이터 변환 요구 사항이 있습니다.
✑ 예약된 시간에 Amazon S3에 도착하는 다양한 파일 형식의 데이터 300GB를 매일 변환합니다.
✑ S3 데이터 레이크에 있는 테라바이트 단위의 보관된 데이터를 일회성으로 변환합니다.
데이터 변환에 대한 회사의 요구 사항을 비용 효율적으로 충족하는 솔루션 조합은 무엇입니까? (3개를 선택하세요.)

  • A. 매일 수신되는 데이터의 경우 AWS Glue 크롤러를 사용하여 스키마를 스캔하고 식별합니다. 최다 투표
  • B. 매일 수신되는 데이터의 경우 Amazon Athena를 사용하여 스키마를 스캔하고 식별합니다.
  • C. 매일 수신되는 데이터의 경우 Amazon Redshift를 사용하여 변환을 수행합니다.
  • D. 매일 수신되는 데이터의 경우 AWS Glue 작업과 함께 AWS Glue 워크플로를 사용하여 변환을 수행합니다. 최다 투표
  • E. 보관된 데이터의 경우 Amazon EMR을 사용하여 데이터 변환을 수행합니다. 최다 투표
  • F. 보관된 데이터의 경우 Amazon SageMaker를 사용하여 데이터 변환을 수행합니다.

Question #: 55
Topic #: 1

A media company wants to perform machine learning and analytics on the data residing in its Amazon S3 data lake. There are two data transformation requirements that will enable the consumers within the company to create reports:
✑ Daily transformations of 300 GB of data with different file formats landing in Amazon S3 at a scheduled time.
✑ One-time transformations of terabytes of archived data residing in the S3 data lake.
Which combination of solutions cost-effectively meets the company's requirements for transforming the data? (Choose three.)

  • A. For daily incoming data, use AWS Glue crawlers to scan and identify the schema. Most Voted
  • B. For daily incoming data, use Amazon Athena to scan and identify the schema.
  • C. For daily incoming data, use Amazon Redshift to perform transformations.
  • D. For daily incoming data, use AWS Glue workflows with AWS Glue jobs to perform transformations. Most Voted
  • E. For archived data, use Amazon EMR to perform data transformations. Most Voted
  • F. For archived data, use Amazon SageMaker to perform data transformations.

정답은 A, D & E입니다. Glue Crawler를 사용하는 옵션 A & D와 Glue Workflows는 일일 트랜잭션에 ETL을 제공합니다. EMR로서의 옵션 E는 보관된 데이터에 대한 데이터 변환을 수행하는 데 도움이 될 수 있습니다. Athena는 스키마를 식별하지 않지만 Glue Catalog를 사용하므로 옵션 B는 잘못된 것입니다. 옵션 C는 Redshift가 지속적이어야 하고 Glue에 비해 비용 효율적인 솔루션을 제공하지 않기 때문에 잘못된 것입니다. 옵션 F는 잘못된 것입니다. Amazon SageMaker는 모든 개발자와 데이터 과학자에게 기계 학습(ML) 모델을 신속하게 구축, 교육 및 배포할 수 있는 기능을 제공하는 완전관리형 서비스이기 때문입니다. 대용량 데이터에는 ETL 기능을 제공하지 않습니다.

Correct answers are A, D & E Options A & D using Glue Crawler and Glue Workflows would provide ETL for daily transactions. Option E as EMR can help perform data transformation for archived data.
