Amazon의 AWS 공인 데이터 분석 – 전문 분야 시험 문제
질문 #: 24
주제 #: 1

한 에너지 회사는 건물에 부착된 센서로부터 실시간으로 전압 데이터를 수집합니다. 회사는 같은 건물에서 갑작스러운 전압 상승 후 10분 이내에 일련의 두 번의 전압 강하가 감지되면 알림을 받기를 원합니다. 모든 알림은 가능한 한 빨리 전달되어야 합니다. 시스템은 가용성이 높아야 합니다. 회사에는 이 모니터링 기능이 다른 도시에 구현될 때 자동으로 확장되는 솔루션이 필요합니다. 알림 시스템은 문제 해결을 위해 Amazon Simple 알림 서비스(Amazon SNS) 주제를 구독합니다.
어떤 솔루션이 이러한 요구 사항을 충족합니까?

  • A. Amazon Managed Streaming for Apache Kafka 클러스터를 생성하여 데이터를 수집합니다. 자동으로 확장되는 Amazon EMR 클러스터에서 Apache Kafka 소비자 API와 함께 Apache Spark 스트리밍을 사용하여 수신 데이터를 처리합니다. Spark Streaming 애플리케이션을 사용하여 알려진 이벤트 시퀀스를 감지하고 SNS 메시지를 보냅니다. 최다 투표
  • B. AWS Lambda 함수 앞에 Amazon API Gateway를 사용하여 REST 기반 웹 서비스를 생성합니다. 현재 수요를 충족하기에 충분한 프로비저닝된 IOPS를 갖춘 PostgreSQL용 Amazon RDS 데이터베이스를 생성합니다. PostgreSQL용 RDS 데이터베이스에 수신 이벤트를 저장하고, 최신 데이터를 쿼리하여 알려진 이벤트 시퀀스를 감지하고, SNS 메시지를 보내도록 Lambda 함수를 구성합니다.
  • C. Amazon Kinesis Data Firehose 전송 스트림을 생성하여 수신 센서 데이터를 캡처합니다. AWS Lambda 변환 함수를 사용하여 알려진 이벤트 시퀀스를 감지하고 SNS 메시지를 보냅니다. 최다 투표
  • D. Amazon Kinesis 데이터 스트림을 생성하여 수신 센서 데이터를 캡처합니다. 알림을 위한 또 다른 스트림을 만듭니다. 두 스트림 모두에서 AWS Application Auto Scaling을 설정합니다. Java용 Amazon Kinesis Data Analytics 애플리케이션을 생성하여 알려진 이벤트 시퀀스를 감지하고 메시지 스트림에 메시지를 추가합니다. 메시지 스트림을 폴링하고 SNS 주제에 게시하도록 AWS Lambda 함수를 구성합니다.

Exam question from Amazon's AWS Certified Data Analytics - Specialty
Question #: 24
Topic #: 1

An energy company collects voltage data in real time from sensors that are attached to buildings. The company wants to receive notifications when a sequence of two voltage drops is detected within 10 minutes of a sudden voltage increase at the same building. All notifications must be delivered as quickly as possible. The system must be highly available. The company needs a solution that will automatically scale when this monitoring feature is implemented in other cities. The notification system is subscribed to an Amazon Simple Notification Service (Amazon SNS) topic for remediation.
Which solution will meet these requirements?

  • A. Create an Amazon Managed Streaming for Apache Kafka cluster to ingest the data. Use an Apache Spark Streaming with Apache Kafka consumer API in an automatically scaled Amazon EMR cluster to process the incoming data. Use the Spark Streaming application to detect the known event sequence and send the SNS message. Most Voted
  • B. Create a REST-based web service by using Amazon API Gateway in front of an AWS Lambda function. Create an Amazon RDS for PostgreSQL database with sufficient Provisioned IOPS to meet current demand. Configure the Lambda function to store incoming events in the RDS for PostgreSQL database, query the latest data to detect the known event sequence, and send the SNS message.
  • C. Create an Amazon Kinesis Data Firehose delivery stream to capture the incoming sensor data. Use an AWS Lambda transformation function to detect the known event sequence and send the SNS message. Most Voted
  • D. Create an Amazon Kinesis data stream to capture the incoming sensor data. Create another stream for notifications. Set up AWS Application Auto Scaling on both streams. Create an Amazon Kinesis Data Analytics for Java application to detect the known event sequence, and add a message to the message stream Configure an AWS Lambda function to poll the message stream and publish to the SNS topic.
