질문 #: 48
주제 #: 1

한 회사는 Amazon Kinesis Data Firehose를 사용하여 AWS WAF에서 Amazon S3 버킷으로 전체 로그를 전달하는 새로운 선거 보고 웹 사이트를 개발했습니다.
이제 회사는 최소한의 개발 노력이 필요한 방식으로 로그 시각화를 통해 드물게 발생하는 데이터 분석을 수행할 수 있는 저렴한 옵션을 찾고 있습니다.
이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. AWS Glue 크롤러를 사용하여 로그에서 Glue 데이터 카탈로그의 테이블을 생성하고 업데이트합니다. Athena를 사용하여 임시 분석을 수행하고 Amazon QuickSight를 사용하여 데이터 시각화를 개발합니다. 최다 투표
  • B. 두 번째 Kinesis Data Firehose 전송 스트림을 생성하여 Amazon OpenSearch Service(Amazon Elasticsearch Service)에 로그 파일을 전송합니다. Amazon ES를 사용하여 임시 분석을 위해 텍스트 기반 로그 검색을 수행하고 데이터 시각화를 위해 OpenSearch 대시보드(Kibana)를 사용합니다.
  • C. 로그를 .csv 형식으로 변환하는 AWS Lambda 함수를 생성합니다. 그런 다음 Kinesis Data Firehose 변환 구성에 함수를 추가합니다. Amazon Redshift를 사용하여 SQL 쿼리를 사용하여 로그에 대한 임시 분석을 수행하고 Amazon QuickSight를 사용하여 데이터 시각화를 개발합니다.
  • D. Amazon EMR 클러스터를 생성하고 Amazon S3를 데이터 소스로 사용합니다. 임시 분석을 수행하고 Amazon QuickSight를 사용하여 데이터 시각화를 개발하는 Apache Spark 작업을 생성합니다.

Exam question from Amazon's AWS Certified Data Analytics - Specialty
Question #: 48
Topic #: 1

A company developed a new elections reporting website that uses Amazon Kinesis Data Firehose to deliver full logs from AWS WAF to an Amazon S3 bucket.
The company is now seeking a low-cost option to perform this infrequent data analysis with visualizations of logs in a way that requires minimal development effort.
Which solution meets these requirements?

  • A. Use an AWS Glue crawler to create and update a table in the Glue data catalog from the logs. Use Athena to perform ad-hoc analyses and use Amazon QuickSight to develop data visualizations. Most Voted
  • B. Create a second Kinesis Data Firehose delivery stream to deliver the log files to Amazon OpenSearch Service (Amazon Elasticsearch Service). Use Amazon ES to perform text-based searches of the logs for ad-hoc analyses and use OpenSearch Dashboards (Kibana) for data visualizations.
  • C. Create an AWS Lambda function to convert the logs into .csv format. Then add the function to the Kinesis Data Firehose transformation configuration. Use Amazon Redshift to perform ad-hoc analyses of the logs using SQL queries and use Amazon QuickSight to develop data visualizations.
  • D. Create an Amazon EMR cluster and use Amazon S3 as the data source. Create an Apache Spark job to perform ad-hoc analyses and use Amazon QuickSight to develop data visualizations.
