  • C. Use Amazon Kinesis Data Firehose to ingest Amazon Connect data and Amazon AppFlow to ingest Salesforce data. Most Voted

Selected Answer: C
The most efficient way to collect data in the data lake with the least operational overhead is option C: Use Amazon Kinesis Data Firehose to ingest Amazon Connect data and Amazon AppFlow to ingest Salesforce data. Amazon Kinesis Data Firehose is a fully managed service that can reliably capture and load streaming data into Amazon S3 with no operational overhead. It can also transform the incoming data before storing it in S3. Amazon AppFlow is a fully managed integration service that can securely transfer data between different SaaS applications, including Salesforce and S3. It supports incremental data transfers and can also transform the data during the transfer process. Using Kinesis Data Firehose for Amazon Connect data and AppFlow for Salesforce data provides a simple and scalable solution that requires minimal operational overhead, allowing the data engineering team to focus on other tasks.
