Amazon의 AWS 공인 데이터 분석 – 전문 분야 시험 문제
질문 #: 135
주제 #: 1

한 회사가 영업 및 마케팅 부서에 분석 서비스를 제공하고 있습니다. 부서에서는
연결을 위해 Amazon Redshift 내부 사용자를 사용하여 Amazon Redshift에서 쿼리를 실행하는 비즈니스 인텔리전스(BI) 도구를 통해서만 데이터에 액세스할 수 있습니다. 각 부서에는 해당 부서에 필요한 권한을 가진 Amazon Redshift 데이터베이스의 사용자가 할당됩니다. 마케팅 데이터 분석가는
회사 데이터 레이크의 마케팅 S3 버킷에 Apache Parquet 형식으로 저장되어 있는 광고 테이블에 대한 직접 액세스 권한을 부여받아야 합니다. 회사 데이터 레이크는 AWS Lake Formation에서 관리합니다. 마지막으로 액세스는 테이블의 프로모션 열 3개로 제한되어야 합니다.
이러한 요구 사항을 충족하는 단계 조합은 무엇입니까? (3개를 선택하세요.)

  • A. 마케팅 Amazon Redshift 사용자가 광고 외부 테이블의 프로모션 열 3개에 액세스할 수 있도록 Amazon Redshift에 권한을 부여합니다.
  • B. Lake Formation에 대한 권한이 있는 Amazon Redshift Spectrum IAM 역할을 생성합니다. Amazon Redshift 클러스터에 연결합니다. 최다 투표
  • C. 마케팅 S3 버킷에 대한 권한이 있는 Amazon Redshift Spectrum IAM 역할을 생성합니다. Amazon Redshift 클러스터에 연결합니다.
  • D. Amazon Redshift Spectrum IAM 역할을 사용하여 Amazon Redshift에서 외부 스키마를 생성합니다. 마케팅 Amazon Redshift 사용자에게 사용 권한을 부여합니다. 최다 투표
  • E. Amazon Redshift Spectrum 역할이 광고 테이블의 3개 프로모션 열에 액세스할 수 있도록 Lake Formation에 권한을 부여합니다. 최다 투표
  • F. 마케팅 IAM 그룹이 광고 테이블의 3개 프로모션 열에 액세스할 수 있도록 Lake Formation에 권한을 부여합니다.

Exam question from Amazon's AWS Certified Data Analytics - Specialty
Question #: 135
Topic #: 1

A company is providing analytics services to its sales and marketing departments. The departments can access the data only through their business intelligence
(BI) tools, which run queries on Amazon Redshift using an Amazon Redshift internal user to connect. Each department is assigned a user in the Amazon Redshift database with the permissions needed for that department. The marketing data analysts must be granted direct access to the advertising table, which is stored in
Apache Parquet format in the marketing S3 bucket of the company data lake. The company data lake is managed by AWS Lake Formation. Finally, access must be limited to the three promotion columns in the table.
Which combination of steps will meet these requirements? (Choose three.)

  • A. Grant permissions in Amazon Redshift to allow the marketing Amazon Redshift user to access the three promotion columns of the advertising external table.
  • B. Create an Amazon Redshift Spectrum IAM role with permissions for Lake Formation. Attach it to the Amazon Redshift cluster. Most Voted
  • C. Create an Amazon Redshift Spectrum IAM role with permissions for the marketing S3 bucket. Attach it to the Amazon Redshift cluster.
  • D. Create an external schema in Amazon Redshift by using the Amazon Redshift Spectrum IAM role. Grant usage to the marketing Amazon Redshift user. Most Voted
  • E. Grant permissions in Lake Formation to allow the Amazon Redshift Spectrum role to access the three promotion columns of the advertising table. Most Voted
  • F. Grant permissions in Lake Formation to allow the marketing IAM group to access the three promotion columns of the advertising table.
