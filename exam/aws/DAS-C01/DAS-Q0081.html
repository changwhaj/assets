온라인 소매업체는 제품 판매 보고 솔루션을 배포해야 합니다. 소스 데이터는 보고를 위해 외부 OLTP(온라인 트랜잭션 처리) 시스템에서 내보내집니다. 롤업 데이터는 전날 활동에 대해 매일 계산됩니다. 보고 시스템에는 다음과 같은 요구 사항이 있습니다.
✑ 1년 동안 쉽게 사용할 수 있는 일일 롤업 데이터를 보유합니다.
✑ 1년 후에는 가끔이지만 즉시 액세스할 수 있도록 일일 롤업 데이터를 보관합니다.
✑ 보고 시스템에 저장된 원본 데이터 내보내기는 5년 동안 보관되어야 합니다. 쿼리 액세스는 처음 90일 이내에 발생할 수 있는 재평가에만 필요합니다.
스토리지 비용을 최소로 유지하면서 이러한 요구 사항을 충족하는 작업 조합은 무엇입니까? (2개를 선택하세요.)

  • A. 처음에는 Amazon S3 Standard-Infrequent Access(S3 Standard-IA) 스토리지 클래스에 소스 데이터를 저장합니다. 생성 후 90일이 지나면 스토리지 클래스를 Amazon S3 Glacier Deep Archive로 변경하고 생성 후 5년이 지나면 데이터를 삭제하는 수명 주기 구성을 적용합니다.
  • B. 처음에는 Amazon S3 Glacier 스토리지 클래스에 소스 데이터를 저장합니다. 생성 후 90일이 지나면 Amazon S3 Glacier에서 Amazon S3 Glacier Deep Archive로 스토리지 클래스를 변경하고 생성 후 5년이 지나면 데이터를 삭제하는 수명 주기 구성을 적용합니다. 최다 투표
  • C. 처음에는 Amazon S3 Standard 스토리지 클래스에 일일 롤업 데이터를 저장합니다. 데이터 생성 후 1년이 지나면 스토리지 클래스를 Amazon S3 Glacier Deep Archive로 변경하는 수명 주기 구성을 적용합니다.
  • D. 처음에는 Amazon S3 Standard 스토리지 클래스에 일일 롤업 데이터를 저장합니다. 데이터 생성 후 1년이 지나면 스토리지 클래스를 Amazon S3 Standard-Infrequent Access(S3 Standard-IA)로 변경하는 수명 주기 구성을 적용합니다. 최다 투표
  • E. 처음에는 Amazon S3 Standard-Infrequent Access(S3 Standard-IA) 스토리지 클래스에 일일 롤업 데이터를 저장합니다. 데이터 생성 후 1년이 지나면 스토리지 클래스를 Amazon S3 Glacier로 변경하는 수명 주기 구성을 적용합니다.

An online retailer needs to deploy a product sales reporting solution. The source data is exported from an external online transaction processing (OLTP) system for reporting. Roll-up data is calculated each day for the previous day's activities. The reporting system has the following requirements:
✑ Have the daily roll-up data readily available for 1 year.
✑ After 1 year, archive the daily roll-up data for occasional but immediate access.
✑ The source data exports stored in the reporting system must be retained for 5 years. Query access will be needed only for re-evaluation, which may occur within the first 90 days.
Which combination of actions will meet these requirements while keeping storage costs to a minimum? (Choose two.)

  • A. Store the source data initially in the Amazon S3 Standard-Infrequent Access (S3 Standard-IA) storage class. Apply a lifecycle configuration that changes the storage class to Amazon S3 Glacier Deep Archive 90 days after creation, and then deletes the data 5 years after creation.
  • B. Store the source data initially in the Amazon S3 Glacier storage class. Apply a lifecycle configuration that changes the storage class from Amazon S3 Glacier to Amazon S3 Glacier Deep Archive 90 days after creation, and then deletes the data 5 years after creation. Most Voted
  • C. Store the daily roll-up data initially in the Amazon S3 Standard storage class. Apply a lifecycle configuration that changes the storage class to Amazon S3 Glacier Deep Archive 1 year after data creation.
  • D. Store the daily roll-up data initially in the Amazon S3 Standard storage class. Apply a lifecycle configuration that changes the storage class to Amazon S3 Standard-Infrequent Access (S3 Standard-IA) 1 year after data creation. Most Voted
  • E. Store the daily roll-up data initially in the Amazon S3 Standard-Infrequent Access (S3 Standard-IA) storage class. Apply a lifecycle configuration that changes the storage class to Amazon S3 Glacier 1 year after data creation.
