질문 #: 30
주제 #: 1

한 회사는 Amazon S3에 있는 데이터를 일괄 처리하기 위해 Apache Hive 스크립트를 개발했습니다. 스크립트는 매일 한 번씩 실행되고 출력을
Amazon S3에 저장해야 합니다. 회사는 스크립트를 테스트했으며 소규모 로컬 3노드 클러스터에서 30분 내에 완료되었습니다.
스크립트를 예약하고 실행하는 데 가장 비용 효과적인 솔루션은 무엇입니까?

  • A. Hive 실행 단계를 통해 Amazon EMR 클러스터를 구동하는 AWS Lambda 함수를 생성합니다. KeepJobFlowAliveWhenNoSteps를 false로 설정하고 종료 방지 플래그를 비활성화합니다. Amazon CloudWatch Events를 사용하여 매일 실행되도록 Lambda 함수를 예약합니다. 최다 투표
  • B. AWS Management Console을 사용하여 Python Hue로 Amazon EMR 클러스터를 가동합니다. 하이브, 아파치 우지. 종료 방지 플래그를 true로 설정하고 클러스터의 핵심 노드에 스팟 인스턴스를 사용합니다. Hive 스크립트를 매일 호출하도록 클러스터에서 Oozie 워크플로를 구성합니다.
  • C. Hive 스크립트를 사용하여 AWS Glue 작업을 생성하여 일괄 작업을 수행합니다. 시간 기반 일정을 사용하여 하루에 한 번 실행되도록 작업을 구성합니다.
  • D. AWS Lambda 레이어를 사용하고 Hive 런타임을 AWS Lambda에 로드하고 Hive 스크립트를 복사합니다. AWS Step Functions를 사용하여 워크플로를 생성하여 매일 실행되도록 Lambda 함수를 예약합니다.

Exam question from Amazon's AWS Certified Data Analytics - Specialty
Question #: 30
Topic #: 1

A company has developed an Apache Hive script to batch process data stared in Amazon S3. The script needs to run once every day and store the output in
Amazon S3. The company tested the script, and it completes within 30 minutes on a small local three-node cluster.
Which solution is the MOST cost-effective for scheduling and executing the script?

  • A. Create an AWS Lambda function to spin up an Amazon EMR cluster with a Hive execution step. Set KeepJobFlowAliveWhenNoSteps to false and disable the termination protection flag. Use Amazon CloudWatch Events to schedule the Lambda function to run daily. Most Voted
  • B. Use the AWS Management Console to spin up an Amazon EMR cluster with Python Hue. Hive, and Apache Oozie. Set the termination protection flag to true and use Spot Instances for the core nodes of the cluster. Configure an Oozie workflow in the cluster to invoke the Hive script daily.
  • C. Create an AWS Glue job with the Hive script to perform the batch operation. Configure the job to run once a day using a time-based schedule.
  • D. Use AWS Lambda layers and load the Hive runtime to AWS Lambda and copy the Hive script. Schedule the Lambda function to run daily by creating a workflow using AWS Step Functions.
