질문 #: 12
주제 #: 1

한 은행 회사는 현재 DS(고밀도 스토리지) 노드가 포함된 Amazon Redshift 클러스터를 사용하여 중요한 데이터를 저장하고 있습니다. 감사 결과 클러스터가 암호화되지 않은 것으로 나타났습니다. 규정 준수 요구 사항에 따르면 중요한 데이터가 포함된 데이터베이스는 자동화된 키 순환 기능을 갖춘 하드웨어 보안 모듈(HSM)을 통해 암호화되어야 합니다.
규정 준수를 달성하려면 어떤 단계 조합이 필요합니까? (2개를 선택하세요.)

  • A. 자동 키 순환 기능이 있는 클라이언트 및 서버 인증서를 사용하여 HSM과의 신뢰할 수 있는 연결을 설정합니다. 최다 투표
  • B. HSM 암호화 옵션과 자동 키 순환을 사용하여 클러스터를 수정합니다.
  • C. 새로운 HSM 암호화 Amazon Redshift 클러스터를 생성하고 데이터를 새 클러스터로 마이그레이션합니다. 최다 투표
  • D. AWS CLI를 통해 키 교체로 HSM을 활성화합니다.
  • E. HSM에서 ECDHE(Elliptic Curve Diffie-Hellman Ephemeral) 암호화를 활성화합니다.

Question #: 12
Topic #: 1

A banking company is currently using an Amazon Redshift cluster with dense storage (DS) nodes to store sensitive data. An audit found that the cluster is unencrypted. Compliance requirements state that a database with sensitive data must be encrypted through a hardware security module (HSM) with automated key rotation.
Which combination of steps is required to achieve compliance? (Choose two.)

  • A. Set up a trusted connection with HSM using a client and server certificate with automatic key rotation. Most Voted
  • B. Modify the cluster with an HSM encryption option and automatic key rotation.
  • C. Create a new HSM-encrypted Amazon Redshift cluster and migrate the data to the new cluster. Most Voted
  • D. Enable HSM with key rotation through the AWS CLI.
  • E. Enable Elliptic Curve Diffie-Hellman Ephemeral (ECDHE) encryption in the HSM.
