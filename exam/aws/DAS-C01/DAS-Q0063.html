데이터 플랫폼 팀은 데이터 처리를 관리하고 다운스트림 데이터 처리의 효율성과 비용에 관심을 갖고 있습니다. 그들은 PySpark를 계속 사용하고 싶어합니다
.
데이터 처리 작업의 효율성을 향상시키고 잘 구성된 솔루션은 무엇입니까?

  • A. 센서 및 장치 데이터를 Kinesis Data Firehose 전송 스트림으로 직접 보내 Apache Parquet 레코드 형식 변환이 활성화된 Amazon S3로 데이터를 보냅니다. PySpark를 실행하는 Amazon EMR을 사용하여 Amazon S3의 데이터를 처리합니다. 최다 투표
  • B. Python 런타임 환경으로 AWS Lambda 함수를 설정합니다. Lambda를 사용하여 연결된 디바이스 및 센서의 개별 Kinesis 데이터 스트림 메시지를 처리합니다.
  • C. Amazon Redshift 클러스터를 시작합니다. 수집된 데이터를 Amazon S3에서 Amazon Redshift로 복사하고 데이터 처리 작업을 Amazon EMR에서 Amazon Redshift로 이동합니다.
  • D. Amazon S3의 작은 데이터 파일을 더 큰 파일로 병합하고 이를 Apache Parquet 형식으로 변환하도록 AWS Glue Python 작업을 설정합니다. 다운스트림 PySpark 작업을 Amazon EMR에서 AWS Glue로 마이그레이션합니다. 최다 투표

Exam question from Amazon's AWS Certified Data Analytics - Specialty
Question #: 63
Topic #: 1

A smart home automation company must efficiently ingest and process messages from various connected devices and sensors. The majority of these messages are comprised of a large number of small files. These messages are ingested using Amazon Kinesis Data Streams and sent to Amazon S3 using a Kinesis data stream consumer application. The Amazon S3 message data is then passed through a processing pipeline built on Amazon EMR running scheduled PySpark jobs.
The data platform team manages data processing and is concerned about the efficiency and cost of downstream data processing. They want to continue to use
PySpark.
Which solution improves the efficiency of the data processing jobs and is well architected?

  • A. Send the sensor and devices data directly to a Kinesis Data Firehose delivery stream to send the data to Amazon S3 with Apache Parquet record format conversion enabled. Use Amazon EMR running PySpark to process the data in Amazon S3. Most Voted
  • B. Set up an AWS Lambda function with a Python runtime environment. Process individual Kinesis data stream messages from the connected devices and sensors using Lambda.
  • C. Launch an Amazon Redshift cluster. Copy the collected data from Amazon S3 to Amazon Redshift and move the data processing jobs from Amazon EMR to Amazon Redshift.
  • D. Set up AWS Glue Python jobs to merge the small data files in Amazon S3 into larger files and transform them to Apache Parquet format. Migrate the downstream PySpark jobs from Amazon EMR to AWS Glue. Most Voted
