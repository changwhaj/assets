AWS DataSync는 온프레미스 스토리지와 AWS 스토리지 서비스 간의 데이터 전송을 단순화하고 자동화하는 완전관리형 서비스입니다. 또한 AWS 스토리지 서비스와 다른 AWS 리전 간의 전송도 지원합니다. 이 경우 정기적으로 두 AWS 리전의 시스템 간에 데이터를 전송하는 데 사용할 수 있습니다. AWS DataSync is a fully managed service that simplifies and automates data transfers between on-premises storage and AWS storage services. It also supports transfers between AWS storage services and different AWS Regions. In this case, it can be used to transfer data between the systems in the two AWS Regions on a periodic basis. AWS DataSync handles the heavy lifting of the data transfer process, including network optimization, parallel data transfer, and data integrity validation. It provides a simple and easy-to-use interface for setting up and managing the data transfer tasks.
