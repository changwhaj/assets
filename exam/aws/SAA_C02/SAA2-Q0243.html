C: aws S3 sync 명령을 사용하여 원본 버킷에서 대상 버킷으로 데이터를 복사합니다. aws S3 sync 명령은 CopyObject API를 사용하여 S3 버킷 간에 객체를 복사합니다. sync 명령은 소스 버킷에 있지만 대상 버킷에는 없는 객체를 식별하기 위해 소스 및 대상 버킷을 나열합니다. 또한 이 명령은 대상 버킷에 있는 객체와 LastModified 날짜가 다른 소스 버킷의 객체를 식별합니다. 버전이 지정된 버킷의 sync 명령은 객체의 현재 버전만 복사합니다. 이전 버전은 복사되지 않습니다. 기본적으로 객체 메타데이터는 보존되지만, AWS 계정에 대한 ACL(액세스 제어 목록)은 FULL_CONTROL로 설정되어 추가 ACL이 제거됩니다. 작업이 실패하면 이전에 복사한 개체를 복제하지 않고 sync 명령을 다시 실행할 수 있습니다.
