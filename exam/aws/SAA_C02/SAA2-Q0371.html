C. Deploy the application to an Amazon S3 bucket that has versioning and cross-Region replication enabled. An Amazon S3 bucket configured with versioning and cross-Region replication provides a low-cost and highly available solution for hosting a static website. With versioning enabled, you can preserve multiple versions of an object in the same bucket, allowing you to revert to a previous version if needed. Cross-Region replication helps to provide high availability by automatically replicating objects in the bucket to another AWS Region, ensuring that the content remains available even if an AWS Region becomes unavailable. By using an S3 bucket configured with versioning and cross-Region replication, you can ensure that your static website is highly available and minimize costs.
