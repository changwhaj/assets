What should a solutions architect do to meet these requirements?

  • A. Use an AWS Storage Gateway tape gateway to copy the on-premises files into Amazon S3.
  • B. Use an AWS Storage Gateway volume gateway to copy the on-premises files into Amazon S3. Most Voted
  • C. Use an AWS Storage Gateway file gateway to copy the on-premises files to Amazon Elastic Block Store (Amazon EBS).
  • D. Attach an Amazon Elastic File System (Amazon EFS) file system to the on-premises servers. Copy the files to Amazon EFS. Most Voted
