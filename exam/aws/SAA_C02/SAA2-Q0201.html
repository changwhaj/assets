Network Load Balancers, the load balancer node that receives the connection uses the following process: 1.Selects a target from the target group for the default rule using a flow hash algorithm. It bases the algorithm on: * The protocol * The source IP address and source port * The destination IP address and destination port * The TCP sequence number 2.Routes each individual TCP connection to a single target for the life of the connection. The TCP connections from a client have different source ports and sequence numbers, and can be routed to different targets.
