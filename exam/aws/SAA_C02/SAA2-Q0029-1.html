Which combination of actions will accomplish this? (Choose two.)

  • A. Ensure the root user uses a strong password.
  • B. Enable multi-factor authentication to the root user.
  • C. Store root user access keys in an encrypted Amazon S3 bucket.
  • D. Add the root user to a group containing administrative permissions.
  • E. Apply the required permissions to the root user with an inline policy document.

sumitchhabra
2021-10-05 06:07
The Question is not talking about Normal or Standard IAM users. Its talking about root users. So B + E should be the correct answer. see this : https://aws.amazon.com/premiumsupport/knowledge-center/security-best-practices/ Also search for the below context in the above url: Limit root user access to your resources Root user account credentials (the root password or root access keys) grant unlimited access to your account and its resources, so it's a best practice to both secure and minimize root user access to your account. Consider the following strategies to limit root user access to your account: Use IAM users for day-to-day access to your account, even if you're the only person accessing it. Eliminate the use of root access keys; instead, rotate them to IAM access keys and then delete the root access keys. Use an MFA device for the root user of your account.
