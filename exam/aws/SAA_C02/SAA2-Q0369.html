A: VMware Cloud on AWS에 Storage Gateway VM을 배포하거나 Amazon EC2에 AMI로 배포할 수도 있습니다. Q: Amazon S3 파일 게이트웨이란 무엇입니까? A: S3 파일 게이트웨이를 사용하면 구성된 S3 버킷을 NFS(네트워크 파일 시스템) 탑재 지점 또는 SMB(서버 메시지 블록) 파일 공유로 사용할 수 있습니다. 애플리케이션은 NFS 또는 SMB를 통해 파일과 디렉터리를 읽고 쓰며 게이트웨이에 파일 서버로 접속합니다. 그러면 게이트웨이는 이러한 파일 작업을 S3 버킷에 대한 객체 요청으로 변환합니다.
