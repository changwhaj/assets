질문 #: 252
주제 #: 1

한 제조 회사는 장비가 고장날 가능성이 있는 시기를 예측하기 위해 기계 학습(ML) 모델을 만들고 싶어합니다. 데이터 과학 팀은 이미 로컬 환경에서 TensorFlow와 사용자 정의 Python 스크립트를 사용하여 딥 러닝 모델을 구축했습니다. 회사는 Amazon SageMaker를 사용하여 모델을 훈련하려고 합니다.

어떤 TensorFlow 추정기 구성이 모델을 가장 비용 효율적으로 교육합니까?

  • A. 컴파일러_config=TrainingCompilerConfig()를 매개변수로 추가하여 SageMaker 훈련 컴파일러를 켜십시오. TensorFlow fit() 메서드 호출 시 스크립트를 추정기에 전달합니다.
  • B. 컴파일러_config=TrainingCompilerConfig()를 매개변수로 추가하여 SageMaker 훈련 컴파일러를 켭니다. use_spot_instances 매개변수를 True로 설정하여 관리형 스팟 훈련을 활성화합니다. TensorFlow fit() 메서드 호출 시 스크립트를 추정기에 전달합니다. 최다 투표
  • C. 분산 데이터 병렬 처리를 사용하도록 훈련 스크립트를 조정합니다. 분포 모수에 적절한 값을 지정합니다. TensorFlow fit() 메서드 호출 시 스크립트를 추정기에 전달합니다.
  • D. 컴파일러_config=TrainingCompilerConfig()를 매개변수로 추가하여 SageMaker 훈련 컴파일러를 켭니다. MaxWaitTimeInSeconds 매개변수를 MaxRuntimeInSeconds 매개변수와 동일하게 설정하십시오. TensorFlow fit() 메서드 호출 시 스크립트를 추정기에 전달합니다.

A manufacturing company wants to create a machine learning (ML) model to predict when equipment is likely to fail. A data science team already constructed a deep learning model by using TensorFlow and a custom Python script in a local environment. The company wants to use Amazon SageMaker to train the model.

Which TensorFlow estimator configuration will train the model MOST cost-effectively?

  • A. Turn on SageMaker Training Compiler by adding compiler_config=TrainingCompilerConfig() as a parameter. Pass the script to the estimator in the call to the TensorFlow fit() method.
  • B. Turn on SageMaker Training Compiler by adding compiler_config=TrainingCompilerConfig() as a parameter. Turn on managed spot training by setting the use_spot_instances parameter to True. Pass the script to the estimator in the call to the TensorFlow fit() method. Most Voted
  • C. Adjust the training script to use distributed data parallelism. Specify appropriate values for the distribution parameter. Pass the script to the estimator in the call to the TensorFlow fit() method.
  • D. Turn on SageMaker Training Compiler by adding compiler_config=TrainingCompilerConfig() as a parameter. Set the MaxWaitTimeInSeconds parameter to be equal to the MaxRuntimeInSeconds parameter. Pass the script to the estimator in the call to the TensorFlow fit() method.
