이 회사는 웹 서버에 대한 각 요청에 대한 소스 IP 주소를 포함하는 기존 전자 상거래 웹 서버의 수 테라바이트에 달하는 데이터를 보유하고 있습니다. 인증된 요청의 경우 레코드에는 요청하는 사용자의 로그인 이름도 포함됩니다.
ML 전문가는 웹 애플리케이션에 새로운 보안 기능을 구현하기 위해 어떤 접근 방식을 취해야 합니까?

  • A. Amazon SageMaker Ground Truth를 사용하여 각 레코드에 액세스 시도 성공 또는 실패로 레이블을 지정합니다. FM(Factorization Machine) 알고리즘을 사용하여 이진 분류 모델을 훈련하려면 Amazon SageMaker를 사용하십시오.
  • B. Amazon SageMaker를 사용하여 IP Insights 알고리즘을 사용하여 모델을 훈련하십시오. 매일 밤 새 로그 데이터를 사용하여 모델 업데이트 및 재교육을 예약합니다. 최다 투표
  • C. Amazon SageMaker Ground Truth를 사용하여 각 레코드에 액세스 시도 성공 또는 실패로 레이블을 지정합니다. IP Insights 알고리즘을 사용하여 이진 분류 모델을 훈련하려면 Amazon SageMaker를 사용하십시오.
  • D. Amazon SageMaker를 사용하여 Object2Vec 알고리즘을 사용하여 모델을 교육합니다. 매일 밤 새 로그 데이터를 사용하여 모델 업데이트 및 재교육을 예약합니다.

Topic #: 1

A company offers an online shopping service to its customers. The company wants to enhance the site's security by requesting additional information when customers access the site from locations that are different from their normal location. The company wants to update the process to call a machine learning (ML) model to determine when additional information should be requested.
The company has several terabytes of data from its existing ecommerce web servers containing the source IP addresses for each request made to the web server. For authenticated requests, the records also contain the login name of the requesting user.
Which approach should an ML specialist take to implement the new security feature in the web application?

  • A. Use Amazon SageMaker Ground Truth to label each record as either a successful or failed access attempt. Use Amazon SageMaker to train a binary classification model using the factorization machines (FM) algorithm.
  • B. Use Amazon SageMaker to train a model using the IP Insights algorithm. Schedule updates and retraining of the model using new log data nightly. Most Voted
  • C. Use Amazon SageMaker Ground Truth to label each record as either a successful or failed access attempt. Use Amazon SageMaker to train a binary classification model using the IP Insights algorithm.
  • D. Use Amazon SageMaker to train a model using the Object2Vec algorithm. Schedule updates and retraining of the model using new log data nightly.

Amazon SageMaker IP Insights is an unsupervised learning algorithm that learns the usage patterns for IPv4 addresses. It is designed to capture associations between IPv4 addresses and various entities, such as user IDs or account numbers. You can use it to identify a user attempting to log into a web service from an anomalous IP address, for example. Or you can use it to identify an account that is attempting to create computing resources from an unusual IP address. Trained IP Insight models can be hosted at an endpoint for making real-time predictions or used for processing batch transforms.
