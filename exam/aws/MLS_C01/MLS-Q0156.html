주제 #: 1

한 회사가 수천 개의 소매점에 도매 의류를 공급합니다. 데이터 과학자는 각 매장의 각 품목에 대한 일일 판매량을 예측하는 모델을 만들어야 합니다. 데이터 과학자는 매장의 절반 이상이 영업을 시작한 지 6개월 미만이라는 사실을 발견했습니다. 판매 데이터는 매주 매우 일관됩니다. 데이터베이스의 일일 데이터는 매주 집계되었으며 판매가 없는 주는 현재 데이터세트에서 생략되었습니다. Amazon S3에서는 5년간(100MB)의 판매 데이터를 사용할 수 있습니다.
개발할 예측 모델의 성능에 부정적인 영향을 미치는 요인은 무엇이며, 이를 완화하기 위해 데이터 과학자는 어떤 조치를 취해야 합니까?
(2개를 선택하세요.)

  • A. 대부분의 매장에서 계절성을 감지하는 것이 문제가 될 것입니다. 더 많은 기록 데이터가 있는 유사한 매장과 새 매장을 연결하려면 범주형 데이터를 요청하세요. 최다 투표
  • B. 판매 데이터의 분산이 충분하지 않습니다. 모델의 일반화 능력을 향상시키기 위해 다른 업계의 외부 판매 데이터를 요청합니다.
  • C. 판매 데이터는 주 단위로 집계됩니다. 일일 모델을 구축할 수 있도록 소스 데이터베이스에서 일일 판매 데이터를 요청합니다. 최다 투표
  • D. 판매 데이터에 품목 판매 항목이 누락되었습니다. 모델 구축을 활성화하려면 원본 데이터베이스의 품목 판매 데이터에 항목이 0개 포함되도록 요청하세요.
  • E. Amazon S3에서는 100MB의 판매 데이터만 사용할 수 있습니다. 모델에 대한 200MB의 교육 데이터를 제공하는 10년간의 판매 데이터를 요청합니다.

Exam question from Amazon's AWS Certified Machine Learning - Specialty
Question #: 156
Topic #: 1

A company supplies wholesale clothing to thousands of retail stores. A data scientist must create a model that predicts the daily sales volume for each item for each store. The data scientist discovers that more than half of the stores have been in business for less than 6 months. Sales data is highly consistent from week to week. Daily data from the database has been aggregated weekly, and weeks with no sales are omitted from the current dataset. Five years (100 MB) of sales data is available in Amazon S3.
Which factors will adversely impact the performance of the forecast model to be developed, and which actions should the data scientist take to mitigate them?
(Choose two.)

  • A. Detecting seasonality for the majority of stores will be an issue. Request categorical data to relate new stores with similar stores that have more historical data. Most Voted
  • B. The sales data does not have enough variance. Request external sales data from other industries to improve the model's ability to generalize.
  • C. Sales data is aggregated by week. Request daily sales data from the source database to enable building a daily model. Most Voted
  • D. The sales data is missing zero entries for item sales. Request that item sales data from the source database include zero entries to enable building the model.
  • E. Only 100 MB of sales data is available in Amazon S3. Request 10 years of sales data, which would provide 200 MB of training data for the model.
