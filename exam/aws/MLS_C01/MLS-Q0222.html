금융 서비스 회사의 데이터 과학자는 Amazon SageMaker를 사용하여 대출 불이행을 예측하는 모델을 교육하고 배포했습니다. 이 모델은 새로운 대출 신청을 분석하고 대출 불이행 위험을 예측합니다. 모델을 훈련하기 위해 데이터 과학자는 데이터베이스에서 대출 데이터를 수동으로 추출했습니다. 데이터 과학자는 SageMaker Studio 노트북에 호스팅된 Jupyter 노트북에서 모델 훈련 및 배포 단계를 수행했습니다. 모델의 예측 정확도는 시간이 지남에 따라 감소합니다.

데이터 과학자가 모델의 정확성을 유지하기 위한 가장 운영상 효율적인 방법은 어떤 단계 조합입니까? (2개를 선택하세요.)

  • A. SageMaker 파이프라인을 사용하여 새로운 데이터를 추출하고, 모델을 교육하고, 모델의 새 버전을 배포하는 자동화된 워크플로를 생성하십시오. 최다 투표
  • B. 모델 드리프트를 확인하기 위해 정확도 임계값으로 SageMaker 모델 모니터를 구성합니다. 임계값이 초과되면 Amazon CloudWatch 경보가 시작됩니다. SageMaker Pipelines의 워크플로를 CloudWatch 경보와 연결하여 자동으로 재교육을 시작하세요. 최다 투표
  • C. Amazon S3에 모델 예측을 저장합니다. Amazon S3에서 예측을 읽고, 모델 예측 정확도의 변경 사항을 확인하고, 중요한 변경 사항이 감지되면 이메일 알림을 보내는 일일 SageMaker 처리 작업을 생성합니다.
  • D. SageMaker Studio 노트북에 호스팅된 Jupyter 노트북의 단계를 다시 실행하여 모델을 재교육하고 모델의 새 버전을 재배포합니다.
  • E. SageMaker Studio 노트북의 교육 및 배포 코드를 Python 스크립트로 내보냅니다. AWS Lambda 함수가 시작할 수 있는 Amazon Elastic Container Service(Amazon ECS) 작업에 스크립트를 패키징합니다.

Exam question from Amazon's AWS Certified Machine Learning - Specialty
Question #: 222
Topic #: 1

A data scientist at a financial services company used Amazon SageMaker to train and deploy a model that predicts loan defaults. The model analyzes new loan applications and predicts the risk of loan default. To train the model, the data scientist manually extracted loan data from a database. The data scientist performed the model training and deployment steps in a Jupyter notebook that is hosted on SageMaker Studio notebooks. The model's prediction accuracy is decreasing over time.

Which combination of steps is the MOST operationally efficient way for the data scientist to maintain the model's accuracy? (Choose two.)

  • A. Use SageMaker Pipelines to create an automated workflow that extracts fresh data, trains the model, and deploys a new version of the model. Most Voted
  • B. Configure SageMaker Model Monitor with an accuracy threshold to check for model drift. Initiate an Amazon CloudWatch alarm when the threshold is exceeded. Connect the workflow in SageMaker Pipelines with the CloudWatch alarm to automatically initiate retraining. Most Voted
  • C. Store the model predictions in Amazon S3. Create a daily SageMaker Processing job that reads the predictions from Amazon S3, checks for changes in model prediction accuracy, and sends an email notification if a significant change is detected.
  • D. Rerun the steps in the Jupyter notebook that is hosted on SageMaker Studio notebooks to retrain the model and redeploy a new version of the model.
  • E. Export the training and deployment code from the SageMaker Studio notebooks into a Python script. Package the script into an Amazon Elastic Container Service (Amazon ECS) task that an AWS Lambda function can initiate.
