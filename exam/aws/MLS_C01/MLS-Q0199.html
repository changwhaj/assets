Amazon의 AWS Certified Machine Learning – 전문 분야 시험 문제
질문 #: 199
주제 #: 1

제조회사는 반품된 스마트폰 중 습기로 인해 손상된 스마트폰을 식별해야 합니다. 이 회사는 각 전화기에 대해 2,000개의 진단 값을 생성하는 자동화된 프로세스를 보유하고 있습니다. 데이터베이스에는 500만 개가 넘는 전화 평가가 포함되어 있습니다. 평가 프로세스는 일관되며 데이터에 누락된 값이 없습니다. 기계 학습(ML) 전문가는 Amazon SageMaker 선형 학습기 ML 모델을 교육하여 사용 가능한 모든 기능을 사용하여 전화기를 습기로 인해 손상되었는지 또는 습기로 인해 손상되지 않았는지 분류했습니다. 모델의 F1 점수는 0.6입니다.

모델 학습의 어떤 변경 사항이 모델의 F1 점수를 가장 향상시킬 수 있습니까? (2개를 선택하세요.)

  • A. SageMaker 선형 학습자 알고리즘을 계속 사용하십시오. SageMaker 주성분 분석(PCA) 알고리즘을 사용하여 기능 수를 줄입니다. 최다 투표
  • B. SageMaker 선형 학습자 알고리즘을 계속 사용합니다. scikit-learn 다차원 스케일링(MDS) 알고리즘을 사용하여 기능 수를 줄입니다.
  • C. SageMaker 선형 학습자 알고리즘을 계속 사용합니다. 예측변수 유형을 회귀변수로 설정합니다.
  • D. k가 1,000 미만인 SageMaker k-평균 알고리즘을 사용하여 모델을 훈련합니다.
  • E. SageMaker k-Nearest Neighbor(k-NN) 알고리즘을 사용합니다. 모델을 학습하려면 차원 감소 목표를 1,000 미만으로 설정하세요. 최다 투표

Exam question from Amazon's AWS Certified Machine Learning - Specialty
Question #: 199
Topic #: 1

A manufacturing company needs to identify returned smartphones that have been damaged by moisture. The company has an automated process that produces 2,000 diagnostic values for each phone. The database contains more than five million phone evaluations. The evaluation process is consistent, and there are no missing values in the data. A machine learning (ML) specialist has trained an Amazon SageMaker linear learner ML model to classify phones as moisture damaged or not moisture damaged by using all available features. The model's F1 score is 0.6.

Which changes in model training would MOST likely improve the model's F1 score? (Choose two.)

  • A. Continue to use the SageMaker linear learner algorithm. Reduce the number of features with the SageMaker principal component analysis (PCA) algorithm. Most Voted
  • B. Continue to use the SageMaker linear learner algorithm. Reduce the number of features with the scikit-learn multi-dimensional scaling (MDS) algorithm.
  • C. Continue to use the SageMaker linear learner algorithm. Set the predictor type to regressor.
  • D. Use the SageMaker k-means algorithm with k of less than 1,000 to train the model.
  • E. Use the SageMaker k-nearest neighbors (k-NN) algorithm. Set a dimension reduction target of less than 1,000 to train the model. Most Voted
