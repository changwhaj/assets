주제 #: 1

한 에너지 회사에는 원격 측정 데이터를 생성하는 풍력 터빈, 기상 관측소 및 태양광 패널이 있습니다. 회사는 이러한 장치에 대한 예측 유지 관리를 수행하려고 합니다. 장치가 다양한 위치에 있으며 인터넷 연결이 불안정합니다.
데이터 과학자 팀은 원격 측정 데이터를 사용하여 기계 학습(ML)을 수행하여 이상 징후를 탐지하고 장치가 악화되기 전에 유지 관리를 예측하고 있습니다. 팀에는 확장 가능하고 안전한 고속 데이터 수집 메커니즘이 필요합니다. 팀은 Amazon S3를 데이터 저장 위치로 사용하기로 결정했습니다.
이러한 요구 사항을 충족하는 접근 방식은 무엇입니까?

  • A. Amazon EC2에서 호스팅되는 웹 서버에 대한 HTTP API 호출을 사용하여 데이터를 수집합니다. Elastic Load Balancer 뒤의 Auto Scaling 구성에서 EC2 인스턴스를 설정하여 Amazon S3에 데이터를 로드합니다.
  • B. MQTT(Message Queuing Telemetry Transport)를 통해 AWS IoT Core로 데이터를 수집합니다. Amazon Kinesis Data Firehose를 사용하여 S3 버킷에 쓰도록 구성된 Amazon Kinesis 데이터 스트림으로 데이터를 보내도록 AWS IoT Core에서 규칙을 설정합니다.
  • C. MQTT(Message Queuing Telemetry Transport)를 통해 AWS IoT Core로 데이터를 수집합니다. 모든 MQTT 데이터를 S3 버킷에 쓰도록 구성된 Amazon Kinesis Data Firehose 전송 스트림으로 전달하도록 AWS IoT Core에서 규칙을 설정합니다. 최다 투표
  • D. MQTT(Message Queuing Telemetry Transport)를 통해 S3 버킷에 쓰도록 구성된 Amazon Kinesis 데이터 스트림으로 데이터를 수집합니다.

Exam question from Amazon's AWS Certified Machine Learning - Specialty
Question #: 184
Topic #: 1

An energy company has wind turbines, weather stations, and solar panels that generate telemetry data. The company wants to perform predictive maintenance on these devices. The devices are in various locations and have unstable internet connectivity.
A team of data scientists is using the telemetry data to perform machine learning (ML) to conduct anomaly detection and predict maintenance before the devices start to deteriorate. The team needs a scalable, secure, high-velocity data ingestion mechanism. The team has decided to use Amazon S3 as the data storage location.
Which approach meets these requirements?

  • A. Ingest the data by using an HTTP API call to a web server that is hosted on Amazon EC2. Set up EC2 instances in an Auto Scaling configuration behind an Elastic Load Balancer to load the data into Amazon S3.
  • B. Ingest the data over Message Queuing Telemetry Transport (MQTT) to AWS IoT Core. Set up a rule in AWS IoT Core to use Amazon Kinesis Data Firehose to send data to an Amazon Kinesis data stream that is configured to write to an S3 bucket.
  • C. Ingest the data over Message Queuing Telemetry Transport (MQTT) to AWS IoT Core. Set up a rule in AWS IoT Core to direct all MQTT data to an Amazon Kinesis Data Firehose delivery stream that is configured to write to an S3 bucket. Most Voted
  • D. Ingest the data over Message Queuing Telemetry Transport (MQTT) to Amazon Kinesis data stream that is configured to write to an S3 bucket.
