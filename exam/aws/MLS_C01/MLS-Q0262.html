최소한의 개발 노력으로 이러한 요구 사항을 충족하는 솔루션은 무엇입니까?

  • A. AWS Panorama를 사용하여 사진 속 유명인을 식별하십시오. AWS CloudTrail을 사용하여 IP 주소 및 타임스탬프 세부 정보를 캡처합니다.
  • B. AWS Panorama를 사용하여 사진 속 유명인을 식별하십시오. AWS Panorama Device SDK를 호출하여 IP 주소 및 타임스탬프 세부 정보를 캡처합니다.
  • C. Amazon Rekognition을 사용하여 사진 속 유명인을 식별합니다. AWS CloudTrail을 사용하여 IP 주소 및 타임스탬프 세부 정보를 캡처합니다. 최다 투표
  • D. Amazon Rekognition을 사용하여 사진 속 유명인을 식별합니다. 텍스트 감지 기능을 사용하여 IP 주소 및 타임스탬프 세부정보를 캡처하세요.

Which solution will meet these requirements with LEAST development effort?

  • A. Use AWS Panorama to identify celebrities in the pictures. Use AWS CloudTrail to capture IP address and timestamp details.
  • B. Use AWS Panorama to identify celebrities in the pictures. Make calls to the AWS Panorama Device SDK to capture IP address and timestamp details.
  • C. Use Amazon Rekognition to identify celebrities in the pictures. Use AWS CloudTrail to capture IP address and timestamp details. Most Voted
  • D. Use Amazon Rekognition to identify celebrities in the pictures. Use the text detection feature to capture IP address and timestamp details.
