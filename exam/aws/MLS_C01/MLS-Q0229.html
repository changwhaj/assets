주제 #: 1

온라인 소매 회사는 고객 서비스를 개선하기 위해 자연어 처리(NLP) 모델을 개발하려고 합니다. 기계 학습(ML) 전문가가 Amazon SageMaker에서 BERT(변환기의 양방향 인코더 표현) 모델에 대한 분산 교육을 설정하고 있습니다. SageMaker는 분산 교육을 위해 8개의 컴퓨팅 인스턴스를 사용합니다.

ML 전문가는 분산 교육 중에 데이터 보안을 보장하려고 합니다. 데이터는 Amazon S3 버킷에 저장됩니다.

ML 전문가는 분산 학습 중에 데이터를 보호하기 위해 어떤 단계 조합을 수행해야 합니까? (3개를 선택하세요.)

  • A. 프라이빗 VPC에서 분산 교육 작업을 실행합니다. 컨테이너 간 트래픽 암호화를 활성화합니다. 최다 투표
  • B. 여러 VPC에서 분산 교육 작업을 실행합니다. VPC 피어링을 활성화합니다.
  • C. S3 VPC 엔드포인트를 생성합니다. 그런 다음 네트워크 경로, 엔드포인트 정책 및 S3 버킷 정책을 구성합니다. 최다 투표
  • D. IAM 역할을 사용하여 SageMaker 리소스에 대한 읽기 전용 액세스 권한을 부여합니다. 최다 투표
  • E. NAT 게이트웨이를 생성합니다. NAT 게이트웨이에 탄력적 IP 주소를 할당합니다.
  • F. 교육 인스턴스와 연결된 보안 그룹의 트래픽을 허용하도록 인바운드 규칙을 구성합니다.

Exam question from Amazon's AWS Certified Machine Learning - Specialty
Question #: 229
Topic #: 1

An online retail company wants to develop a natural language processing (NLP) model to improve customer service. A machine learning (ML) specialist is setting up distributed training of a Bidirectional Encoder Representations from Transformers (BERT) model on Amazon SageMaker. SageMaker will use eight compute instances for the distributed training.

The ML specialist wants to ensure the security of the data during the distributed training. The data is stored in an Amazon S3 bucket.

Which combination of steps should the ML specialist take to protect the data during the distributed training? (Choose three.)

  • A. Run distributed training jobs in a private VPC. Enable inter-container traffic encryption. Most Voted
  • B. Run distributed training jobs across multiple VPCs. Enable VPC peering.
  • C. Create an S3 VPC endpoint. Then configure network routes, endpoint policies, and S3 bucket policies. Most Voted
  • D. Grant read-only access to SageMaker resources by using an IAM role. Most Voted
  • E. Create a NAT gateway. Assign an Elastic IP address for the NAT gateway.
  • F. Configure an inbound rule to allow traffic from a security group that is associated with the training instances.
