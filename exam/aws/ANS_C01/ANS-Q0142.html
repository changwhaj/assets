Amazon AWS Certified Advanced Networking - Specialty ANS-C01 시험문제
질문 #: 142
주제 #: 1

한 IoT 회사는 미국과 남아시아에 배포된 수천 개의 센서로부터 데이터를 수집합니다. 센서는 UDP를 기반으로 구축된 독점 통신 프로토콜을 사용하여 Amazon EC2 인스턴스 집합에 데이터를 보냅니다. 인스턴스는 Auto Scaling 그룹에 속하며 NLB(Network Load Balancer) 뒤에서 실행됩니다. 인스턴스, Auto Scaling 그룹 및 NLB는 us-west-2 리전에 배포됩니다.

남아시아 센서의 데이터가 인터넷을 통해 전송되는 동안 손실되어 EC2 인스턴스에 도달하지 못하는 경우가 있습니다.

이 문제를 해결하려면 어떤 솔루션이 필요합니까? (2개를 선택하세요.)

  • A. 기존 NLB와 함께 AWS Global Accelerator를 사용하십시오. 최다 투표
  • B. Amazon CloudFront 배포판을 생성합니다. 기존 NLB를 원본으로 지정합니다.
  • C. ap-south-1 지역에 EC2 인스턴스와 NLB의 두 번째 배포를 생성합니다. Amazon Route 53 지연 시간 라우팅 정책을 사용하여 지연 시간이 가장 짧은 지역을 확인하세요. 최다 투표
  • D. ap-south-1 지역에 EC2 인스턴스와 NLB의 두 번째 배포를 생성합니다. 패킷이 삭제된 경우 Amazon Route 53 장애 조치 라우팅 정책을 사용하여 대체 리전을 확인합니다.
  • E. 최신 ENA(Elastic Network Adapter) 드라이버를 사용하여 EC2 인스턴스에서 향상된 네트워킹을 활성화합니다.

Exam question from Amazon's AWS Certified Advanced Networking - Specialty ANS-C01
Question #: 142
Topic #: 1

An IoT company collects data from thousands of sensors that are deployed in the Unites States and South Asia. The sensors use a proprietary communication protocol that is built on UDP to send the data to a fleet of Amazon EC2 instances. The instances are in an Auto Scaling group and run behind a Network Load Balancer (NLB). The instances, Auto Scaling group, and NLB are deployed in the us-west-2 Region.

Occasionally, the data from the sensors in South Asia gets lost in transit over the internet and does not reach the EC2 instances.

Which solutions will resolve this issue? (Choose two.)

  • A. Use AWS Global Accelerator with the existing NLB. Most Voted
  • B. Create an Amazon CloudFront distribution. Specify the existing NLB as the origin.
  • C. Create a second deployment of the EC2 instances and the NLB in the ap-south-1 Region. Use an Amazon Route 53 latency routing policy to resolve to the Region that provides the least latency. Most Voted
  • D. Create a second deployment of the EC2 instances and the NLB in the ap-south-1 Region. Use an Amazon Route 53 failover routing policy to resolve to an alternate Region in case packets are dropped.
  • E. Turn on enhanced networking on the EC2 instances by using the most recent Elastic Network Adapter (ENA) drivers.
